# Rethinking Superpixel Segmentation from Biologically Inspired Mechanisms

## Quick Facts
- arXiv ID: 2309.13438
- Source URL: https://arxiv.org/abs/2309.13438
- Reference count: 40
- Primary result: Achieves state-of-the-art superpixel segmentation performance on BSDS500 and NYUv2 datasets using biomimetic mechanisms

## Executive Summary
This paper introduces a novel superpixel segmentation network that draws inspiration from biological visual mechanisms. The proposed method integrates two key components: the Enhanced Screening Module (ESM) that simulates multilevel synchronization in the visual cortex, and the Boundary-Aware Label (BAL) that emulates spatial frequency sensitivity. The network demonstrates superior performance across multiple metrics on standard benchmarks, showing improved boundary adherence and semantic coherence without requiring fine-tuning for different datasets.

## Method Summary
The method combines an encoder-decoder architecture with two biologically-inspired modules. The Enhanced Screening Module (ESM) integrates low-level texture/color features with high-level semantic information through interactive projection mechanisms, simulating the ventral visual pathway's multilevel synchronization. The Boundary-Aware Label (BAL) encodes ground truth labels using Gaussian distributions where variance decreases with distance from edges, amplifying the network's responsiveness to boundaries. The network is trained using cross-entropy loss with BAL-encoded labels and L2 reconstruction loss, achieving state-of-the-art results on BSDS500 and NYUv2 datasets.

## Key Results
- Achieves state-of-the-art ASA (achievable segmentation accuracy) scores on BSDS500 and NYUv2 datasets
- Demonstrates superior boundary adherence with high BR-BP (boundary recall-precision) scores
- Maintains strong compactness (CO) metric performance while improving boundary precision
