# On the Accuracy of Hotelling-Type Asymmetric Tensor Deflation: A Random Tensor Analysis

## Quick Facts
- arXiv ID: 2310.18717
- Source URL: https://arxiv.org/abs/2310.18717
- Reference count: 0
- One-line primary result: Analyzes Hotelling-type tensor deflation accuracy under noise using random tensor theory

## Executive Summary
This work studies the accuracy of Hotelling-type tensor deflation for low-rank asymmetric tensors corrupted by Gaussian noise. The paper characterizes analytically the estimated singular values and alignment between estimated and true singular vectors at each deflation step. By leveraging recent advances in random tensor theory, the authors derive a system of equations that describes these limits as functions of signal-to-noise ratios and cross-term alignments. The framework enables construction of estimators for underlying model parameters and is illustrated on a rank-2, order-3 tensor model.

## Method Summary
The paper analyzes a low-rank asymmetric tensor model with correlated components and additive Gaussian noise. The Hotelling deflation procedure iteratively computes rank-one approximations and subtracts estimated components. Using random tensor theory, the authors associate each deflated tensor with a structured random matrix and analyze its spectral properties through the Stieltjes transform. This leads to a system of equations characterizing the asymptotic limits of estimated singular values and alignment measures as dimensions grow.

## Key Results
- Derives a system of nonlinear equations linking estimated singular values and alignment parameters
- Characterizes accuracy of tensor deflation as a function of signal-to-noise ratios and cross-term alignments
- Constructs estimators for underlying model parameters using the derived system
- Illustrates the approach on a rank-2, order-3 tensor model showing evolution across deflation steps
