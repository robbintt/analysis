# Hybrid Quantum Neural Network in High-dimensional Data Classification

## Quick Facts
- arXiv ID: 2312.01024
- Source URL: https://arxiv.org/abs/2312.01024
- Reference count: 13
- Key outcome: Hybrid model achieves 89.79% accuracy with 226.46 MB model size vs 88.69% and 382.74 MB for classical ResNeXt CNN

## Executive Summary
This research introduces a hybrid quantum neural network architecture that combines classical convolutional layers with a quantum binary classifier for high-dimensional data classification. The model was evaluated on the Bird-CLEF 2021 dataset, demonstrating superior performance compared to a classical ResNeXt CNN. The hybrid approach achieves higher accuracy while maintaining a smaller model size, though with slightly increased training time.

## Method Summary
The hybrid architecture uses a pre-trained ResNeXt CNN to extract features from high-dimensional audio data, which are then encoded into a single-qubit quantum state using Z-feature maps. The Real Amplitude ansatz optimizes the quantum state for classification, with the model trained using Adam optimizer and Cross-Entropy loss. The quantum neural network is implemented using IBM Qiskit's Sampler primitive.

## Key Results
- Achieved 89.79% accuracy compared to 88.69% for classical ResNeXt CNN
- Reduced model size from 382.74 MB to 226.46 MB
- Training time increased from 422.15 to 443.48 seconds
- Single-qubit quantum circuit effectively processed features from pre-trained CNN

- Core assumption: The quantum state can effectively capture the relevant information from the classical features for classification.

- Core assumption: The ansatz can represent the decision boundary for the binary classification task.

  3. Real Amplitude ansatz optimizes the quantum state.
  4. Quantum state is measured to obtain classification probabilities.
  5. Cross-entropy loss is computed and used to update model parameters.
- Design tradeoffs:
  - Model size vs. accuracy: The hybrid model achieves higher accuracy with a smaller model size compared to the classical ResNeXt CNN.
  - Training time vs. performance: The hybrid model has a slightly longer training time but offers improved accuracy and model size.
