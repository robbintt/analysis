# Age-Stratified Differences in Morphological Connectivity Patterns in ASD: An sMRI and Machine Learning Approach

## Quick Facts
- arXiv ID: 2308.07356
- Source URL: https://arxiv.org/abs/2308.07356
- Reference count: 28
- Key outcome: Age-stratified morphological connectivity features achieved highest classification accuracy (75.8%) in 6-11 year old ASD group using random forest

## Executive Summary
This study investigates age-stratified differences in morphological connectivity patterns in autism spectrum disorder (ASD) using structural MRI and machine learning. The researchers analyzed 710 subjects from ABIDE databases, divided into three age groups (6-11, 11-18, and 6-18 years), and extracted morphological features (MF) and morphological connectivity features (MCF) from 148 brain regions. Using random forest classification, the 6-11 age group achieved the highest accuracy, with MCF outperforming MF. The study demonstrates that age-stratified approaches using morphological connectivity features can effectively discriminate ASD cases, with the frontal lobe contributing most features for classification.

## Method Summary
The study used structural MRI data from 710 subjects (313 ASD, 397 TD) from ABIDE-I and ABIDE-II databases. Data was divided into three age groups: 6-11, 11-18, and 6-18 years. FreeSurfer preprocessing with Destrieux atlas parcellation extracted morphological features (area, thickness, volume, mean curvature) from 148 regions. Morphological connectivity features were computed as Euclidean distances between regional morphological profiles. Features were selected using two-sample t-test (p<0.05), then random forest classifiers were trained with 80/20 train-test splits. Performance was evaluated using accuracy, F1 score, recall, and precision.

## Key Results
- The 6-11 age group achieved highest classification accuracy at 75.8% with MCF features
- MCF features outperformed MF features across all age groups
- 6-11 age group with MCF achieved 83.1% F1 score, 86% recall, and 80.4% precision
- Frontal lobe contributed the most features for classification

## Architecture Onboarding

- Component map: ABIDE-I/II download -> Age stratification (6-11, 11-18, 6-18) -> FreeSurfer recon-all -> Destrieux parcellation -> Feature extraction (MF: 592, MCF: 10,878) -> t-test feature selection -> Random Forest training (80/20 split) -> Performance evaluation
- Critical path: The FreeSurfer preprocessing must complete successfully before any feature extraction can occur; feature selection must finish before RF training
- Design tradeoffs: Euclidean distance for MCF is computationally efficient but may miss non-linear relationships; using only RF avoids hyperparameter tuning complexity but may miss better-performing algorithms
- Failure signatures: Low classification accuracy across all age groups suggests preprocessing errors or insufficient feature discriminative power; highest accuracy in older groups would contradict the study's findings and suggest different developmental patterns
- First 3 experiments:
  1. Run FreeSurfer recon-all on a small subset of ABIDE data and verify Destrieux parcellation quality visually

## Limitations
- The study assumes Euclidean distance adequately captures morphological connectivity patterns without validation of alternative metrics
- Claims about accelerated early brain growth in ASD are based on classification performance rather than direct neurodevelopmental measurements
- Default random forest parameters were used without optimization, potentially limiting generalizability

## Confidence

- High confidence: Age-stratified classification performance differences (empirical results are directly reported)
- Medium confidence: MCF superiority over MF for classification (supported by results but mechanism not fully validated)
- Low confidence: Developmental mechanism explanations (based on assumptions rather than direct neurodevelopmental measurements)
