# Bengali Document Layout Analysis -- A YOLOV8 Based Ensembling Approach

## Executive Summary
This paper presents an ensemble-based approach for Bengali Document Layout Analysis (DLA) using YOLOv8. The authors address the challenges of complex Bengali script by employing data augmentation and a two-stage prediction strategy. Their method combines a general YOLOv8 model with a specialized image model, along with post-processing techniques like convex hull filling to improve segmentation accuracy. The ensemble approach outperforms individual models, achieving strong results on the BaDLAD dataset.

## Method Summary
The approach uses a two-stage prediction strategy with YOLOv8-seg models. First, a general model detects all four element types (text boxes, paragraphs, images, tables). Then a specialized image model refines image segmentation. Post-processing with convex hull filling addresses gaps in table masks. The method employs extensive data augmentation including Mixup, copy-paste, mosaic, and perspective transformations to improve robustness for Bengali documents.

## Key Results
- Public DICE score of 0.89537 on BaDLAD dataset
- Ensemble approach outperforms individual base architectures
- Convex hull filling successfully repairs gaps in table masks at class intersections
- Specialized image model improves accuracy for image segmentation where general model underperforms

- Design tradeoffs:

  - Convex hull filling fixes gaps but may oversimplify complex table shapes
