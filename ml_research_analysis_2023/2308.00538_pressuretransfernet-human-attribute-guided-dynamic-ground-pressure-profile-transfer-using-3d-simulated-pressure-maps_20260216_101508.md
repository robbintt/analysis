# PressureTransferNet: Human Attribute Guided Dynamic Ground Pressure Profile Transfer using 3D simulated Pressure Maps

## Quick Facts
- arXiv ID: 2308.00538
- Source URL: https://arxiv.org/abs/2308.00538
- Reference count: 0
- Key outcome: PressureTransferNet achieves a binary R-square value of 0.79 on areas with ground contact and a macro F1 score of 0.911±0.015 on physical pressure mat data

## Executive Summary
PressureTransferNet introduces a novel method for Human Activity Recognition (HAR) that generates body-specific dynamic ground pressure profiles by transferring attributes between pressure maps. The approach leverages existing pressure data from different individuals and uses a U-Net-inspired encoder-decoder neural network to synthesize pressure maps that reflect target human attributes like height, weight, and gender. The model takes a source pressure map and target attribute vector as inputs, producing a new pressure map representing the target body type performing the same activity. Evaluation on both simulated and real-world datasets demonstrates the effectiveness of the method for data augmentation, denoising, sensor simulation, and anomaly detection in applications like sports science and rehabilitation.

## Method Summary
PressureTransferNet employs a U-Net-inspired encoder-decoder architecture to transfer human attributes between pressure maps. The encoder extracts motion features from 30 consecutive frames of 80×28 pressure sensor data, generating a 128-dimensional latent representation. This representation is concatenated with the target human attributes (height, weight, gender) and passed to the decoder, which outputs the resulting pressure sensor map for 30 frames. The model is trained on a diverse dataset generated using the PresSim pipeline and SMPL-X models, with a content loss function ensuring the preservation of essential pressure distribution patterns. Evaluation includes binary R-square metrics for shape matching and F1-score validation on physical pressure mat data.

## Key Results
- Achieves binary R-square value of 0.79 on areas with ground contact for attribute transfer
- Macro F1 score of 0.911±0.015 on physical pressure mat data validates synthesized pressure map quality
- Shows improved HAR classification accuracy when trained on combined real and synthetic pressure maps

## Architecture Onboarding

- Component map: PresSim pipeline → SMPL-X models → PressureTransferNet → Classifier
- Critical path: 3D pose → PresSim simulation → pressure map → PressureTransferNet → attribute-modified pressure map → classifier evaluation
- Design tradeoffs:
  - Using synthetic data allows unlimited training samples but may not capture all real-world variations
  - The 30-frame sliding window balances temporal context with computational efficiency
  - Concatenating attributes at the latent level vs. input level affects the model's ability to learn attribute-independent motion features
- Failure signatures:
  - High RMSE values indicate poor pressure map reconstruction
  - Low binary R2 scores suggest the synthesized pressure shapes don't match ground truth
  - Minimal improvement in classifier F1 score when adding synthetic data indicates poor quality synthetic maps
- First 3 experiments:
  1. Train PressureTransferNet on synthetic data and evaluate on held-out synthetic sequences to establish baseline performance
  2. Fine-tune the trained model on a small real-world dataset to assess transfer learning potential
  3. Generate synthetic pressure maps for a new subject not in the training set and validate using the classifier to test generalization

## Limitations
- The evaluation relies heavily on simulated data, which may not capture all real-world complexities of pressure distribution
- The model shows good performance on synthetic sequences but experiences performance degradation (binary R2 drops from 0.79 to 0.70) on unseen motion sequences, indicating limited generalization
- The study uses a relatively small real-world dataset (7 subjects) for validation, which may not represent the full diversity of human movement patterns

## Confidence
