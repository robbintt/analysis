# Generating Multidimensional Clusters With Support Lines

## Quick Facts
- arXiv ID: 2301.10327
- Source URL: https://arxiv.org/abs/2301.10327
- Reference count: 40
- Key outcome: Clugen generates synthetic multidimensional clusters supported by line segments, enabling controlled cluster elongation and eccentricity with flexible parameterization.

## Executive Summary
This paper introduces Clugen, a novel algorithm for generating synthetic multidimensional clusters supported by line segments. The method provides precise control over cluster elongation by placing cluster centers along line segments with configurable directions and lengths. Clugen supports arbitrary statistical distributions for both projection placement and final point dispersion, allowing rich and diverse cluster shapes. The algorithm is implemented in four open-source languages (Python, R, Julia, MATLAB/Octave) and includes comprehensive testing. A usage example demonstrates Clugen's ability to generate challenging benchmark datasets for evaluating clustering algorithms under controlled conditions.

## Method Summary
Clugen generates multidimensional synthetic clusters by anchoring them along line segments. The algorithm places cluster centers, generates line segments with controllable directions and lengths, then distributes points along and around these lines using configurable statistical distributions. Users provide PRNG seeds or objects to ensure reproducibility. The method supports arbitrary distributions for both projection placement and final point dispersion, applied dimension-wise along line scaffolds. Four open-source implementations are provided across Python, R, Julia, and MATLAB/Octave.

## Key Results
- Clugen enables reproducible synthetic data generation with full parameterization control
- Four open-source implementations available across Python, R, Julia, and MATLAB/Octave
- Clustering performance degrades with increasing cluster elongation, with AHC and fuzzy c-means performing best
