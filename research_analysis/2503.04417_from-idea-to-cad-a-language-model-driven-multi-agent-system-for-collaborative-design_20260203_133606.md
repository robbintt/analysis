## üìã Executive Summary

> **Current approaches to AI-assisted design are hindered by a critical gap between abstract conceptual inputs (such as sketches or text) and the precise, parametric engineering models required for industrial manufacturing.**

This paper addresses the challenge of automating the translation of multimodal inputs into fully parametric, code-based CAD models using the CadQuery Python library. The key innovation is a **Vision Language Model (VLM)-based Multi-Agent System (MAS)** that replicates the structure of an industrial product development team operating within the V-model framework.

Technically, the system deploys three specialized LLM-based agents:
*   **RequirementsEngineer:** Parses user inputs and manages specifications.
*   **CadEngineer:** Formulates construction plans and writes code.
*   **QualityAssuranceEngineer:** Performs visual verification to ensure geometric compliance.

The architecture features a dual-loop feedback mechanism: an **inner loop** facilitates automated correction between the CadEngineer and QualityAssuranceEngineer to refine code, while an **outer loop** enables collaborative validation between the RequirementsEngineer and the user. The system demonstrated robust performance in end-to-end generation tasks, successfully creating parametric CAD models from multimodal inputs without manual intervention.

This research represents a significant paradigm shift from static 3D mesh generation to dynamic, parametric engineering workflows, setting a new precedent for Human-AI collaboration in complex technical automation.

*   **RequirementsEngineer:** Responsible for clarifying specifications and interacting directly with the user to define constraints.
*   **CadEngineer:** Responsible for creating construction plans and generating the actual CAD code.
*   **QualityAssuranceEngineer:** Responsible for verifying model compliance and visual correctness.
