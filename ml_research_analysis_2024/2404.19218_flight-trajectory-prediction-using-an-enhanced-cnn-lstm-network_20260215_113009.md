# Flight Trajectory Prediction Using an Enhanced CNN-LSTM Network

## Quick Facts
- arXiv ID: 2404.19218
- Source URL: https://arxiv.org/abs/2404.19218
- Reference count: 17
- Achieves 32% improvement in ADE and 34% improvement in FDE over baseline CNN-LSTM

## Executive Summary
This paper addresses the challenge of accurately predicting fighter aircraft trajectories in close-range air combat, where high speeds, diverse tactical maneuvers, and rapidly changing situations create prediction difficulties. The authors propose an enhanced CNN-LSTM network that combines convolutional layers for spatial feature extraction, a social-pooling module to capture interactions between multiple fighters, and an attention mechanism to identify trajectory mutations. The model processes historical trajectory data to predict future positions, achieving improvements of 32% in ADE and 34% in FDE compared to the baseline CNN-LSTM method. Experiments using real flight data from air combat simulations demonstrate the effectiveness of the attention mechanism and social-pooling components in improving prediction accuracy for both simple and complex trajectory scenarios.

## Method Summary
The enhanced CNN-LSTM network processes 8 historical trajectory points (x, y, z coordinates) to predict 8 future positions. The architecture combines 1D CNN layers for spatial feature extraction, social-pooling module for capturing interactions between multiple fighters, attention mechanism for identifying trajectory mutations, and LSTM layers for temporal feature extraction. The model is trained using Adam optimizer with learning rate 1e-4 and decay rate 0.5, batch size 64, for 100 epochs on data split 4:1 for training and testing.

## Key Results
- Achieves 32% improvement in Average Displacement Error (ADE) over baseline CNN-LSTM
- Achieves 34% improvement in Final Displacement Error (FDE) over baseline CNN-LSTM
- Demonstrates effectiveness of attention mechanism and social-pooling components in both simple and complex trajectory scenarios

### Mechanism 1
- Claim: The social-pooling module improves prediction accuracy by aggregating spatial features between multiple fighters.
- Mechanism: The social-pooling layer captures interactions between fighters by combining their spatial features through a neighbor-based aggregation function. This allows the model to account for relative positions and tactical relationships between aircraft.
- Core assumption: Fighters in close-range air combat influence each other's trajectories through competitive and collaborative interactions.

### Mechanism 2
- Claim: The attention mechanism improves prediction accuracy by focusing on trajectory mutation points.
- Mechanism: The modified Luong Attention assigns higher weights to segments where trajectories undergo changes, allowing the network to better predict complex and variable trajectories. The attention weights are calculated based on the importance of features at each time step.
- Core assumption: Trajectory mutations (sudden changes in direction or speed) contain critical information for future trajectory prediction.
- Evidence anchors:

### Mechanism 3
- Claim: Combining CNN for spatial feature extraction with LSTM for temporal feature extraction creates a complementary dual-feature learning system.
- Mechanism: CNN layers extract spatial patterns from historical trajectory data, while LSTM layers capture temporal dependencies and long-term sequential patterns. The merged features provide a comprehensive representation for trajectory prediction.
- Core assumption: Fighter trajectories contain both spatial patterns (maneuvers, formations) and temporal patterns (timing, sequence dependencies) that require different network architectures to extract effectively.

## Architecture Onboarding

- Component map: Input → CNN → [Social-pooling + Attention] → LSTM → FC → Output
- Critical path: Input → CNN → [Social-pooling + Attention] → LSTM → FC → Output
  The attention and social-pooling modules operate in parallel before feeding into LSTM.
- Design tradeoffs:
  - CNN vs RNN for spatial features: CNN provides translation invariance and parameter sharing
  - Attention placement: Encoder-side attention focuses on input features vs decoder-side attention on context
  - Social-pooling complexity: Adds computation but captures fighter interactions
  - Prediction window size: 8 steps balances accuracy and computational load
- Failure signatures:
  - Poor ADE/FDE on simple trajectories: Attention mechanism overfits to mutations
  - High computational time: Social-pooling or attention mechanisms too complex
  - Inconsistent predictions: LSTM failing to capture temporal patterns
  - Mode collapse: Network predicting only straight trajectories
- First 3 experiments:
  1. Baseline test: Run model with only CNN-LSTM (no attention, no social-pooling) to establish baseline ADE/FDE
  2. Ablation test: Remove attention mechanism to measure its contribution to mutation handling
  3. Ablation test: Remove social-pooling to measure its contribution to fighter interaction modeling
