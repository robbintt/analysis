# ReCODE: Modeling Repeat Consumption with Neural ODE

## Quick Facts
- arXiv ID: 2405.16550
- Source URL: https://arxiv.org/abs/2405.16550
- Reference count: 39
- Primary result: ReCODE achieves up to 45.7% relative improvement in Recall@50 and 55.6% relative improvement in NDCG@50 over state-of-the-art sequential recommendation models on music datasets

## Executive Summary
This paper introduces ReCODE, a novel framework for modeling repeat consumption patterns in recommender systems. The authors propose using neural ordinary differential equations (ODEs) to capture the dynamic temporal patterns between repeated user interactions with items, without relying on predefined temporal distributions. ReCODE consists of two components: a static preference prediction module and a dynamic repeat intention module that models the temporal dynamics using neural ODEs. The framework is designed to be model-agnostic, allowing integration with various base recommendation models.

## Method Summary
ReCODE addresses the challenge of modeling repeat consumption in recommender systems by combining static preference modeling with dynamic temporal modeling using neural ODEs. The framework captures both the user's static preferences for items and the dynamic patterns that govern repeat consumption behavior. The dynamic module models the time intervals between repeated interactions using neural ODEs, which can learn complex temporal patterns without assuming specific distributions. This approach allows ReCODE to better capture the nuanced temporal dynamics of repeat consumption that traditional methods often miss through heuristic assumptions.

## Key Results
- ReCODE with MF as static module achieves up to 45.7% relative improvement in Recall@50
- ReCODE with MF as static module achieves up to 55.6% relative improvement in NDCG@50
- ReCODE outperforms existing baseline methods on two real-world music recommendation datasets

## Why This Works (Mechanism)
ReCODE works by explicitly modeling both static preferences and dynamic temporal patterns between repeated consumptions. Traditional recommender systems often fail to capture the nuanced temporal dynamics of repeat behavior, which is crucial for accurate recommendations. By using neural ODEs, ReCODE can learn complex temporal patterns without being constrained by predefined distributions, allowing it to capture the irregular and evolving nature of repeat consumption behavior.

## Foundational Learning
- Neural Ordinary Differential Equations: Continuous-depth models that can learn complex temporal dynamics without discretization; needed to capture irregular time intervals between repeated consumptions
- Repeat Consumption Patterns: Users frequently interact with the same items; understanding these patterns is crucial for accurate recommendations in domains like music
- Model-Agnostic Frameworks: Design approaches that can integrate with various base recommendation models; needed for broad applicability across different recommendation scenarios

## Architecture Onboarding

**Component Map**: Static Preference Module -> Dynamic Repeat Intention Module (Neural ODE) -> Final Prediction

**Critical Path**: User-item interaction history → Static preference computation → Temporal pattern modeling via ODE → Repeat intention prediction → Combined recommendation score

**Design Tradeoffs**: 
- Computational complexity of ODE solvers vs. modeling accuracy
- Static vs. dynamic modeling balance
- Model-agnostic design vs. potential performance optimization for specific base models

**Failure Signatures**:
- ODE solver instability during training
- Overfitting to temporal patterns in training data
- Degradation of static preference modeling when dynamic patterns dominate
