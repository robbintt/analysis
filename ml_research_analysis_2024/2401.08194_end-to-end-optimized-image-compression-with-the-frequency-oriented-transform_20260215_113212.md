# End-to-End Optimized Image Compression with the Frequency-Oriented Transform
## Quick Facts
- arXiv ID: 2401.08194
- Source URL: https://arxiv.org/abs/2401.08194
- Reference count: 40
- Outperforms traditional codecs including H.266/VVC on MS-SSIM metric

## Executive Summary
This paper introduces a novel end-to-end optimized image compression framework that leverages frequency-oriented transform to align with human visual system characteristics. The approach decomposes input images into multiple frequency bands, enabling scalable coding through selective transmission of frequency components. The method demonstrates superior performance on perceptual quality metrics compared to traditional codecs while maintaining semantic fidelity for downstream vision tasks.

## Method Summary
The proposed compression model consists of four key components: spatial sampling for resolution reduction, frequency-oriented transform for band decomposition, entropy estimation for bit allocation, and frequency-aware fusion for reconstruction. The frequency-oriented transform separates the image signal into distinct frequency bands, allowing for efficient compression by exploiting the varying importance of different frequency components to human perception. The model is trained end-to-end, optimizing all components jointly for rate-distortion performance.

## Key Results
- Achieves better MS-SSIM performance than traditional codecs including H.266/VVC
- Maintains semantic fidelity for object detection and semantic segmentation tasks
- Enables scalable coding through selective frequency band transmission

## Why This Works (Mechanism)
The method exploits the human visual system's varying sensitivity to different spatial frequencies. By decomposing images into frequency bands and applying different compression strategies to each band based on their perceptual importance, the model can achieve better rate-distortion performance than traditional codecs that treat all frequencies equally. The frequency-aware fusion component ensures that the reconstructed image maintains both perceptual quality and semantic information.

## Architecture Onboarding
Component Map: Input Image -> Spatial Sampling -> Frequency-Oriented Transform -> Entropy Estimation -> Frequency-Aware Fusion -> Output Image

Critical Path: The frequency-oriented transform and frequency-aware fusion components form the critical path, as they directly impact both compression efficiency and reconstruction quality.

Design Tradeoffs: The method trades computational complexity for improved perceptual quality and semantic preservation. The frequency decomposition enables scalability but adds implementation complexity compared to traditional codecs.

Failure Signatures: Potential failure modes include spectral leakage during frequency decomposition, suboptimal entropy estimation leading to bitrate inefficiency, and fusion artifacts when combining frequency bands.
