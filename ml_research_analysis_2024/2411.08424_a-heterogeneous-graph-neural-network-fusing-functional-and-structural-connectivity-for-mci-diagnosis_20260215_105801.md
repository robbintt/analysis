# A Heterogeneous Graph Neural Network Fusing Functional and Structural Connectivity for MCI Diagnosis

## Quick Facts
- arXiv ID: 2411.08424
- Source URL: https://arxiv.org/abs/2411.08424
- Reference count: 40
- Key outcome: Achieves 93.3% mean classification accuracy for MCI diagnosis on ADNI-3 dataset

## Executive Summary
This paper presents a novel heterogeneous graph neural network (HGNN) approach for mild cognitive impairment (MCI) diagnosis by integrating functional connectivity from resting-state fMRI (rs-fMRI) and structural connectivity from diffusion tensor imaging (DTI). The method constructs heterogeneous graphs with carefully designed meta-paths that capture both intra-modal relationships (homo-meta-paths) and cross-modal interactions (hetero-meta-paths) based on structure-function coupling and brain community searching. The approach includes a heterogeneous graph pooling strategy to balance different types of connections and prevent feature confusion, along with a graph augmentation technique to address sample imbalance issues common in clinical datasets.

## Method Summary
The proposed method constructs heterogeneous graphs from dual-modal brain connectivity data, using homo-meta-paths to capture intra-modal relationships and hetero-meta-paths for cross-modal interactions. A heterogeneous graph pooling strategy automatically balances different meta-path types while preventing feature confusion after pooling. The method also introduces graph augmentation that dynamically reconstructs functional connectivity while preserving structural connectivity as a semantic constraint, effectively addressing sample imbalance issues. The HGNN backbone processes these graphs through HAN layers, heterogeneous pooling layers, and readout layers to produce MCI classifications.

## Key Results
- Achieves 93.3% mean classification accuracy for MCI diagnosis on ADNI-3 dataset
- Outperforms existing state-of-the-art methods (82.6%-93.0% accuracy range)
- Ablation study confirms effectiveness of each proposed component, with augmentation method particularly improving performance on imbalanced datasets

## Why This Works (Mechanism)

### Mechanism 1
The proposed HGNN integrates structural and functional connectivity by constructing meta-paths that capture both intra-modal (homo-meta-paths) and inter-modal (hetero-meta-paths) relationships. By defining four meta-paths (Φ1: Nf → Nf, Φ2: Nd → Nd, Φ3: Nf → Nd, Φ4: Nd → Nf), the model explicitly preserves the heterogeneous nature of dual-modal information while enabling cross-modal interaction learning.

### Mechanism 2
The heterogeneous graph pooling strategy automatically balances different types of meta-paths and prevents feature confusion after pooling. The pooling layer applies separate attention mechanisms to different node types, then summarizes the initial pooling matrices using a dense layer to maintain node-specific information while avoiding cross-type feature mixing.

### Mechanism 3
The heterogeneous graph augmentation method addresses sample imbalance by dynamically reconstructing functional connectivity while preserving structural connectivity as a semantic constraint. The augmentation generates new functional connectivity matrices through dynamic window analysis and subgraph frequency-based reconstruction, while keeping the structural connectivity fixed to maintain consistency.

## Architecture Onboarding

- **Component map**: Input (rs-fMRI + DTI) → Graph Construction (homo-meta-paths + hetero-meta-paths) → HG Augmentation (dynamic FC reconstruction) → Backbone (HAN layers → HG pooling layers → readout layers) → Classification (MLP)

- **Critical path**: Graph Construction → HG Augmentation → Backbone Processing → Classification. The graph construction and augmentation steps are critical because they define the input representation that all subsequent layers process.

- **Design tradeoffs**: Using heterogeneous graphs preserves multimodal information but increases model complexity; fixed structural connectivity in augmentation ensures consistency but may limit augmentation diversity; separate pooling for different node types prevents confusion but requires careful hyperparameter tuning.

- **Failure signatures**: Overfitting to specific meta-paths if attention mechanisms are not properly regularized; loss of important cross-modal relationships if pooling ratios are not balanced; insufficient augmentation if the dynamic reconstruction parameters are not properly tuned.

## Limitations
- The method requires both rs-fMRI and DTI data, limiting applicability when only one modality is available
- Heterogeneous graph construction relies heavily on quality of structure-function coupling calculations
- Augmentation method may generate synthetic samples that don't fully capture complexity of real MCI progression
