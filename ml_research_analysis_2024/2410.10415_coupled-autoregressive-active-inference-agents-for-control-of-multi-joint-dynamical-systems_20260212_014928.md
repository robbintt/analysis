# Coupled autoregressive active inference agents for control of multi-joint dynamical systems

## Quick Facts
- arXiv ID: 2410.10415
- Source URL: https://arxiv.org/abs/2410.10415
- Reference count: 31
- Primary result: CARX-EFE agents outperform uncoupled ARX-EFE agents on double mass-spring-damper systems in goal alignment and prediction error

## Executive Summary
This paper introduces Coupled ARX-EFE (CARX-EFE) agents for controlling multi-joint dynamical systems without prior knowledge of system dynamics. The approach couples multiple scalar autoregressive model-based active inference agents by sharing memory buffers, enabling explicit incorporation of correlations between system outputs. The method is validated on a double mass-spring-damper system, demonstrating superior performance in stabilization, prediction accuracy, and control efficiency compared to uncoupled agents.

## Method Summary
CARX-EFE agents extend traditional ARX-EFE by coupling multiple agents through shared memory buffers that contain historical data from interconnected system components. Each agent performs Bayesian parameter inference and control optimization by minimizing expected free energy over a finite time horizon. The coupling mechanism allows agents to explicitly model correlations between outputs of interconnected bodies, improving both prediction accuracy and control performance. The approach is specifically designed for mechanical systems with multiple interconnected bodies where component outputs are correlated.

## Key Results
- CARX-EFE agents achieve better stabilization of double mass-spring-damper systems compared to uncoupled ARX-EFE agents
- Prediction variance is significantly lower in coupled agents, indicating more accurate modeling of system dynamics
- Control strategies from CARX-EFE agents converge more rapidly to desired positions and maintain sustained stability around goal states

## Why This Works (Mechanism)
The coupling mechanism works by allowing agents to share historical data through memory buffers, creating explicit dependencies between the predictions and control actions of interconnected system components. This shared information enables each agent to account for the correlation structure between outputs, leading to more accurate system identification and better-informed control decisions. The Bayesian parameter inference framework naturally handles uncertainty while the expected free energy minimization ensures goal-directed behavior that adapts to the coupled system dynamics.

2. **Autoregressive Modeling (ARX)** - Why needed: Enables prediction of future states based on historical data without requiring explicit system dynamics. Quick check: Confirm prediction accuracy improves with increased memory buffer length.

3. **Bayesian Parameter Inference** - Why needed: Allows for principled uncertainty quantification and online learning of system parameters. Quick check: Ensure parameter estimates converge to true values as more data becomes available.

4. **Expected Free Energy Minimization** - Why needed: Balances exploration and exploitation while maintaining goal-directed behavior. Quick check: Verify agents achieve goals while minimizing control effort.

5. **Memory Buffer Sharing** - Why needed: Enables explicit modeling of correlations between interconnected system components. Quick check: Confirm coupling improves prediction accuracy over uncoupled baselines.

6. **Multi-agent Coordination** - Why needed: Facilitates distributed control of complex systems with multiple interconnected components. Quick check: Validate that coupled agents outperform independent agents on correlated systems.

## Architecture Onboarding

Component Map: Sensor Inputs -> ARX Models -> Bayesian Inference -> EFE Minimization -> Control Outputs -> System Response -> Shared Memory Buffers (between agents)

Critical Path: Sensor Data → Memory Buffer → ARX Prediction → Bayesian Update → EFE Optimization → Control Signal

Design Tradeoffs:
- Memory buffer length vs. computational complexity
- Coupling strength vs. agent independence
- Prediction horizon length vs. optimization tractability
- Model complexity vs. sample efficiency

Failure Signatures:
- Divergence of parameter estimates indicating poor system identification
- Oscillatory control signals suggesting insufficient coupling
- High prediction variance revealing model inadequacy
- Slow convergence pointing to suboptimal coupling configuration

- Computational overhead from memory buffer sharing is not quantified
- No theoretical guarantees provided for convergence and stability in coupled setting
