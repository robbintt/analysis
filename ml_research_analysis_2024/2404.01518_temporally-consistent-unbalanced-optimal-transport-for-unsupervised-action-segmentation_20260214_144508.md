# Temporally Consistent Unbalanced Optimal Transport for Unsupervised Action Segmentation

## Quick Facts
- arXiv ID: 2404.01518
- Source URL: https://arxiv.org/abs/2404.01518
- Authors: Ming Xu; Stephen Gould
- Reference count: 40
- Proposes ASOT, a post-processing method for unsupervised video action segmentation using optimal transport

## Executive Summary
This paper introduces a post-processing method called ASOT for unsupervised video action segmentation that leverages optimal transport theory. The method formulates the decoding of temporally consistent segmentations as a fused, unbalanced Gromov-Wasserstein optimal transport problem. This approach combines visual information from frame-action affinity matrices with temporal consistency constraints, while allowing for varying action frequencies across videos. The method is demonstrated to achieve state-of-the-art results on multiple benchmark datasets including Breakfast, 50-Salads, YouTube Instructions, and Desktop Assembly.

## Method Summary
The proposed ASOT method addresses unsupervised action segmentation by formulating it as an optimal transport problem. It fuses visual information from frame-action affinity matrices with temporal consistency through a Gromov-Wasserstein component. The unbalanced transport formulation is particularly important as it allows for varying action frequencies across different videos. The method is used to generate pseudo-labels for self-training in an unsupervised representation learning pipeline. ASOT is shown to be effective not only for unsupervised segmentation but also as post-processing for supervised methods.

## Key Results
- Achieves state-of-the-art results for unsupervised action segmentation on Breakfast, 50-Salads, YouTube Instructions, and Desktop Assembly datasets
- Demonstrates effectiveness as post-processing for supervised methods
- Provides comprehensive ablation studies on hyperparameters
- Includes implementation details and code availability

## Why This Works (Mechanism)
The method works by framing action segmentation as an optimal transport problem that balances visual consistency with temporal coherence. The fused approach combines frame-action affinities (visual information) with a Gromov-Wasserstein component that enforces temporal consistency. The unbalanced transport formulation is crucial as it accommodates the reality that action frequencies vary across different videos. This dual optimization allows the model to maintain both visual fidelity and temporal smoothness in the segmentation results.

  - Why needed: Enables comparison of frame and action representations in different spaces

  - Why needed: Enables unsupervised learning from noisy segmentation outputs

  - Why needed: Provides visual information for the transport formulation

## Architecture Onboarding

**Component Map**: Frame-Action Affinity Matrices -> ASOT Formulation -> Pseudo-labels -> Self-training

**Critical Path**: The core computational path involves computing affinity matrices from video frames, applying the ASOT formulation to generate consistent segmentations, and using these as pseudo-labels for self-training.

**Design Tradeoffs**: The method trades computational complexity for segmentation accuracy. The optimal transport formulation is computationally intensive but provides superior temporal consistency compared to simpler methods.

**Failure Signatures**: Poor affinity matrix quality leads to incorrect segmentations. The method may struggle with videos containing actions that are visually similar or with complex action distributions.
