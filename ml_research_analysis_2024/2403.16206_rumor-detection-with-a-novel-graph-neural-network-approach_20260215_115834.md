# Rumor Detection with a novel graph neural network approach

## Quick Facts
- arXiv ID: 2403.16206
- Source URL: https://arxiv.org/abs/2403.16206
- Reference count: 0
- Primary result: Joint learning of user correlation and information propagation patterns improves rumor detection accuracy

## Executive Summary
This paper introduces MODEL, a graph neural network approach for early rumor detection on social media. The method jointly learns representations from user correlation patterns (via user-tweet bipartite graphs) and information propagation structures (via tree structures) to improve detection accuracy. Evaluated on Twitter15 and Twitter16 datasets, MODEL outperforms state-of-the-art methods in both accuracy and F1 score. The approach also demonstrates robustness to adversarial attacks and effective performance in early detection scenarios, highlighting the importance of considering user coordination patterns alongside propagation structures.

## Method Summary
MODEL processes social media data by constructing a user-tweet bipartite graph to capture user coordination patterns and a tree structure to model information diffusion. The method uses graph neural networks (GCNs) to learn representations from both structures, which are then concatenated and fed into a classification layer. To handle scalability, the approach employs subgraph sampling focusing on k-hop neighboring nodes around source tweets. The model incorporates user profile features and tweet text embeddings, with GRU encoders processing textual content. Training uses 5-fold cross-validation with cross-entropy loss for classification.

## Key Results
- MODEL achieves superior accuracy and F1 scores compared to state-of-the-art rumor detection methods on Twitter15 and Twitter16 datasets
- The approach demonstrates robustness to adversarial attacks, requiring higher attack costs to fool the model
- Early detection performance is effective, with the model successfully identifying rumors in early stages of propagation
- Joint learning of user correlation and information propagation patterns provides complementary signals that enhance detection accuracy

## Limitations
- Evaluation is limited to two specific Twitter datasets, which may not generalize to other platforms or rumor types
- The preprocessing requirements, particularly the need to collect user profile information via Twitter API, could introduce data quality variations that affect reproducibility
- The subgraph sampling approach may miss broader coordination patterns that extend beyond k-hop neighborhoods
