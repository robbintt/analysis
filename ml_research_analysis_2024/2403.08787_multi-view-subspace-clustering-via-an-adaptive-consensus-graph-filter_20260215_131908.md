# Multi-view Subspace Clustering via An Adaptive Consensus Graph Filter

## Quick Facts

## Executive Summary
This paper introduces MVSC2GF, a novel multi-view subspace clustering method that leverages an adaptive consensus graph filter to smooth data features across different views. The key innovation is the design of a regularizer for the reconstruction coefficient matrix in each view, which is derived from a consensus reconstruction coefficient matrix that consolidates consistent information from different views. The method iteratively learns the consensus reconstruction coefficient matrix and the reconstruction coefficient matrix in each view through an optimization algorithm. Extensive experiments on seven benchmark multi-view datasets demonstrate that MVSC2GF outperforms several state-of-the-art methods, achieving significant improvements in clustering accuracy.

## Method Summary
MVSC2GF employs a consensus graph filter to smooth data features across multiple views, aiming to capture consistent information from different perspectives. The method introduces a regularizer for the reconstruction coefficient matrix in each view, which is derived from a consensus reconstruction coefficient matrix. This consensus matrix consolidates consistent information from different views, enabling the method to leverage the complementary information across views effectively. The authors develop an optimization algorithm that iteratively learns the consensus reconstruction coefficient matrix and the reconstruction coefficient matrix in each view. The algorithm alternates between updating the consensus matrix and the view-specific matrices until convergence. The final clustering results are obtained by applying spectral clustering on the learned consensus matrix.
