# Multi-level Conflict-Aware Network for Multi-modal Sentiment Analysis

## Quick Facts
- arXiv ID: 2502.09675
- Source URL: https://arxiv.org/abs/2502.09675
- Authors: Yubo Gao; Haotian Wu; Lei Zhang
- Reference count: 0
- Key outcome: MCAN achieves state-of-the-art performance on CMU-MOSI and CMU-MOSEI datasets with accuracy scores of 84.5% (Acc2) and 85.8% (Acc2) respectively

## Executive Summary
This paper introduces MCAN, a multi-level conflict-aware network designed to address limitations in existing multimodal sentiment analysis approaches by explicitly modeling both alignment and conflict between modalities. The proposed architecture consists of two main branches: a main branch that progressively fuses and aligns cross-modal representations while segregating conflict constituents, and a conflict modeling branch that further processes these conflicts through hierarchical modules. The method achieves state-of-the-art performance on CMU-MOSI and CMU-MOSEI datasets without relying on unstable generated labels, instead encouraging inconsistent predictions through its architecture design.

## Method Summary
MCAN employs a hierarchical architecture with two parallel branches operating at micro and macro levels. The main branch uses Micro-MSIN and Macro-MSIN modules to progressively fuse cross-modal representations while segregating conflict constituents. Simultaneously, the conflict modeling branch employs Micro-CACA and Macro-CACA modules to further model conflicts. The network encourages inconsistent predictions rather than depending on generated labels, addressing stability issues in conflict detection. The architecture progressively processes information through multiple levels, allowing for nuanced conflict modeling between different modalities in sentiment analysis tasks.

## Key Results
- Achieves state-of-the-art performance on CMU-MOSI: 84.5% accuracy (Acc2), 43.1% (Acc7), and 0.811 Pearson correlation
- Achieves state-of-the-art performance on CMU-MOSEI: 85.8% accuracy (Acc2), 51.6% (Acc7), and 0.798 Pearson correlation
- Demonstrates effective conflict modeling without dependence on unstable generated labels

## Why This Works (Mechanism)
The hierarchical conflict modeling approach works by simultaneously processing alignment and conflict at multiple granularities. The micro-level modules capture fine-grained conflicts between individual features, while macro-level modules identify broader pattern-level disagreements. By encouraging inconsistent predictions rather than relying on generated labels, the system avoids the instability associated with label-based conflict detection methods. The dual-branch architecture allows for progressive refinement of both aligned and conflicted representations, leading to more robust multimodal sentiment analysis.

## Architecture Onboarding

Component Map: Input -> Feature Extraction -> Micro-MSIN/Micro-CACA -> Macro-MSIN/Macro-CACA -> Prediction

Critical Path: The main branch (Micro-MSIN -> Macro-MSIN) and conflict branch (Micro-CACA -> Macro-CACA) operate in parallel, with their outputs combined for final prediction. The critical path involves progressive refinement through both alignment and conflict modeling at each hierarchical level.

Design Tradeoffs: The architecture trades computational complexity for improved conflict modeling capability. By maintaining parallel branches for alignment and conflict, it captures more nuanced relationships but requires more parameters and computational resources compared to simpler fusion approaches.

Failure Signatures: The system may struggle when conflicts are subtle or when one modality is significantly more reliable than others. Performance could degrade if the hierarchical modules fail to properly segregate aligned from conflicted information.
