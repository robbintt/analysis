# Multi-Agent Reinforcement Learning with Communication-Constrained Priors

## Quick Facts
- arXiv ID: 2512.03528
- Source URL: https://arxiv.org/abs/2512.03528
- Authors: Guang Yang; Tianpei Yang; Jingwen Qiao; Yanqing Wu; Jing Huo; Xingguo Chen; Yang Gao
- Reference count: 38
- This paper addresses multi-agent reinforcement learning in communication-constrained environments by modeling communication-constrained priors and using a dual mutual information estimator to enhance reliable messages and mitigate corrupted messages

## Executive Summary
This paper tackles the challenge of multi-agent reinforcement learning (MARL) in communication-constrained environments where messages can be lossy or lossless. The authors propose a novel framework that models communication-constrained priors to distinguish between reliable and corrupted messages, using a dual mutual information estimator (Du-MIE) to quantify their impact on agent behavior. The method integrates this information into the reward function to enhance the positive influence of reliable messages while mitigating the negative effects of corrupted communication. The CC-MADDPG algorithm demonstrates superior performance compared to existing baselines across multiple benchmark environments, maintaining robust performance even under severe communication constraints.

## Method Summary
The proposed framework models communication-constrained priors by distinguishing between lossy and lossless messages in multi-agent systems. A dual mutual information estimator (Du-MIE) is employed to quantify the impact of different message types on agent behavior. This estimation is incorporated into the reward function to amplify the influence of reliable messages while reducing the negative impact of corrupted ones. The approach is implemented within the MADDPG framework, creating CC-MADDPG. The method addresses the challenge of partial observability and communication constraints by learning which messages to trust and how to adjust behavior accordingly, enabling more effective coordination among agents even when communication is unreliable.

## Key Results
- CC-MADDPG achieves average rewards of up to 138.0 in challenging communication-constrained scenarios
- Outperforms FC-MADDPG (1.5) and Dropout-MADDPG (68.7) under heavy communication constraints
- Demonstrates robust performance across multiple benchmark environments with lossy communication

## Why This Works (Mechanism)
The framework works by explicitly modeling the reliability of communication messages through mutual information estimation. By distinguishing between lossy and lossless messages, the system can prioritize information from reliable sources while being skeptical of potentially corrupted messages. The dual mutual information estimator provides a quantitative measure of message reliability that can be directly incorporated into the reward structure, allowing agents to learn which communication channels to trust and how to adjust their behavior accordingly. This selective attention to reliable information enables more effective coordination despite communication constraints.

## Architecture Onboarding

### Component Map
Observation -> Du-MIE Estimator -> Reward Function -> Actor-Critic Networks -> Action Selection -> Environment

### Critical Path
The critical path involves receiving observations, estimating message reliability through Du-MIE, shaping rewards based on reliability, and updating actor-critic networks to select actions that maximize the reliability-weighted rewards. This creates a feedback loop where communication quality directly influences learning outcomes.

### Design Tradeoffs
The framework trades computational overhead for improved communication reliability modeling. The Du-MIE estimator adds complexity but enables more nuanced handling of communication constraints. The approach assumes a specific communication model and may not generalize to all network topologies or dynamic communication conditions.

### Failure Signatures
Potential failures include overestimation of message reliability leading to poor decisions, computational bottlenecks from the Du-MIE estimator in large-scale systems, and inability to adapt to changing communication patterns. The method may also struggle with highly dynamic environments where communication reliability varies rapidly.
