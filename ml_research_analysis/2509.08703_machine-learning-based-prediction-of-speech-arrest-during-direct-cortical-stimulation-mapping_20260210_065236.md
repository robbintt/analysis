# Machine Learning-Based Prediction of Speech Arrest During Direct Cortical Stimulation Mapping

## Quick Facts
- arXiv ID: 2509.08703
- Source URL: https://arxiv.org/abs/2509.08703
- Authors: Nikasadat Emami; Amirhossein Khalilian-Gourtani; Jianghao Qian; Antoine Ratouchniak; Xupeng Chen; Yao Wang; Adeen Flinker
- Reference count: 16
- The paper presents a machine learning framework for predicting speech-critical cortical regions from ECoG data, aiming to reduce reliance on invasive Electrical Stimulation Mapping (ESM) during presurgical evaluation.

## Executive Summary
This study introduces a machine learning framework to predict speech-critical cortical regions from ECoG data, potentially reducing the need for invasive Electrical Stimulation Mapping (ESM) during presurgical evaluation. The core method combines anatomical region encoding with functional connectivity features, using a two-stage classification pipeline: trial-level predictions from an RBF-kernel SVM followed by histogram-based aggregation with an MLP. Models incorporating both spatial and network features achieved performance nearly identical to full-feature models, with the best model achieving ROC-AUC of 0.87 and PR-AUC of 0.57 in held-out participants. High-Gamma signals improved performance over raw signals, and MLP-based aggregation outperformed simple averaging.

## Method Summary
The study used ECoG data from 5 patients with intractable epilepsy, who performed a speech production task while implanted with subdural grids. The pipeline involved preprocessing with common average reference and High-Gamma extraction (70-150 Hz), followed by feature extraction including anatomical region encoding (DKT Atlas), functional connectivity metrics (strength, eigenvector centrality, clustering coefficient), and local signal features (NMF correlations, mean activity). Trial-level predictions were made using RBF-kernel SVM, then aggregated using histogram-based MLP that incorporated region information. Models were evaluated using cross-validation and leave-one-participant-out validation, with performance compared against ESM-identified critical electrodes.

## Key Results
- Best model achieved ROC-AUC of 0.87 and PR-AUC of 0.57 in leave-one-participant-out validation
- High-Gamma signals consistently outperformed raw signals across all evaluation metrics
- Anatomical region + connectivity features matched performance of full feature set (PR-CV 0.55, ROC-CV 0.85)
- Histogram-based MLP aggregation improved performance over simple averaging (PR-LOO: +0.14)

## Why This Works (Mechanism)

### Mechanism 1
- **Claim:** High-Gamma band (70–150 Hz) signals provide more informative representations of task-relevant neural activity than raw broadband ECoG signals for identifying speech-critical cortex.
- **Mechanism:** Raw ECoG is bandpass filtered to 70–150 Hz, then Hilbert transform extracts the analytic amplitude envelope. This isolates high-frequency neural dynamics associated with local population firing, filtering out lower-frequency components and artifacts that may add noise.
- **Core assumption:** Speech-critical regions exhibit distinguishable high-frequency activation patterns during speech tasks that generalize across individuals.
- **Evidence anchors:**
  - [abstract] "High-Gamma signals outperformed raw signals"
  - [section IV.D, Table IV] High-Gamma achieved PR-CV 0.55 vs 0.51 for raw; ROC-CV 0.85 vs 0.83, with consistent gains in LOO validation
  - [corpus] Weak direct evidence; related papers focus on other modalities. Signal preprocessing choices remain domain-specific.
- **Break condition:** If critical language information were predominantly carried in lower-frequency bands (theta, alpha) or if High-Gamma responses were highly variable across patients, this filtering would discard discriminative signal.

### Mechanism 2
- **Claim:** Anatomical region encoding combined with functional connectivity features captures complementary spatial and network-level information, achieving performance equivalent to the full feature set.
- **Mechanism:** Each electrode is assigned to 1 of 26 anatomical regions (DKT Atlas, one-hot encoded). Separately, trial-wise Pearson correlation matrices between all electrode pairs yield graph metrics: strength, eigenvector centrality, clustering coefficient. These are concatenated as input features.
- **Core assumption:** Speech-critical electrodes occupy both distinctive anatomical locations AND distinctive network positions (high centrality, specific connectivity patterns) that are consistent enough across individuals to be learned.
- **Evidence anchors:**
  - [abstract] "models combining region and connectivity features matched the performance of the full feature set"
  - [section V.A, Table I] Region + Connectivity: PR-CV 0.55, ROC-CV 0.85—identical to all features. Wilcoxon test confirmed significant improvement over either alone.
  - [corpus] Hsieh et al. (cited in paper) found language-critical sites act as connectors between subnetworks, supporting connectivity relevance.
- **Break condition:** If critical regions have highly variable anatomical distribution or network roles across patients (high inter-subject variability), fixed region encodings and subject-specific connectivity patterns would not generalize.

### Mechanism 3
- **Claim:** Aggregating trial-level predictions via histogram distributions and region-aware MLP captures non-linear patterns in prediction score distributions that simple averaging misses.
- **Mechanism:** Trial-level classifier outputs probability scores per electrode-trial. Instead of averaging, scores are binned into a 10-bin histogram [0,1], concatenated with 26-dim region encoding, and processed by 2-layer MLP (64→32 units, ReLU, Sigmoid output). Focal loss handles ~1:5 class imbalance.
- **Core assumption:** The shape of the trial-level score distribution—not just central tendency—encodes information about electrode criticality, and this relationship is non-linear and region-dependent.
- **Evidence anchors:**
  - [section IV.D] "there may be a nonlinear relationship between the shape of the score distribution... and the true likelihood"
  - [section V.B, Table III] MLP (Hist + Region): PR-LOO 0.55, ROC-LOO 0.84 vs averaging PR-LOO 0.41, ROC-LOO 0.76
  - [corpus] No direct corpus evidence for this specific aggregation approach; appears novel to this work.
- **Break condition:** If trial-level predictions are highly consistent (low variance) across trials for each electrode, the histogram distribution collapses to a narrow peak, providing no additional information beyond the mean.

## Foundational Learning

- **Concept:** Common Average Reference (CAR) and signal normalization
  - **Why needed here:** ECoG signals have shared noise (line noise, movement artifacts). CAR subtracts the mean across all electrodes from each channel, reducing common-mode noise before feature extraction.
  - **Quick check question:** If you skipped CAR and used a single reference electrode instead, what type of artifact would likely dominate your signals?

- **Concept:** High-Gamma analytic amplitude via Hilbert transform
  - **Why needed here:** The paper's best results use High-Gamma power, not raw voltage. Hilbert transform extracts the instantaneous amplitude envelope from bandpass-filtered signals.
  - **Quick check question:** Given a 70–150 Hz bandpass signal, what does the Hilbert envelope represent physiologically?

- **Concept:** Graph metrics (strength, eigenvector centrality, clustering coefficient)
  - **Why needed here:** Functional connectivity is converted to interpretable network features. Strength = total connection weight; EC = influence through well-connected neighbors; CC = local clustering density.
  - **Quick check question:** An electrode with high eigenvector centrality but low strength—what might this indicate about its network role?

## Architecture Onboarding

- **Component map:** Preprocessing (CAR → epoch → baseline → High-Gamma) -> Feature extraction (NMF, Mean Activity, Region, Connectivity) -> Trial-level SVM -> Histogram-MLP aggregation
- **Critical path:** High-Gamma extraction → Region + Connectivity features → RBF-SVM trial predictions → Histogram-MLP aggregation. Errors in early preprocessing (incorrect CAR, wrong frequency band) cascade through all stages.
- **Design tradeoffs:**
  - **Feature completeness vs. parsimony:** Region + Connectivity alone matches full features—simpler model, easier interpretation, reduced overfitting risk
  - **Averaging vs. MLP aggregation:** MLP adds complexity but improves LOO generalization (PR-LOO: +0.14); worth it for cross-subject deployment
  - **Linear vs. RBF-SVM:** RBF improves ROC-LOO from 0.84 to 0.87, but Linear SVM is faster and more interpretable
  - **Raw vs. High-Gamma:** High-Gamma requires additional processing but consistently improves performance
- **Failure signatures:**
  - **Large CV-to-LOO gap** (e.g., ROC-CV 0.85 vs ROC-LOO 0.77 baseline): indicates overfitting to subject-specific patterns; address with regularization, feature reduction
  - **High ROC-AUC with low PR-AUC** (0.87 vs 0.57 best model): expected under 17% positive class prevalence; indicates precision challenges at all thresholds
  - **Near-uniform histogram distributions**: trial-level predictions have high variance or classifier is underfitting; check feature quality and classifier hyperparameters
- **First 3 experiments:**
  1. **Feature ablation replication:** Train with Region-only, Connectivity-only, Region+Connectivity, All-features using Linear SVM + averaging. Confirm Region+Connectivity ≈ All-features.
  2. **Aggregation comparison:** Swap averaging for histogram-only MLP, then histogram+region MLP. Expect stepwise improvement, especially in LOO.
  3. **Signal type swap:** Replace High-Gamma with raw CAR-referenced signals, keeping all else constant. Expect performance drop across all metrics.

## Open Questions the Paper Calls Out

### Open Question 1
- Question: Can the proposed machine learning framework be validated in a prospective clinical setting to guide surgical resection decisions?
- Basis in paper: [explicit] The authors state their framework "offers a promising direction towards augmenting traditional ESM procedures" but the current study is retrospective.
- Why unresolved: The current results are based on analyzing historical ECoG data and correlating it with past ESM results, rather than using the model to influence or guide actual surgical outcomes in real-time.
- What evidence would resolve it: A prospective clinical trial where surgeons use the model's predictions to plan resections, with post-operative language outcomes serving as the ground truth.

### Open Question 2
- Question: How can the model's predictive power for the minority class (critical electrodes) be improved to achieve clinically sufficient precision?
- Basis in paper: [explicit] The authors acknowledge that "modest AUC-PR scores are because of the class imbalance" (17% critical), noting it is "very hard to achieve both high recall and high precision."
- Why unresolved: The severe class imbalance inherent to functional mapping (few sites are truly critical) limits the Positive Predictive Value (Precision), meaning a high number of false positives could occur if used as a primary screening tool.
- What evidence would resolve it: Implementation of advanced imbalanced learning techniques (e.g., synthetic oversampling, cost-sensitive learning) or aggregation of multi-center data to improve the PR-AUC beyond the reported 0.57.

### Open Question 3
- Question: Does the predictive utility of "Region + Connectivity" features generalize to stereo-electroencephalography (SEEG) recordings with sparse spatial sampling?
- Basis in paper: [inferred] The study relies exclusively on subdural ECoG grids (standard 8 × 8 macro contacts) which provide dense, regular surface coverage.
- Why unresolved: SEEG uses linear depth electrodes with sparser, irregular sampling, which may disrupt the graph-theoretical metrics (clustering coefficient, eigenvector centrality) that depend on the dense neighborhood relationships present in grid data.
- What evidence would resolve it: Evaluating the model on a cohort of patients implanted with SEEG electrodes to verify if connectivity features retain their predictive power.

### Open Question 4
- Question: Do local neural activity features (NMF, Mean Activity) fail to improve model performance due to inherent biological redundancy or limitations in the current feature extraction method?
- Basis in paper: [inferred] The results showed that models using only "Region + Connectivity" matched the performance of "All Features," implying that the extracted local signal features (NMF, Mean Activity) provided negligible additional information.
- Why unresolved: It is unclear if local activity is truly subservient to network properties for this task, or if the specific implementation of NMF/averaging simply failed to capture the relevant local dynamics (e.g., phase information or temporal dispersion).
- What evidence would resolve it: A comparison using end-to-end deep learning models (e.g., CNNs) that can learn non-linear local features directly from the raw signal, testing if this improves performance over the manual NMF/statistical approach.

## Limitations

- Limited dataset size (5 participants, 151 electrodes) constrains generalization claims; model performance in LOO validation, while improved over baselines, may not hold in larger, more diverse populations
- The anatomical encoding assumes consistent regional boundaries across individuals via DKT Atlas, but individual anatomical variability could reduce cross-subject transfer
- High-Gamma preprocessing choices (70–150 Hz bandpass, Hilbert transform) are standard but not validated against alternatives; other frequency bands or preprocessing methods might capture complementary information

## Confidence

- **High confidence:** High-Gamma signals outperform raw signals (supported by consistent cross-validation and LOO results)
- **High confidence:** Anatomical + connectivity features match full-feature performance (statistically validated via Wilcoxon test)
- **Medium confidence:** MLP aggregation provides meaningful improvement (novel approach with promising but not extensively validated results)
- **Medium confidence:** Cross-participant generalization potential (limited by small sample size despite LOO validation)

## Next Checks
