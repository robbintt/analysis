# Communication Enables Cooperation in LLM Agents: A Comparison with Curriculum-Based Approaches

## Quick Facts
- arXiv ID: 2510.05748
- Source URL: https://arxiv.org/abs/2510.05748
- Reference count: 40
- One-word communication channel increased cooperation from 0% to 48.3% in 4-player Stag Hunt game

## Executive Summary
This paper investigates two approaches for eliciting cooperation in multi-agent LLM systems: direct communication and curriculum learning. In a 4-player Stag Hunt game, a one-word "cheap talk" channel dramatically increased cooperation from 0% to 48.3%, demonstrating communication as a robust coordination mechanism. In contrast, curriculum learning proved highly sensitive to design choices - a pedagogical sequence through progressively complex games reduced agent payoffs by 27.4% in an Iterated Public Goods Game with Punishment. Qualitative analysis revealed that curricula emphasizing defection-equilibrium games induced "learned pessimism," where agents overgeneralized lessons from short-horizon games to contexts where cooperation was viable.

## Method Summary
The study employed a controlled experimental framework comparing communication and curriculum learning approaches in multi-agent coordination problems. Agents were LLM-based players in sequential games, with the communication condition allowing pre-round one-word exchanges. The curriculum condition exposed agents to progressively complex games (Coordination → Prisoner's Dilemma → Stag Hunt → Public Goods with Punishment). Performance was measured through cooperation rates, aggregate payoffs, and qualitative analysis of agent decision patterns across game sequences.

## Key Results
- One-word communication channel increased cooperation from 0% to 48.3% in 4-player Stag Hunt game
- Curriculum learning reduced payoffs by 27.4% in Iterated Public Goods Game with Punishment
- Curricula emphasizing defection-equilibrium games induced "learned pessimism" where agents overgeneralized lessons to cooperative contexts

## Why This Works (Mechanism)
The communication channel provides agents with a coordination signal that helps them align expectations and establish common knowledge about intended strategies. This cheap talk mechanism overcomes the information asymmetry that prevents cooperation in pure strategy games. For curriculum learning, the mechanism involves experiential learning through progressive game exposure, but the study found this approach can backfire when the sequence emphasizes defection-equilibrium games, leading agents to develop pessimistic strategic priors that persist across contexts.

## Architecture Onboarding

**Component Map**
LLM agents -> Game environment -> Communication channel OR Curriculum sequence -> Performance metrics

**Critical Path**
Game state encoding -> Agent strategy selection -> (Optional) pre-round communication -> Action execution -> Payoff calculation -> Strategy update (for curriculum)

**Design Tradeoffs**
Communication: Simple, robust, but may not generalize to no-communication settings
Curriculum: Potentially more transferrable, but highly sensitive to sequencing and may induce negative priors

**Failure Signatures**
Communication failure: Coordination breakdown, persistent disagreement on focal points
Curriculum failure: Learned pessimism, overgeneralization from defection-equilibrium games, performance degradation

**First Experiments**
1. Test communication effect across 3-5 structurally diverse coordination games
2. Systematically vary curriculum sequence parameters to identify optimal progression
3. Implement interpretability analysis of agent decision processes before/after curriculum training

## Limitations
- Experimental scope limited to single game variant and specific curriculum sequence
- 48.3% cooperation rate may not generalize to games with different payoff structures
- "Learned pessimism" interpretation lacks direct evidence of agent strategic representations

## Confidence
- Communication advantage in Stag Hunt: **High confidence**
- Curriculum learning generally inferior: **Medium confidence**
- "Learned pessimism" as explanation: **Low confidence**

## Next Checks
1. Replicate Stag Hunt communication experiment across 3-5 structurally diverse coordination games (e.g., Chicken, Battle of the Sexes, Prisoner's Dilemma) to test generalizability
2. Systematically vary curriculum sequence parameters (game ordering, duration, complexity progression) to identify whether negative outcomes are design-specific
3. Implement interpretability analysis of agent decision processes before and after curriculum training to directly observe whether strategic pessimism emerges as hypothesized
