# Leveraging Gait Patterns as Biomarkers: An attention-guided Deep Multiple Instance Learning Network for Scoliosis Classification

## Quick Facts
- **arXiv ID:** 2504.03894
- **Source URL:** https://arxiv.org/abs/2504.03894
- **Reference count:** 0
- **Primary result:** Proposed Gait-MIL achieves 84.8% accuracy, 99.0% sensitivity, and 79.6% specificity for scoliosis classification from gait patterns

## Executive Summary
This study presents Gait-MIL, an attention-guided Deep Multiple Instance Learning framework for detecting scoliosis using gait patterns. The method addresses early scoliosis detection challenges by leveraging video-based gait analysis as a non-invasive alternative to traditional X-ray screening. Gait-MIL clusters gait frames into phases and uses attention mechanisms to focus on informative frames and phases for classification. Evaluated on the Scoliosis1K dataset, the approach achieves 84.8% accuracy, 99.0% sensitivity, and 79.6% specificity, significantly improving classification of challenging Neutral cases compared to prior methods. It also demonstrates robustness in imbalanced data scenarios, making it suitable for large-scale screening.

## Method Summary
Gait-MIL addresses scoliosis classification through a multi-stage framework that processes gait video sequences. The approach first extracts spatial and temporal features from each gait frame using a feature extractor. These features are then clustered into temporal phases using a clustering module. Attention mechanisms operate at two levels: phase attention to identify informative gait phases, and frame attention to select key frames within each phase. The attention-guided features are then fed into a MIL-based classifier that aggregates information across instances to make the final classification decision. This hierarchical attention design enables the model to focus on the most relevant gait patterns while maintaining computational efficiency.

## Key Results
- Achieved 84.8% accuracy, 99.0% sensitivity, and 79.6% specificity on Scoliosis1K dataset
- Outperformed baseline methods in classifying Neutral cases, addressing a key limitation of existing approaches
- Demonstrated robustness to imbalanced datasets, crucial for real-world screening applications

## Why This Works (Mechanism)
The framework leverages the natural temporal structure of gait patterns and the inherent variability in scoliosis presentation. By clustering frames into phases, the model captures the cyclical nature of walking while reducing temporal redundancy. The dual attention mechanism allows the network to adaptively focus on both the most informative phases of the gait cycle and the most discriminative frames within those phases. This selective attention helps the model identify subtle gait abnormalities associated with scoliosis while ignoring irrelevant variations. The MIL formulation is particularly suited for this task because it can handle the uncertainty in which specific frames or phases are most diagnostic, treating the entire sequence as a bag of instances.

## Architecture Onboarding

**Component Map:** Video frames -> Feature Extractor -> Frame Clustering -> Phase Attention + Frame Attention -> MIL Classifier -> Classification

**Critical Path:** The attention-guided feature aggregation represents the critical path, where phase attention selects relevant gait phases and frame attention selects key frames, feeding into the MIL classifier for final decision-making.

**Design Tradeoffs:** The framework balances model complexity with interpretability by using attention mechanisms that provide insight into which gait phases and frames drive classification decisions. The clustering approach trades some temporal precision for computational efficiency and robustness to individual gait variations.

**Failure Signatures:** Poor clustering quality could lead to mixing informative and uninformative frames, diluting discriminative features. Ineffective attention mechanisms might focus on irrelevant gait patterns or miss subtle scoliosis indicators. MIL formulation issues could result in aggregation strategies that don't capture the true diagnostic information distributed across the gait sequence.

**First Experiments:** 1) Evaluate feature extractor performance on individual frame classification before temporal modeling. 2) Test attention mechanism effectiveness by comparing attention-weighted features against average pooling. 3) Validate clustering quality by measuring intra-cluster similarity and inter-cluster differences.
