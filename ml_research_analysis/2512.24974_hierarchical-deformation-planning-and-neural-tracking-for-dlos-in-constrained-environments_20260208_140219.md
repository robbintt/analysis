# Hierarchical Deformation Planning and Neural Tracking for DLOs in Constrained Environments

## Quick Facts
- arXiv ID: 2512.24974
- Source URL: https://arxiv.org/abs/2512.24974
- Reference count: 0
- Primary result: Combines hierarchical deformation planning with neural tracking to manipulate DLOs in constrained environments with superior success rate, accuracy, and efficiency compared to baselines

## Executive Summary
This paper addresses the challenge of manipulating deformable linear objects (DLOs) in cluttered environments by proposing a hierarchical framework that combines global homotopic path planning with local neural model predictive control (MPC) tracking. The approach first generates spatial path sets satisfying homotopy constraints for DLO keypoints, then synthesizes an optimal temporal deformation sequence within these corridors. During execution, a data-driven deformation model enables accurate closed-loop tracking. The framework was validated on 200 simulation trials and 5 physical robot experiments, demonstrating improved performance over state-of-the-art baselines.

## Method Summary
The framework employs a hierarchical decomposition strategy: first generating homotopically distinct path sets for DLO keypoints in the spatial domain, then using path-set-guided optimization to synthesize optimal temporal deformation sequences. A neural model predictive control approach with a data-driven deformation model tracks the planned sequence during execution. The method assumes reliable keypoint detection and DLO state estimation, with training data covering the operational deformation space. The planner restricts optimization to pre-computed homotopy classes to reduce complexity, while the neural MPC provides real-time correction for tracking errors.

## Key Results
- Achieved superior success rate compared to state-of-the-art baselines in both simulation (200 trials) and physical experiments (5 trials)
- Demonstrated accurate deformation tracking through neural MPC with learned deformation dynamics
- Showed computational efficiency by restricting optimization to valid homotopy classes
- Validated framework effectiveness in constrained environments with obstacles

## Why This Works (Mechanism)

### Mechanism 1
- Claim: Hierarchical decomposition separates global homotopic path planning from local temporal optimization, reducing the search space complexity for DLO manipulation in cluttered environments
- Mechanism: The framework first discretizes the continuous deformation space into homotopically distinct path classes for DLO keypoints. By constraining subsequent optimization within these pre-computed spatial corridors, the planner avoids exploring infeasible configurations and focuses computational resources on temporally optimal sequences within valid homotopy classes
- Core assumption: DLO keypoints can be reliably identified and their homotopy class membership remains stable during manipulation
- Evidence anchors: [abstract]: "generating a spatial path set that satisfies homotopic constraints for DLO keypoints"
- Break condition: When obstacles create ambiguous homotopy boundaries or when keypoint perception fails due to occlusion

### Mechanism 2
- Claim: Path-set-guided optimization improves temporal deformation synthesis by restricting the search space to spatially feasible corridors
- Mechanism: Once homotopic path sets are established, the optimizer searches for optimal temporal sequences only within these constrained regions. This reduces the high-dimensional continuous deformation problem to a tractable optimization over time-parameterized paths
- Core assumption: The optimal deformation lies within one of the pre-computed homotopy classes; no cross-homotopy transitions are needed during execution
- Evidence anchors: [abstract]: "path-set-guided optimization method is applied to synthesize an optimal temporal deformation sequence"
- Break condition: When dynamic obstacles or execution errors require real-time homotopy class switching

### Mechanism 3
- Claim: Neural model predictive control with learned deformation dynamics enables accurate closed-loop tracking that compensates for model inaccuracies
- Mechanism: A data-driven model learns the mapping from robot actions to DLO state changes. The MPC controller uses this model in a receding-horizon framework, continuously re-optimizing control inputs based on observed state deviations
- Core assumption: The training data sufficiently covers the deformation distribution encountered during execution
- Evidence anchors: [abstract]: "neural model predictive control approach, leveraging a data-driven deformation model, is designed to accurately track the planned DLO deformation sequence"
- Break condition: When out-of-distribution deformations cause prediction errors that accumulate faster than MPC can correct

## Foundational Learning

- Concept: **Homotopy Classes and Topological Path Planning**
  - Why needed here: The spatial planner relies on homotopic constraints to discretize the path space. Understanding how paths relate through continuous deformation around obstacles is essential for implementing the path set generator
  - Quick check question: Given two points and three obstacles, can you sketch the distinct homotopy classes of paths connecting them?

- Concept: **Model Predictive Control (MPC)**
  - Why needed here: The tracking layer uses neural MPC for closed-loop control. Understanding receding-horizon optimization, state prediction, and constraint handling is critical for implementation
  - Quick check question: What happens to MPC performance when the prediction model has systematic bias?

- Concept: **DLO State Representation (Keypoints vs. Continuous)**
  - Why needed here: The paper uses keypoints for planning but DLOs are continuous objects. Understanding the discretization tradeoffs affects both planning fidelity and tracking accuracy
  - Quick check question: How does keypoint density affect the minimum curvature radius a planned path can represent?

## Architecture Onboarding

- Component map:
  - Spatial Path Generator -> Temporal Optimizer -> Data-Driven Deformation Model -> Neural MPC Tracker -> State Estimator

- Critical path:
  1. Perception → obstacle map + initial DLO state
  2. Path Generator → candidate homotopic path sets
  3. Temporal Optimizer → optimal deformation sequence selection
  4. Neural MPC → real-time control commands
  5. State Estimator → feedback for closed-loop correction

- Design tradeoffs:
  - Path set cardinality vs. optimization time: More homotopy classes increase robustness but slow planning
  - MPC horizon length vs. real-time feasibility: Longer horizons improve tracking but increase latency
  - Keypoint density vs. perception reliability: More points improve resolution but complicate tracking
  - Assumption: Training data distribution covers operational regime

- Failure signatures:
  - Planner produces collision trajectories → obstacle map resolution insufficient or homotopy computation error
  - MPC tracking diverges → model prediction error exceeds correction capacity
  - Sudden tracking failure mid-execution → occlusion-induced state estimation failure
  - Optimizer timeout → path set too large or horizon too long

- First 3 experiments:
  1. Validate path set generation: Single DLO, 2-3 static obstacles, verify all homotopy classes correctly identified; measure planning time vs. path count
  2. Test MPC tracking accuracy: Execute planned deformations without obstacles; quantify prediction error vs. horizon length
  3. Full system integration: Physical robot, multi-obstacle environment; measure success rate, tracking error, and failure modes vs. baselines

## Open Questions the Paper Calls Out

### Open Question 1
- Question: Can the framework effectively operate in dynamic environments where obstacles move or change shape during the manipulation task?
- Basis in paper: [inferred] The abstract describes generating a "spatial path set" based on "homotopic constraints," which typically relies on a static environment map to define valid deformation corridors
- Why unresolved: Re-calculating the global path set and homotopy classes in real-time for moving obstacles introduces significant computational overhead that may exceed the control loop limits
- What evidence would resolve it: Successful simulation trials where obstacles change position after the initial spatial path set is generated

### Open Question 2
- Question: How sensitive is the data-driven deformation model to significant variations in DLO material properties, such as extreme stiffness or high plasticity?
- Basis in paper: [inferred] The method relies on a "data-driven deformation model" for neural tracking, suggesting performance is tied to the training distribution
- Why unresolved: Neural network approximators often struggle to extrapolate to physical dynamics outside the training data, potentially leading to tracking failure if the DLO behaves unexpectedly
- What evidence would resolve it: Ablation studies showing success rates across DLOs with widely varying Young's moduli or bending limits using the same trained model

### Open Question 3
- Question: Does the homotopy-based planning approach extend to tasks requiring topological changes, such as knot tying or self-interaction?
- Basis in paper: [inferred] The paper focuses on satisfying "homotopic constraints" to navigate *around* obstacles, which inherently preserves the topology of the DLO relative to its environment
- Why unresolved: Manipulation tasks like knotting require intentionally altering the DLO's topological state, which might be restricted by a planner designed strictly to maintain homotopic consistency
- What evidence would resolve it: Demonstration of the framework planning and executing a knot-tying task involving self-collision

## Limitations
- Framework assumes static obstacles and may struggle with dynamic environments requiring real-time homotopy class updates
- Neural MPC performance depends on training data coverage, with potential tracking failure on out-of-distribution deformations
- Method may not extend to tasks requiring intentional topological changes like knot tying or self-collision

## Confidence
- **High confidence**: The conceptual framework of hierarchical planning (global homotopic path sets + local temporal optimization) is well-grounded in existing literature on motion planning under constraints
- **Medium confidence**: Neural MPC with learned dynamics can effectively track planned deformations, given supporting evidence from related DLO tracking work
- **Low confidence**: Specific implementation details for homotopy computation, optimization objectives, and neural model architecture cannot be verified from the provided corpus
