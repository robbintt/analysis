# Algerian Dialect
## Quick Facts
- arXiv ID: 2512.19543
- Source URL: https://arxiv.org/abs/2512.19543
- Reference count: 4
- Primary result: Large-scale sentiment-annotated corpus of 45,000 Algerian Arabic YouTube comments

## Executive Summary
The Algerian Dialect dataset addresses a critical gap in Arabic natural language processing by providing the first large-scale sentiment-annotated corpus for Algerian Arabic. This resource consists of 45,000 manually labeled YouTube comments collected from over 30 Algerian media channels, capturing authentic informal language including slang, emojis, and code-switching patterns. The dataset represents a significant advancement for dialectal Arabic sentiment analysis, offering rich metadata and a realistic sentiment distribution that reflects real-world online discourse patterns.

## Method Summary
The dataset was constructed by collecting YouTube comments from Algerian press and media channels using the YouTube Data API, yielding authentic informal language samples. Native speakers manually annotated each comment into five sentiment categories ranging from very negative to very positive. The preprocessing pipeline included Arabic text normalization and emoji tokenization, with evaluation of different strategies to optimize model performance. Transformer-based models were trained and evaluated on the dataset, demonstrating competitive accuracy and F1-scores for sentiment classification tasks.

## Key Results
- 45,000 manually annotated YouTube comments in Algerian Arabic
- Five-level sentiment classification (very negative to very positive)
- Transformer models achieved competitive accuracy and F1-scores
- Dataset includes rich metadata: timestamps, like counts, and video URLs

## Why This Works (Mechanism)
The dataset captures authentic Algerian Arabic usage patterns through social media comments, providing realistic informal language data. The five-level sentiment annotation scheme allows for nuanced sentiment classification, while the large sample size (45,000 comments) enables robust model training and evaluation.
