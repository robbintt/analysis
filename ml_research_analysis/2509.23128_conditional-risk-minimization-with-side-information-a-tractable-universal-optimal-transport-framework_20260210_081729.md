# Conditional Risk Minimization with Side Information: A Tractable, Universal Optimal Transport Framework

## Quick Facts
- arXiv ID: 2509.23128
- Source URL: https://arxiv.org/abs/2509.23128
- Reference count: 40
- Key outcome: Develops a universal optimal transport framework for conditional risk minimization with side information

## Executive Summary
This paper introduces a novel universal optimal transport framework for conditional risk minimization with side information. The key innovation is a union-ball formulation that explicitly accounts for uncertainty in both the reference distribution and robustness radius, unifying existing OT-based approaches and enabling exact reformulations for a broad class of risk functionals. The framework yields convex reformulations for virtually all major risk measures studied in the literature, including those previously beyond reach such as rank-dependent expected utility.

## Method Summary
The framework transforms conditional distributionally robust optimization (DRO) problems into tractable convex programs by reformulating them as a union of standard Wasserstein DRO problems. For conditional expectation problems, the approach simplifies to tractable convex programs using cutting-plane algorithms. The method supports a broad class of risk measures including distortion risk measures and rank-dependent expected utility, which were previously difficult to handle in conditional settings.

## Key Results
- Achieves superior stability and convergence compared to unconditional benchmarks and existing conditional approaches
- Yields convex reformulations for virtually all major risk measures, including rank-dependent expected utility
- Simplifies conditional expectation problems to tractable convex programs using cutting-plane algorithms
- Consistently outperforms 5 baselines in numerical experiments on portfolio allocation

## Why This Works (Mechanism)
The framework works by explicitly modeling uncertainty in both the reference distribution and robustness radius through a union-ball formulation. This unified approach transforms the conditional DRO problem into a min-max optimization that can be solved efficiently using cutting-plane algorithms. The union structure allows leveraging existing results from standard DRO while naturally handling the conditional nature of the problem.

## Foundational Learning
- **Union-ball formulation**: A novel way to represent conditional DRO problems that unifies existing approaches; needed to handle uncertainty in both reference distribution and radius
- **Cutting-plane algorithm**: Iterative method that solves convex restricted master problems; needed for tractable solution of the reformulated min-max problem
- **Distortion risk measures**: Class of risk functionals including VaR and CVaR; needed as they were previously difficult to handle in conditional settings
- **Wasserstein distance**: Metric used to define uncertainty sets; needed as the foundation for the DRO formulation

## Architecture Onboarding
**Component Map**: Data Generation -> Geometry Setup -> Algorithm Implementation -> Evaluation Loop

**Critical Path**: The cutting-plane algorithm (Algorithm 1) is the core component. It iteratively solves a convex restricted master problem (Step 4) and computes the worst-case scenario (Step 6) until convergence.

**Design Tradeoffs**: The union-ball approach trades off between tractability and expressiveness. While it enables convex reformulations for complex risk measures, it requires solving a min-max problem that grows with the number of scenarios. The cutting-plane method is provably convergent but may be slow for very large datasets.

**Failure Signatures**: 
- Feasibility failure if δ₀ < δₘᵢₙ
- Slow convergence if scenario pool grows too large or tolerance is too tight
- Numerical instability when handling complex distortion functions
