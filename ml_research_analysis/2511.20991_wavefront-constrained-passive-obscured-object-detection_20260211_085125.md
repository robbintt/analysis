# Wavefront-Constrained Passive Obscured Object Detection

## Quick Facts
- arXiv ID: 2511.20991
- Source URL: https://arxiv.org/abs/2511.20991
- Authors: Zhiwen Zheng; Yiwei Ouyang; Zhao Huang; Tao Zhang; Xiaoshuai Zhang; Huiyu Zhou; Wenwen Tang; Shaowei Jiang; Jin Liu; Xingru Huang
- Reference count: 12
- Primary result: WavePCNet achieves MAE of 0.159, Max-F of 0.666, and Mean-F of 0.634 on obscured object detection

## Executive Summary
This paper tackles the challenging problem of detecting and segmenting obscured objects in non-line-of-sight scenarios using only faint scattered light patterns. The authors introduce WavePCNet, a physics-driven deep learning framework that leverages wavefront propagation constraints to recover object boundaries from multiply-scattered speckle fields. By combining a Tri-Phase Wavefront Complex-Propagation Reprojection (TriWCP) module with High-Frequency Cross-Layer Compensation, the method addresses both the ambiguity of wavefront distortion and the loss of high-frequency information in obscured imaging.

## Method Summary
WavePCNet is a physics-driven deep learning framework designed for obscured object detection in non-line-of-sight scenarios. The method integrates a Tri-Phase Wavefront Complex-Propagation Reprojection (TriWCP) module with High-Frequency Cross-Layer Compensation. TriWCP enforces coherent light propagation constraints in the complex domain to reduce ambiguity from wavefront distortion, while the compensation module enhances residual high-frequency features and cross-layer semantic consistency. The framework is evaluated on four physically collected datasets and demonstrates state-of-the-art performance with real-time inference capability.

## Key Results
- Achieves MAE of 0.159, Max-F of 0.666, and Mean-F of 0.634 on obscured object detection
- Outperforms existing methods on four physically collected datasets
- Maintains computational efficiency with real-time inference capability

## Why This Works (Mechanism)
The method works by enforcing coherent light propagation constraints through the TriWCP module, which reduces ambiguity caused by wavefront distortion in multiply-scattered speckle fields. The High-Frequency Cross-Layer Compensation addresses the loss of high-frequency information and improves cross-layer semantic consistency. This physics-informed approach allows the network to recover obscured object boundaries from severely degraded input signals by leveraging both the physical properties of light propagation and deep learning's feature extraction capabilities.

## Foundational Learning

**Wavefront propagation physics**: Understanding how light scatters and propagates through obscuring media is essential for designing appropriate constraints. *Quick check*: Verify that the wavefront model accurately represents the physical scattering process in the experimental setup.

**Complex domain signal processing**: Working in the complex domain allows encoding both amplitude and phase information critical for wavefront reconstruction. *Quick check*: Confirm that complex operations preserve information content better than real-valued alternatives.

**Cross-layer feature fusion**: Combining features from different network depths helps capture both semantic context and fine details. *Quick check*: Validate that high-frequency compensation actually recovers lost edge information in the final segmentation.

## Architecture Onboarding

Component map: Input -> TriWCP module -> High-Frequency Cross-Layer Compensation -> Output segmentation

Critical path: The TriWCP module processes the input speckle field, enforcing wavefront constraints before passing features to the compensation module, which then produces the final segmentation. This physics-first approach ensures that learned features respect the underlying optical physics.

Design tradeoffs: The physics-driven approach trades some architectural flexibility for improved generalization to the physical constraints of the problem. While this may limit performance on datasets that violate the assumed physics, it provides better interpretability and potentially better generalization to similar NLoS scenarios.

Failure signatures: Poor performance on datasets with significantly different scattering media or environmental conditions than those in the training data. Also likely to struggle when the input signal-to-noise ratio drops below the threshold used in the physical experiments.
