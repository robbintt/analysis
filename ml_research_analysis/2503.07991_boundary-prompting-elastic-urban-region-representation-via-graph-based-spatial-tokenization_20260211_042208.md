# Boundary Prompting: Elastic Urban Region Representation via Graph-based Spatial Tokenization

## Quick Facts
- arXiv ID: 2503.07991
- Source URL: https://arxiv.org/abs/2503.07991
- Authors: Haojia Zhu; Jiahui Jin; Dong Kan; Rouxi Shen; Ruize Wang; Xiangguo Sun; Jinghui Zhang
- Reference count: 40
- Primary result: 69.7% improvement in R2 score over state-of-the-art methods for urban region representation

## Executive Summary
This paper introduces the Boundary Prompting Urban Region Representation Framework (BPURF) to address the limitation of fixed-region boundaries in urban modeling. The framework treats urban entities as spatial tokens and integrates them into a unified token graph, enabling elastic region definitions through boundary prompting. By employing a multi-channel message passing model and a fast token set extraction strategy, BPURF achieves significant improvements in representing dynamic urban regions across different scales and sizes.

## Method Summary
BPURF addresses fixed-region boundary limitations by treating urban entities as spatial tokens and constructing a unified token graph. The framework employs a multi-channel message passing model to embed token sets corresponding to region boundaries, allowing elastic region definitions. A fast token set extraction strategy enables efficient online extraction during both training and inference. The approach integrates POI data into spatial token representations and uses boundary prompting to learn flexible region boundaries that adapt to urban dynamics.

## Key Results
- Achieves up to 69.7% improvement in R2 score compared to state-of-the-art methods
- Demonstrates stable performance across different region sizes and urban scales
- Shows effectiveness across multiple urban datasets with varying characteristics

## Why This Works (Mechanism)
The framework works by transforming urban entities into spatial tokens that can interact through a unified graph structure. The multi-channel message passing mechanism allows information to flow between tokens, capturing complex spatial relationships and dependencies. Boundary prompting enables the system to learn elastic region definitions by adjusting token interactions based on contextual information. The fast token set extraction strategy ensures computational efficiency during both training and inference phases, making the approach practical for real-world applications.
