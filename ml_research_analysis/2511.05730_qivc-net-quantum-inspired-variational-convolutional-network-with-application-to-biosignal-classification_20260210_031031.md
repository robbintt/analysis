# QiVC-Net: Quantum-Inspired Variational Convolutional Network, with Application to Biosignal Classification

## Quick Facts
- arXiv ID: 2511.05730
- Source URL: https://arxiv.org/abs/2511.05730
- Reference count: 40
- Primary result: State-of-the-art 97.84% and 97.89% accuracy on PhysioNet CinC 2016 and PhysioNet CirCor DigiScope 2022 PCG datasets

## Executive Summary
QiVC-Net introduces a quantum-inspired variational convolutional network for phonocardiogram classification that leverages a rotated ensemble mechanism to achieve uncertainty-aware modeling without increasing model parameters. The approach applies structured, low-dimensional subspace rotations to convolutional weights, enabling the network to capture diverse representations while maintaining computational efficiency. The model demonstrates state-of-the-art performance on two major PCG classification benchmarks while offering improved robustness to noise and class imbalance.

## Method Summary
The core innovation of QiVC-Net is the Quantum-Inspired Rotated Ensemble (QiRE) mechanism, which applies controlled rotations to convolutional weight matrices within low-dimensional subspaces. This rotation-based approach enables the model to generate diverse ensemble predictions while sharing the same underlying parameters, effectively creating an ensemble without the computational overhead of multiple independent models. The variational aspect comes from treating these rotated weights as samples from a learned distribution, allowing the network to quantify prediction uncertainty. The architecture combines standard convolutional layers with the QiRE mechanism at key points to maintain both feature extraction capability and uncertainty modeling.

## Key Results
- Achieved 97.84% accuracy on PhysioNet CinC 2016 dataset
- Achieved 97.89% accuracy on PhysioNet CirCor DigiScope 2022 dataset
- Outperformed previous state-of-the-art methods on both benchmarks
- Demonstrated improved robustness to noise and class imbalance compared to baseline approaches

## Why This Works (Mechanism)
The QiRE mechanism works by applying quantum-inspired rotations to convolutional weights, creating an implicit ensemble of models that share parameters but operate in different representational subspaces. These rotations are structured to preserve important signal characteristics while introducing diversity in the feature space. By treating the rotated weights as samples from a variational distribution, the network can estimate prediction uncertainty through the variance across ensemble members. This approach is computationally efficient because it avoids the need for multiple forward passes with independent parameter sets, instead using mathematical transformations of a single set of learned weights.

## Architecture Onboarding

Component Map: Input -> 1D Convolutional Layers -> QiRE Blocks -> Classification Head -> Output

Critical Path: The signal flows through initial convolutional layers for feature extraction, then through QiRE blocks where the quantum-inspired rotations occur, followed by final classification layers. The QiRE blocks are the critical innovation point where uncertainty-aware modeling is introduced.

Design Tradeoffs: The QiRE mechanism trades explicit ensemble diversity for parameter efficiency by using rotations instead of separate parameter sets. This reduces memory requirements but may limit the diversity achievable compared to full ensembles. The variational aspect adds complexity to training but enables uncertainty quantification. The choice of rotation dimensionality affects both computational cost and representational power.

Failure Signatures: Poor uncertainty calibration would manifest as overconfident predictions on out-of-distribution data. If rotations are too constrained, the model may fail to capture sufficient diversity, leading to underdispersed predictions. Computational overhead during inference could become prohibitive if rotations are not efficiently implemented.
