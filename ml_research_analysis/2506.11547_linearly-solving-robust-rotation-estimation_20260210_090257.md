# Linearly Solving Robust Rotation Estimation

## Quick Facts
- arXiv ID: 2506.11547
- Source URL: https://arxiv.org/abs/2506.11547
- Authors: Yinlong Liu; Tianyu Huang; Zhi-Xin Yang
- Reference count: 40

## Executive Summary
This paper introduces a novel approach to robust rotation estimation by reformulating the problem as linear model fitting using quaternion circles and stereographic projection. The method represents rotation constraints as great circles on the unit quaternion sphere, where each observation pair generates a quaternion circle and the solution corresponds to their intersection. By projecting these circles to 3D curves using stereographic projection, the algorithm enables efficient voting-based computation. The approach achieves linear time complexity O(N) and can handle extremely large-scale problems (10^6 data points) with severe corruption (99% outliers) in under 0.5 seconds using GPU acceleration.

## Method Summary
The proposed method transforms rotation estimation into a linear model fitting problem by representing rotation constraints as great circles (quaternion circles) on the unit quaternion sphere. Each observation pair generates a quaternion circle, and the rotation solution corresponds to the intersection point of these circles. The algorithm employs stereographic projection to map quaternion circles to 3D curves in Euclidean space, enabling efficient voting-based computation to find the optimal rotation. This reformulation allows the method to achieve linear time complexity O(N) while maintaining robustness to extreme outlier ratios, making it particularly suitable for large-scale applications in computer vision and robotics.

## Key Results
- Achieves linear time complexity O(N) for robust rotation estimation
- Handles large-scale problems (10^6 points) with 99% outlier ratio in under 0.5 seconds using GPU
- Demonstrates superior accuracy and efficiency compared to state-of-the-art robust rotation estimation algorithms

## Why This Works (Mechanism)
The method works by leveraging the geometric properties of quaternion representations of rotations. By recognizing that rotation constraints naturally form great circles on the unit quaternion sphere, the problem can be transformed into finding the intersection of these circles. Stereographic projection preserves the essential geometric relationships while mapping the problem to Euclidean space, where voting algorithms can efficiently identify the consensus rotation. This approach effectively linearizes what would otherwise be a non-convex optimization problem.

## Architecture Onboarding

Component map: Observation pairs -> Quaternion circles -> Stereographic projection -> 3D curves -> Voting algorithm -> Rotation estimate

Critical path: The pipeline processes each observation pair independently to generate quaternion circles, then applies stereographic projection to create 3D curves that can be processed in parallel using voting algorithms to identify the consensus rotation.

Design tradeoffs: The method trades exact geometric computations on the sphere for approximate but computationally efficient operations in Euclidean space. This enables linear scaling but requires careful handling of projection artifacts and voting threshold selection.

Failure signatures: The algorithm may fail when quaternion circles are nearly parallel (nearly degenerate configurations) or when outliers align to create false consensus among the projected curves. Performance degradation is expected for nearly coplanar point sets.
