# Synthetic Data Generation by Supervised Neural Gas Network for Physiological Emotion Recognition Data

## Quick Facts
- arXiv ID: 2501.16353
- Source URL: https://arxiv.org/abs/2501.16353
- Reference count: 40
- Primary result: SNG network achieves up to 97.37% accuracy on ESD dataset while significantly improving processing time for synthetic data generation

## Executive Summary
This study addresses the critical challenge of data scarcity in physiological emotion recognition by introducing a Supervised Neural Gas (SNG) network for synthetic data generation. The SNG network combines topological organization with feature-space proximity to efficiently create real-world-like synthetic datasets that preserve intrinsic patterns of physiological emotion data. Through experiments on EEG and ECG/GSR datasets, the research demonstrates that SNG outperforms most evaluated models in both accuracy and computational efficiency, achieving high classification performance while maintaining data distribution fidelity.

## Method Summary
The proposed approach employs a Supervised Neural Gas network that organizes input data based on both topological and feature-space proximity, enabling efficient synthetic data generation. The SNG network learns the underlying structure of physiological emotion data and generates synthetic samples that closely mimic original distributions. The method was evaluated on two datasets - an EEG dataset and an ECG/GSR dataset - using classification accuracy as the primary performance metric. The synthetic data generation process preserves intrinsic patterns while addressing the data scarcity problem common in emotion recognition applications.

## Key Results
- SNG achieves up to 97.37% accuracy on the ESD dataset
- Mean squared error of 0.059 demonstrates high fidelity to original EEG data distributions
- Significant improvements in processing time compared to alternative generative models

## Why This Works (Mechanism)
The Supervised Neural Gas network effectively combines supervised learning principles with unsupervised topological organization to capture complex patterns in physiological emotion data. By organizing neurons based on both topological and feature-space proximity, SNG can efficiently learn the manifold structure of emotion-related physiological signals. This dual-organization approach enables the network to generate synthetic samples that maintain the intrinsic relationships and distributions present in the original data, resulting in high-quality synthetic datasets suitable for emotion recognition tasks.

## Limitations
- Evaluation primarily relies on classification accuracy metrics, which may not fully capture synthetic data quality

- Computational efficiency analysis lacks detailed scalability assessment across different hardware and dataset scales
