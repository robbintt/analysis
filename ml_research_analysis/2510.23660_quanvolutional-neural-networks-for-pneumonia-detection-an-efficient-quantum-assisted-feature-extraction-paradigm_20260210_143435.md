# Quanvolutional Neural Networks for Pneumonia Detection: An Efficient Quantum-Assisted Feature Extraction Paradigm

## Quick Facts
- arXiv ID: 2510.23660
- Source URL: https://arxiv.org/abs/2510.23660
- Reference count: 24
- Primary result: QNN model achieves 83.33% validation accuracy vs 73.33% for classical CNN on PneumoniaMNIST dataset

## Executive Summary
This paper introduces a hybrid quantum-classical model using Quanvolutional Neural Networks (QNNs) for pneumonia detection from chest X-ray images. The model employs a parameterized quantum circuit to process 2×2 image patches, using rotational Y-gates for data encoding and entangling layers to generate non-classical feature representations, which are then classified using a classical neural network. Experiments on the PneumoniaMNIST dataset show that the QNN model outperforms a classical CNN baseline under identical conditions, demonstrating enhanced convergence and sample efficiency. The results highlight the potential of quantum-enhanced feature extraction for medical image analysis, especially with limited labeled data.

## Method Summary
The paper presents a hybrid quantum-classical approach that integrates parameterized quantum circuits (PQC) into the feature extraction pipeline of a convolutional neural network for pneumonia detection. The Quanvolutional layer processes 2×2 patches from chest X-ray images using rotational Y-gates for data encoding and CNOT gates for entanglement, creating non-classical feature representations. These quantum-processed features are then fed into a classical neural network for classification. The model is trained and evaluated on the PneumoniaMNIST dataset, with systematic comparison against a classical CNN baseline using identical hyperparameters and architecture depth.

## Key Results
- QNN model achieves 83.33% validation accuracy compared to 73.33% for classical CNN baseline
- Quantum-assisted feature extraction demonstrates enhanced convergence and sample efficiency
- Model shows improved performance particularly in scenarios with limited labeled data

## Why This Works (Mechanism)
The quantum advantage in this pneumonia detection model stems from the quantum circuit's ability to process image patches through quantum superposition and entanglement. The parameterized quantum circuit (PQC) uses rotational Y-gates to encode classical image data into quantum states, creating quantum feature maps that capture complex correlations between pixel values. The entangling layers (CNOT gates) generate non-classical feature representations that may be more expressive than classical convolutions, particularly for capturing subtle patterns in medical images. These quantum-processed features are then passed to a classical neural network, creating a hybrid architecture that leverages quantum computational advantages while maintaining classical classification capabilities.
