# Physics-informed transfer learning for SHM via feature selection

## Quick Facts
- arXiv ID: 2507.19519
- Source URL: https://arxiv.org/abs/2507.19519
- Reference count: 40
- Primary result: Physics-informed feature selection using MAC criterion significantly improves transfer learning accuracy in structural health monitoring between dissimilar structures.

## Executive Summary
This paper addresses the critical challenge of transferring damage classifiers between structurally dissimilar systems in population-based structural health monitoring (PBSHM). The authors propose a novel approach that leverages physics-informed feature selection using the Modal Assurance Criterion (MAC) to identify vibration-based features with related conditional distributions across domains.

## Method Summary
The authors propose a physics-informed transfer learning framework that uses the Modal Assurance Criterion (MAC) to identify vibration-based features with related conditional distributions across domains. The MAC criterion measures the orthogonality between mode shapes, and features are selected based on their MAC values exceeding a threshold. The approach consists of two main steps: (1) calculating MAC values between source and target structures to identify physically similar features, and (2) applying transfer learning with these selected features. The method is validated using two case studies: (1) simulated cantilever beams with varying boundary conditions, and (2) experimental data from two steel plates with different geometries and boundary conditions.

## Key Results
The method achieves significant performance improvements in transfer learning accuracy. For the cantilever beam case study, the proposed approach increased accuracy from 76.3% to 98.4% when transferring between structures with different boundary conditions. For the steel plate case study, accuracy improved from 68.2% to 93.7% when transferring between structures with different geometries. The results demonstrate that physics-informed feature selection using MAC criterion can effectively identify transferable features even between structurally dissimilar systems.

## Why This Works (Mechanism)
The MAC criterion effectively captures the physical similarity between mode shapes of different structures, allowing the selection of features that share similar underlying physical principles despite structural differences. By focusing on features with high MAC values, the method ensures that the transferred knowledge is based on physically meaningful similarities rather than arbitrary statistical correlations. This physics-informed approach addresses the domain shift problem by identifying features whose conditional distributions are related across domains, enabling more effective knowledge transfer.

## Foundational Learning
The approach builds upon existing transfer learning methods in structural health monitoring, particularly domain adaptation techniques. It extends traditional transfer learning by incorporating physics-based feature selection rather than relying solely on statistical methods. The method also leverages established modal analysis techniques, specifically the MAC criterion, which has been widely used in structural dynamics for comparing mode shapes.

## Architecture Onboarding
The method uses a relatively simple architecture consisting of a feature selection module based on MAC criterion followed by a standard transfer learning classifier. The feature selection module calculates MAC values between all possible feature pairs from source and target domains, selecting those exceeding a threshold. The transfer learning classifier can be any standard model suitable for the specific SHM task, though the paper uses a support vector machine for the cantilever beam case and a neural network for the steel plate case.

## Open Questions the Paper Calls Out
The authors acknowledge several open questions: (1) The optimal threshold for MAC-based feature selection remains an empirical question requiring further investigation, (2) The method's performance on larger-scale real-world structures with more complex damage scenarios needs validation, (3) The computational cost of calculating MAC values for large feature spaces could be prohibitive in some applications, and (4) The generalizability of the approach to non-vibration-based features and other domains beyond structural health monitoring.

## Limitations
The method has several limitations: (1) It requires access to both source and target structural data for MAC calculation, which may not always be feasible in practice, (2) The effectiveness depends on the availability of sufficient mode shape information, which can be challenging to obtain for complex structures, (3) The approach is primarily validated on relatively simple structures (beams and plates), limiting confidence in performance on real-world complex structures, and (4) The computational cost increases with the number of features and structures, potentially limiting scalability.

## Confidence
Moderate confidence. The method demonstrates strong theoretical grounding by combining physics-based feature selection with established transfer learning principles. The validation on two distinct case studies (simulated beams and experimental plates) provides reasonable evidence of effectiveness. However, the relatively simple test cases and lack of testing on real-world complex structures limit confidence in broader applicability. The moderate confidence rating reflects the solid methodology but limited scope of validation.

## Next Checks
Key next steps include: (1) Testing the method on more complex, real-world structures with multiple damage locations and types, (2) Evaluating computational efficiency and scalability for large feature spaces and multiple structures, (3) Investigating optimal MAC threshold selection through systematic sensitivity analysis, (4) Exploring extension to other feature types beyond vibration modes (e.g., acoustic emission, strain measurements), and (5) Comparing performance against purely data-driven transfer learning approaches without physics-based feature selection.
