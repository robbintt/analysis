# OceanAI: A Conversational Platform for Accurate, Transparent, Near-Real-Time Oceanographic Insights
## Quick Facts
- arXiv ID: 2511.01019
- Source URL: https://arxiv.org/abs/2511.01019
- Reference count: 22
- Primary result: OceanAI integrates LLMs with NOAA data APIs to deliver authoritative, reproducible oceanographic insights with full metadata transparency.

## Executive Summary
OceanAI is a conversational AI platform designed to address the problem of "hallucinations" in scientific domains by grounding large language model outputs in authoritative, real-time NOAA oceanographic data. The system uses function calling to retrieve, process, and visualize structured datasets like sea level trends and sea surface temperature (SST), ensuring outputs are both accurate and transparent. In blind comparisons, OceanAI consistently provided NOAA-sourced values with full metadata, while other models either declined to answer or gave unsupported results. This approach bridges the usability gap between conversational AI and expert-level data portals, enabling interdisciplinary access to ocean science with technical rigor and reproducibility.

## Method Summary
OceanAI integrates large language models (LLMs) with NOAA's OceanWatch and CO-OPS APIs using function calling to retrieve and process real-time oceanographic data. The system translates natural language queries into structured API calls, retrieves datasets, and uses parameter optimization to filter and visualize results. Outputs include detailed metadata such as data source, measurement units, and processing steps, ensuring transparency and reproducibility. The platform supports multiple visualization formats (line plots, heatmaps, etc.) and is built with a modular architecture for scalability and adaptability to other scientific domains.

## Key Results
- OceanAI consistently provided NOAA-sourced values with full metadata in blind comparisons, while other models declined or gave unsupported results.
- The system ensures outputs are grounded in authoritative data, reducing hallucinations common in conversational AI.
- Integration with NOAA APIs enables real-time access to sea level trends, SST, and other oceanographic datasets with reproducible results.

## Why This Works (Mechanism)
OceanAI leverages function calling to bridge the gap between natural language queries and structured oceanographic data retrieval. By grounding LLM responses in NOAA's authoritative datasets, the system ensures accuracy and transparency. The modular architecture allows for seamless integration of multiple data sources and visualization tools, while the inclusion of full metadata supports reproducibility. This approach addresses the usability gap between conversational AI and expert-level data portals, making oceanographic insights accessible to interdisciplinary users.
