# Fuzzy Propositional Formulas under the Stable Model Semantics

## Quick Facts
- arXiv ID: 2506.12804
- Source URL: https://arxiv.org/abs/2506.12804
- Authors: Joohyung Lee; Yi Wang
- Reference count: 40
- Key outcome: Defines fuzzy stable model semantics equivalent to fuzzy equilibrium logic, extending ASP to dynamic domains with graded truth values

## Executive Summary
This paper introduces a stable model semantics for fuzzy propositional formulas that generalizes both fuzzy propositional logic and classical stable models. The approach extends the reduct-based stable model definition to many-valued settings using fuzzy interpretations, enabling the representation of defaults with graded truth values in dynamic domains. The authors demonstrate that this fuzzy stable model semantics is essentially equivalent to fuzzy equilibrium logic but simpler, and show how it extends ASP capabilities through a trust dynamics example. The framework naturally extends several properties of Boolean stable models and can be reduced to a threshold-based version.

## Method Summary
The paper defines fuzzy propositional formulas using a generalization of fuzzy logic programming syntax, where formulas are constructed from fuzzy propositional variables using conjunction, disjunction, implication, and negation operators. The fuzzy stable model semantics is defined through a fuzzy reduct operation that extends the classical reduct to many-valued interpretations. A fuzzy interpretation is considered a stable model if it is the least model of its fuzzy reduct. The authors establish the equivalence between this semantics and fuzzy equilibrium logic through a reduction, showing that the fuzzy reduct-based approach provides a simpler alternative for computing stable models. The framework supports reasoning about defaults with fuzzy truth values, particularly useful for modeling dynamic domains where truth values can change over time.

## Key Results
- Fuzzy stable model semantics generalizes both fuzzy propositional logic and classical stable models
- The approach is shown to be essentially equivalent to fuzzy equilibrium logic but computationally simpler
- The framework extends ASP capabilities to dynamic domains with graded truth values, demonstrated through a trust dynamics example

## Why This Works (Mechanism)
The mechanism works by extending the classical reduct operation to fuzzy interpretations, where the fuzzy reduct of a formula with respect to a fuzzy interpretation is computed by replacing atoms with their truth values when they are true enough in the interpretation. A fuzzy interpretation becomes a stable model when it is the least fixed point of its fuzzy reduct, ensuring that the model satisfies all rules in the most conservative way possible given the fuzzy truth values. This extension preserves the core principles of stable model semantics while accommodating graded truth values through the use of t-norms and t-conorms for conjunction and disjunction operations. The equivalence to fuzzy equilibrium logic is established through a reduction that maps fuzzy propositional formulas to their equilibrium logic counterparts, demonstrating that the fuzzy reduct-based approach provides a simpler computational path while maintaining semantic equivalence.

## Limitations
- Computational complexity is Σ₂ᴾ-hard, significantly constraining practical applications for large-scale problems
- Empirical validation is limited to a single trust dynamics example, which may not demonstrate full generality across different application domains
- The translation from fuzzy logic programming to propositional logic is technically complex and may introduce subtle errors in practical implementations

- Extension of Boolean stable model properties: Medium (appears sound but requires careful verification in complex scenarios)
