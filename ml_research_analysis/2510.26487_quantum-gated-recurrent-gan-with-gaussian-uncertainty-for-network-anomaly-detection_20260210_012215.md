# Quantum Gated Recurrent GAN with Gaussian Uncertainty for Network Anomaly Detection

## Quick Facts
- arXiv ID: 2510.26487
- Source URL: https://arxiv.org/abs/2510.26487
- Reference count: 36
- Achieves TaF1 score of 89.43% on HAI industrial control system dataset

## Executive Summary
This paper introduces a Quantum Gated Recurrent Unit (QGRU)-based Generative Adversarial Network enhanced with Successive Data Injection (SuDaI) for network anomaly detection in multivariate time-series data. The hybrid quantum-classical architecture employs classical preprocessing that feeds into Variational Quantum Circuits, enabling efficient encoding of high-dimensional features within limited qubit resources. Anomalies are detected through a multi-stage gating mechanism that leverages predictive uncertainty, Wasserstein critic scores, and reconstruction errors. The model demonstrates strong performance on the HAI industrial control system dataset and shows practical feasibility through successful deployment on real IBM Quantum hardware without retraining.

## Method Summary
The proposed QGRU-GAN framework combines quantum computing with classical machine learning for network anomaly detection. The architecture uses a hybrid approach where classical preprocessing extracts relevant features from network traffic data, which are then encoded into quantum states through variational quantum circuits. The QGRU component handles temporal dependencies in the time-series data, while the GAN framework generates synthetic data for comparison against real network traffic. The Successive Data Injection mechanism allows for continuous learning and adaptation to evolving network patterns. Anomaly detection is performed through a multi-stage process that evaluates prediction uncertainty, Wasserstein critic scores, and reconstruction errors to identify deviations from normal network behavior.

## Key Results
- Achieves TaF1 score of 89.43% on HAI industrial control system dataset
- Successfully deployed on real IBM Quantum hardware without retraining
- Outperforms classical and quantum baselines in network anomaly detection tasks

## Why This Works (Mechanism)
The hybrid quantum-classical architecture enables efficient processing of high-dimensional network features by leveraging quantum superposition and entanglement properties. The QGRU component captures temporal dependencies in network traffic patterns through quantum-enhanced recurrent processing, while the GAN framework provides a generative model for understanding normal network behavior. The multi-stage gating mechanism combines multiple uncertainty measures to create a robust anomaly detection system that can adapt to evolving network patterns. The Successive Data Injection mechanism allows the model to continuously update its understanding of network behavior without requiring complete retraining.
