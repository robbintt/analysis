# Exploring Diffusion Models for Generative Forecasting of Financial Charts

## Quick Facts
- arXiv ID: 2509.02308
- Source URL: https://arxiv.org/abs/2509.02308
- Reference count: 23
- Primary result: 68.89% accuracy on predicting next-candle chart patterns using diffusion models

## Executive Summary
This paper explores an innovative approach to financial forecasting by treating time-series financial data as visual patterns and using diffusion models to generate future candlestick chart images. The authors fine-tune Stable Diffusion 1.5 to predict next-period candle charts based on current chart images combined with technical indicator prompts (RSI, MACD). The methodology constructs paired datasets of current and future charts with evaluation markers, achieving an overall accuracy of 68.89% on Bitcoin futures data. While the majority class performs reasonably well (81.59% F1-score), minority classes show poor performance due to data imbalance and model limitations.

## Method Summary
The approach converts financial time-series data into visual candlestick charts, then fine-tunes a diffusion model to generate future chart images conditioned on current charts and technical indicator prompts. The dataset consists of paired current and future charts with color-coded evaluation markers (red for >2% gain, blue for >2% loss, black otherwise). The model uses instruction-based fine-tuning with prompts containing RSI and MACD values. Evaluation employs an RGB-based classification method to compare generated images against ground truth. The methodology treats financial forecasting as an image generation problem, leveraging diffusion models' ability to capture complex patterns while incorporating technical indicators as conditioning information.

## Key Results
- Overall accuracy of 68.89% on predicting next-candle chart patterns
- Majority class (black) achieves 81.59% F1-score, while minority classes perform poorly
- Model successfully captures chart patterns including volume and moving averages
- RGB-based evaluation method shows feasibility but highlights limitations for financial forecasting

## Why This Works (Mechanism)
The approach works by leveraging diffusion models' ability to learn complex visual patterns and generate coherent future states conditioned on current information. By converting financial data to images, the model can capture spatial and visual relationships between price movements, volumes, and technical indicators that might be harder to learn from raw numerical sequences. The conditioning mechanism using technical indicator prompts guides the generation process toward incorporating fundamental analysis signals. The paired dataset construction enables supervised learning of temporal transitions between chart states, while the evaluation marker system provides clear ground truth for model assessment.

## Architecture Onboarding

**Component Map**
Stable Diffusion 1.5 model -> Fine-tuning pipeline -> Conditioning mechanism (RSI/MACD prompts) -> Generation process -> RGB classification evaluation

**Critical Path**
The critical path flows from input chart image and technical indicator prompts through the conditioned diffusion model to generate future chart predictions, which are then evaluated using RGB classification against ground truth markers.

**Design Tradeoffs**
The approach trades precise numerical prediction for pattern-based visual forecasting, potentially losing granular price information but gaining the ability to capture complex visual relationships. Using Stable Diffusion 1.5 provides a robust starting point but may limit generation quality compared to newer architectures. The RGB-based evaluation is simple to implement but fails to capture nuanced financial patterns, representing a tradeoff between evaluation simplicity and accuracy.

**Failure Signatures**
Poor performance on minority classes (red/blue markers) indicates the model struggles with significant price movements. Inability to accurately predict precise price levels suggests information loss in the image conversion process. High variance in generated outputs for similar inputs indicates insufficient conditioning signal strength or model capacity limitations.

## Limitations
- Image-based approach loses precise numerical information crucial for financial forecasting
- RGB-based evaluation is overly simplistic and fails to capture nuanced chart patterns
- Limited dataset scope (Bitcoin futures only) raises concerns about generalizability
- Poor minority class performance due to data imbalance and model constraints
- Use of older Stable Diffusion 1.5 architecture may limit generation quality
