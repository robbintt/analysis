# Efficient Best-of-Both-Worlds Algorithms for Contextual Combinatorial Semi-Bandits

## Quick Facts
- arXiv ID: 2508.18768
- Source URL: https://arxiv.org/abs/2508.18768
- Reference count: 40
- This paper presents the first best-of-both-worlds algorithm for contextual combinatorial semi-bandits that achieves O(√T) regret in the adversarial regime and O(ln T) regret in the corrupted stochastic regime.

## Executive Summary
This paper introduces a novel best-of-both-worlds algorithm for contextual combinatorial semi-bandits that simultaneously achieves optimal performance in both adversarial and corrupted stochastic regimes. The algorithm builds on Follow-the-Regularized-Leader (FTRL) with Shannon entropy regularization, enhanced by an efficient projection subroutine that transforms high-dimensional optimization into a single-variable root-finding problem. The method delivers substantial computational speed-ups while maintaining theoretical guarantees, making it practical for large-scale real-time applications.

## Method Summary
The core approach employs FTRL with Shannon entropy regularization combined with an efficient projection subroutine. The key innovation is leveraging Karush-Kuhn-Tucker (KKT) conditions to transform the K-dimensional convex projection problem into a single-variable root-finding problem, dramatically reducing computational complexity per round. This optimization enables the algorithm to scale efficiently to large problem instances while maintaining the attractive regret bounds characteristic of best-of-both-worlds algorithms.

## Key Results
- Achieves O(√m ln(K/m)T) regret in the adversarial regime
- Achieves O(κ²m^(3/2) ln((K-m)T)/∆_min) regret in the stochastic regime
- Delivers substantial per-round speed-ups through KKT-based projection optimization
- Demonstrates practical efficiency gains across empirical evaluations

## Why This Works (Mechanism)
The algorithm's efficiency stems from transforming a high-dimensional K-dimensional convex projection into a single-variable root-finding problem using KKT conditions. This mathematical transformation preserves the optimal solution while dramatically reducing computational complexity. The FTRL framework with Shannon entropy regularization provides the theoretical foundation for achieving best-of-both-worlds guarantees, while the efficient projection subroutine makes these guarantees practically achievable in real-time applications.

## Architecture Onboarding

**Component Map**: FTRL Optimizer -> KKT-based Projector -> Action Selector -> Environment Feedback -> Regret Calculator

**Critical Path**: The core computational bottleneck lies in the KKT-based projection step, which must be executed efficiently each round to maintain practical scalability. The algorithm's performance depends critically on the speed and numerical stability of solving the transformed single-variable root-finding problem.

**Design Tradeoffs**: The choice of Shannon entropy regularization balances computational tractability with regret guarantees, while the KKT-based projection trades implementation complexity for dramatic speed-ups. The algorithm assumes known corruption levels, which may not hold in practice.

**Failure Signatures**: Performance degradation occurs when corruption estimates are inaccurate, when the root-finding procedure fails to converge numerically, or when the problem dimensions K and m are extremely large relative to available computational resources.
