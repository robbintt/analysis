# Bitcoin's Edge: Embedded Sentiment in Blockchain Transactional Data

## Quick Facts
- arXiv ID: 2504.13598
- Source URL: https://arxiv.org/abs/2504.13598
- Reference count: 40
- Primary result: Blockchain-embedded sentiment can predict cryptocurrency price movements

## Executive Summary
This study demonstrates that sentiment embedded in blockchain transaction data can predict cryptocurrency price movements. The authors extracted sentiment features from Bitcoin and Ethereum transaction data using natural language processing techniques including topic modeling and sentiment analysis, then combined these with price data to train machine learning classifiers. Their Bitcoin model achieved 60.53% accuracy in predicting 1-day ahead price movements, representing a 10.53% improvement over baseline models. The research found that rule-based sentiment analysis methods (VADER and TextBlob) outperformed transformer-based models for this task, and that topic modeling revealed Bitcoin-focused themes more prominently than Ethereum-focused ones in the embedded messages.

## Method Summary
The researchers collected 4,893 Bitcoin and 3,357 Ethereum transactions containing readable text from blockchain data. They applied natural language processing techniques including topic modeling and sentiment analysis to extract sentiment features from the embedded messages. These features were combined with historical price data and used to train machine learning classifiers for predicting 1-day ahead price movements. The study compared rule-based sentiment analysis methods (VADER and TextBlob) against transformer-based models, and tested the predictive power across both Bitcoin and Ethereum datasets.

## Key Results
- Bitcoin model achieved 60.53% accuracy in predicting 1-day ahead price movements
- 10.53% improvement over baseline models for Bitcoin price prediction
- Rule-based sentiment analysis methods (VADER and TextBlob) outperformed transformer-based models
- Topic modeling revealed Bitcoin-focused themes more prominently than Ethereum-focused ones
