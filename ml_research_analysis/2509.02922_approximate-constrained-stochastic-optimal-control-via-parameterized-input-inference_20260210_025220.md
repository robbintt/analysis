# Approximate constrained stochastic optimal control via parameterized input inference

## Quick Facts
- arXiv ID: 2509.02922
- Source URL: https://arxiv.org/abs/2509.02922
- Authors: Shahbaz P Qadri Syed; He Bai
- Reference count: 24
- Primary result: Novel EM-based PIIC algorithm for constrained stochastic optimal control using barrier functions and parameter space decomposition

## Executive Summary
This paper addresses the challenge of stochastic optimal control (SOC) with state, control, and structural constraints through a novel parameterized input inference for control (PIIC) algorithm. The authors formulate SOC as an inference problem on a probabilistic graphical model and employ Expectation-Maximization to infer control parameters that satisfy various constraints. The method uses barrier functions to handle inequality constraints and decomposes the control parameter space to enforce structural constraints on distributed controllers, providing a unified framework that encompasses existing algorithms like LQR and I2C as special cases.

The PIIC algorithm demonstrates superior performance compared to ILQG baselines across multiple benchmark problems including unicycle obstacle avoidance, formation control, and quadcopter navigation. The method guarantees convergence to local optima while effectively handling constraints through factor graph optimization smoothing, which outperforms unscented smoothing in empirical tests. The approach provides a practical solution for complex control problems where constraints must be satisfied while optimizing performance in stochastic environments.

## Method Summary
The PIIC algorithm treats stochastic optimal control as an inference problem on a probabilistic graphical model, where control parameters are inferred via Expectation-Maximization iterations. The method decomposes into two main steps: an expectation step that smooths state-control trajectories using factor graph optimization, and a maximization step that updates control parameters on non-zero subsets to preserve structural constraints. Barrier functions are employed to handle inequality constraints by penalizing constraint violations in the cost function. The control parameter space is decomposed into non-zero subsets, allowing the algorithm to enforce structural constraints on distributed controllers by updating only relevant parameters during maximization. This decomposition strategy ensures that the resulting controllers maintain desired structural properties while optimizing performance under uncertainty.

## Key Results
- PIIC outperforms ILQG baseline with reduced mean cost and variance across benchmark problems
- Factor graph optimization smoothing shows superior performance compared to unscented smoothing
- Barrier function parameter γ demonstrates empirical effectiveness in constraint satisfaction
- Trade-offs between centralized and distributed controller structures are demonstrated in formation control
- Nonlinear basis functions enable effective adaptation to dynamic environment changes in quadcopter navigation

## Why This Works (Mechanism)
The PIIC algorithm works by transforming the constrained stochastic optimal control problem into a probabilistic inference framework. By treating control parameters as latent variables to be inferred, the method leverages the Expectation-Maximization algorithm to iteratively refine both state trajectories and control policies. The expectation step smooths trajectories through factor graph optimization, which provides a principled way to handle the stochastic nature of the system. The maximization step then updates control parameters in a way that respects structural constraints by operating only on pre-defined non-zero subsets. Barrier functions provide a continuous mechanism for enforcing inequality constraints, allowing the algorithm to navigate the trade-off between constraint satisfaction and performance optimization. This decomposition of the problem into inference and parameter update steps, combined with careful handling of constraints through barrier functions and parameter space decomposition, enables the algorithm to find locally optimal solutions that satisfy all problem requirements.

## Architecture Onboarding

Component Map:
SOC Problem -> Probabilistic Formulation -> EM Iteration (E-step: Factor Graph Smoothing -> M-step: Parameter Update) -> Barrier Function Handling -> Structural Constraint Decomposition -> Local Optimum

Critical Path:
Problem Formulation → EM Iteration Loop → Barrier Function Evaluation → Structural Constraint Check → Convergence Verification

Design Tradeoffs:
- Barrier parameter γ: larger values enforce constraints more strictly but may reduce performance; smaller values allow better performance but risk constraint violations
- Parameter space decomposition granularity: finer decomposition provides more control over structure but increases computational complexity
- Smoothing method choice: factor graph optimization offers better accuracy but higher computational cost compared to unscented smoothing
- Controller structure: centralized controllers may achieve better performance but distributed controllers offer scalability and robustness

Failure Signatures:
- Constraint violations increasing despite barrier functions suggests γ is too small or barrier function is improperly configured
- Slow convergence or oscillation in EM iterations indicates poor initialization or ill-conditioned cost function
- Distributed controllers failing to coordinate suggests incorrect parameter space decomposition or communication topology issues
- Performance degradation under high noise suggests smoothing method is insufficient for the noise level

## Limitations
- Barrier function performance needs empirical validation across diverse constraint geometries and densities
- Automated selection of non-zero parameter subsets for complex distributed architectures is not fully explored
- Theoretical understanding of when factor graph optimization smoothing outperforms unscented smoothing is incomplete
- Experimental validation focuses primarily on ILQG comparison rather than explicitly showing relationships to LQR and I2C

## Confidence
- High: The core EM-based inference formulation and its application to SOC problems
- Medium: The effectiveness of barrier functions for constraint satisfaction and the decomposition strategy for structural constraints
- Low: The claimed relationship to existing algorithms and the theoretical guarantees of the smoothing approach
