# XiChen: An observation-scalable fully AI-driven global weather forecasting system with 4D variational knowledge

## Quick Facts
- arXiv ID: 2507.09202
- Source URL: https://arxiv.org/abs/2507.09202
- Authors: Wuxin Wang; Weicheng Ni; Lilan Huang; Tao Hao; Ben Fei; Shuo Ma; Taikang Yuan; Yanlai Zhao; Kefeng Deng; Xiaoyong Li; Hongze Leng; Boheng Duan; Lei Bai; Weimin Zhang; Kaijun Ren; Junqiang Song
- Reference count: 40
- Primary result: Fully AI-driven global weather forecasting system achieving skillful lead times beyond 8.75 days with integrated 4DVar-based data assimilation.

## Executive Summary
XiChen is a fully AI-driven global weather forecasting system that integrates data assimilation and medium-range forecasting within a 15-second pipeline. Unlike prior approaches that rely on separate NWP systems for initial conditions, XiChen leverages a foundation model pre-trained for weather forecasting and fine-tuned for both observation operators and data assimilation. By using the gradient of a 4DVar cost function as a unifying information interface, XiChen flexibly assimilates heterogeneous observations—including raw satellite radiances—while maintaining physical consistency. Single-point perturbation experiments show that XiChen propagates observational impacts in a flow-dependent, physically consistent manner.

## Method Summary
XiChen employs a staged training approach: first pre-training a Conditional Hybrid Neural Operator (CHNO) for weather forecasting using ERA5 reanalysis, then fine-tuning observation operators for satellite radiances, and finally training a data assimilation model that operates on 4DVar gradients. The system uses a cascade DA framework with prepbufr conventional observations, MHS, ASCAT, AMSU-A, and SATWND data. During inference, DA runs for 12-hour windows with a 3-day short-term forecast branch followed by a 7-day medium-range branch. The CHNO architecture combines global mixing via Adaptive Fourier Neural Operators with local convolutional branches and cross-attention conditioning.

## Key Results
- XiChen achieves skillful lead times beyond 8.75 days, outperforming GFS system and rivaling IFS HRES for T2M, T850, MSLP, U10, and V10 variables.
- The system effectively integrates satellite data, significantly reducing forecast errors compared to conventional DA approaches.
- Tropical cyclone tracking and intensity forecasts improve with satellite assimilation, demonstrating practical forecasting benefits.

## Why This Works (Mechanism)
XiChen's success stems from its unified foundation model approach that treats data assimilation and forecasting as different fine-tuning tasks of the same architecture. By using 4DVar gradients as the information interface, the system can flexibly assimilate heterogeneous observations while maintaining physical consistency. The cascade DA design allows progressive integration of different observation types, each with specialized observation operators trained on the same foundation model.

## Architecture Onboarding
**Component map**: Observation data → Observation operators → 4DVar gradient computation → DA model → Forecast model (XiChen-Short) → Forecast model (XiChen-Medium)

**Critical path**: Observation operators → 4DVar gradient computation → DA model → Initial conditions for forecasting

**Design tradeoffs**: XiChen trades architectural complexity for unified training, accepting the challenge of multi-task learning in exchange for flexible observation assimilation and faster inference times.

**Failure signatures**: Over-smoothed forecasts and TC intensity underestimation indicate loss of small-scale variability; unstable 4DVar gradients suggest issues with observation error specifications or quality control.

## Limitations
- Critical hyperparameters (hidden dimensions, learning rates, batch sizes, DA optimization settings) are unspecified, preventing exact replication
- Dataset URLs are placeholders, requiring independent sourcing of ERA5, prepbufr, and satellite data
- Over-smoothing tendency may limit forecast accuracy for extreme weather events and tropical cyclone intensity
