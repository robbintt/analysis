# Convolutional Deep Colorization for Image Compression: A Color Grid Based Approach

## Quick Facts
- **arXiv ID:** 2502.05402
- **Source URL:** https://arxiv.org/abs/2502.05402
- **Reference count:** 40
- **Primary result:** Proposed color grid method achieved compression ratios approaching 1/3 of original size while maintaining high recolorization quality

## Executive Summary
This paper introduces a novel image compression approach using convolutional neural networks for selective color retention. The authors propose a "color grid" method where color information is retained at regular intervals across images, significantly reducing storage requirements while maintaining sufficient data for accurate color reconstruction. The Crayon model architecture, derived from XiaoNet, processes grayscale images with sparse color points to predict missing color information. Tested on the Imagenette dataset with varying color retention intervals, the approach demonstrates promising potential for image compression while achieving compression ratios comparable to existing methods while maintaining colorization accuracy.

## Method Summary
The approach employs a convolutional neural network architecture called Crayon, which is derived from the XiaoNet base model. The method works by retaining color information at regular intervals across images in a grid pattern, creating sparse color data points. During compression, the original image is converted to grayscale and only selected color values at grid positions are stored. For decompression, the Crayon model takes the grayscale image and sparse color grid as inputs to predict and reconstruct the full-color image. The system was tested with varying color retention intervals (n-values) to determine the optimal balance between compression ratio and colorization quality.

## Key Results
- Achieved compression ratios approaching 1/3 of original image size
- Optimal performance achieved at n=20 color retention interval
- Maintained high peak signal-to-noise ratio (PSNR) and color-tone similarity index measure (CSIM) values before degradation
- Demonstrated high recolorization quality while significantly reducing storage requirements

## Why This Works (Mechanism)
The color grid approach works by exploiting the spatial correlation in natural images. By retaining color information at regular grid intervals, the model leverages the convolutional neural network's ability to learn and extrapolate color patterns from local context. The Crayon model learns to interpolate missing color information based on the sparse color points and the grayscale structure, effectively reconstructing the full-color image from minimal color data. This selective retention strategy reduces storage requirements while maintaining sufficient information for accurate color reconstruction.

## Architecture Onboarding

**Component Map:** Input (Grayscale Image + Color Grid) -> Crayon Model (CNN layers) -> Output (Full-color Image)

**Critical Path:** Grayscale image and sparse color points enter the Crayon model, which processes through convolutional layers to learn spatial color patterns, then generates the full-color output through learned color interpolation.

**Design Tradeoffs:** The approach trades some color accuracy for significant storage reduction. Higher n-values (larger grid spacing) provide better compression but may reduce color reconstruction quality. The model complexity must balance reconstruction accuracy with computational efficiency.

**Failure Signatures:** Poor color reconstruction in areas with high color complexity or rapid color transitions. Degradation becomes noticeable when grid spacing is too large for the model to accurately interpolate missing color information. Loss of fine color details in texture-rich regions.

## Limitations
- Reported compression ratios may be overly optimistic without accounting for model weights and metadata storage overhead
- Lack of comparison against established compression standards (JPEG, WebP, AVIF) on identical metrics
- Limited evaluation to a single dataset (Imagenette), restricting generalizability to diverse image types
- No analysis of computational complexity during encoding and decoding phases

## Confidence

- **Medium confidence:** Claimed compression ratios due to incomplete accounting of all storage requirements
