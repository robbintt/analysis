# Sparse-mode Dynamic Mode Decomposition for Disambiguating Local and Global Structures

## Quick Facts
- arXiv ID: 2507.19787
- Source URL: https://arxiv.org/abs/2507.19787
- Reference count: 40
- Introduces sparse-mode DMD that automatically separates discrete and continuous spectral components

## Executive Summary
This work presents sparse-mode dynamic mode decomposition (DMD), a novel variant that leverages sparsity-promoting regularization to disambiguate between spatially local and global DMD modes. The algorithm automatically separates discrete (local) and continuous (global) spectral components in an unsupervised manner while maintaining noise-robustness. By utilizing variable projection with proximal gradient methods, sparse-mode DMD handles noisy, unevenly-sampled data and supports various sparse regularizers. The method successfully recovers spatiotemporal modes in synthetic video data, optical waveguide systems, and real-world sea surface temperature data, showing improved accuracy over standard DMD particularly in isolating local features like the El Niño warming band.

## Method Summary
Sparse-mode DMD extends optimized DMD by incorporating sparsity-promoting regularization to automatically separate local and global structures. The method employs variable projection to optimize over both the Koopman operator and the sparse regularization parameters simultaneously. Proximal gradient methods are used to handle the non-smooth optimization problem arising from the sparsity constraints. The algorithm is designed to work with noisy, unevenly-sampled data and supports multiple sparse regularizers including L1, L0, and group sparsity. The key innovation is the ability to automatically disambiguate between discrete (local) and continuous (global) spectral components without requiring prior knowledge of the system's structure.

## Key Results
- Successfully separates discrete and continuous spectral components in synthetic video data with mixed local/global structures
- Accurately identifies both discrete and continuous spectra in optical waveguide systems, outperforming standard DMD
- Isolates El Niño warming band as a local feature while maintaining global seasonal patterns in sea surface temperature data
- Demonstrates noise-robustness across all applications while maintaining mode quality

## Why This Works (Mechanism)
Sparse-mode DMD works by introducing sparsity constraints that encourage the decomposition to separate spatially localized structures from globally distributed ones. The variable projection method decouples the optimization of the Koopman operator from the sparse regularization parameters, making the problem more tractable. The proximal gradient method efficiently handles the non-smooth nature of sparsity-promoting regularizers. By automatically learning which modes should be sparse (local) versus dense (global), the algorithm can disambiguate structures that would be mixed in standard DMD, particularly when local and global features have similar temporal dynamics.

## Foundational Learning
**Dynamic Mode Decomposition (DMD)**: Data-driven technique for extracting spatial-temporal coherent structures from high-dimensional data; needed for understanding the baseline method being extended
Quick check: Can identify dominant spatiotemporal patterns in fluid flow data

**Koopman Operator Theory**: Infinite-dimensional linear operator that describes nonlinear dynamical systems; provides theoretical foundation for DMD
Quick check: Can represent nonlinear systems as linear evolution in an infinite-dimensional space

**Sparsity-promoting Regularization**: Techniques that encourage solutions with few non-zero components; essential for separating local from global structures
Quick check: Can recover sparse representations in compressed sensing applications

**Variable Projection Method**: Optimization technique that projects out some variables analytically; crucial for efficient optimization in sparse-mode DMD
Quick check: Can solve separable nonlinear least squares problems more efficiently than joint optimization

**Proximal Gradient Methods**: Optimization algorithms for non-smooth objective functions; necessary for handling L1 and related sparsity-inducing norms
Quick check: Can minimize composite objective functions with smooth and non-smooth parts

## Architecture Onboarding

**Component Map**: Data → Sparse-mode DMD (Variable Projection + Proximal Gradient) → Separated Local Modes + Separated Global Modes

**Critical Path**: Data preprocessing → Matrix construction → Variable projection optimization → Sparse regularization parameter selection → Mode separation → Post-processing

**Design Tradeoffs**: 
- Sparsity regularization strength vs. mode fidelity
- Computational cost of variable projection vs. separation quality
- Choice of regularizer (L1, L0, group sparsity) vs. structure preservation
- Noise robustness vs. sensitivity to local features

**Failure Signatures**: 
- Over-regularization leading to loss of global structures
- Under-regularization failing to separate local from global modes
- Mode mixing when local and global features have similar temporal frequencies
- Computational instability with very large datasets

## Limitations
- Scalability concerns with extremely large datasets due to computational demands of variable projection method
- Performance on highly nonlinear systems where Koopman operator assumptions may break down
- Potential mode mixing when local and global features have similar temporal frequencies
- Sensitivity to regularization parameter choices not extensively explored
