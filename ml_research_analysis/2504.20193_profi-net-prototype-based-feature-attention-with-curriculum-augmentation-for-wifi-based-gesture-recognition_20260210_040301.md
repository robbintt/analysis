# ProFi-Net: Prototype-based Feature Attention with Curriculum Augmentation for WiFi-based Gesture Recognition

## Quick Facts
- **arXiv ID**: 2504.20193
- **Source URL**: https://arxiv.org/abs/2504.20193
- **Reference count**: 24
- **Primary result**: ProFi-Net achieves 66.5% accuracy in 5-way 1-shot and 84.1% in 5-way 5-shot settings for WiFi-based gesture recognition

## Executive Summary
This paper introduces ProFi-Net, a novel few-shot learning framework specifically designed for WiFi-based gesture recognition. The approach addresses two key challenges in this domain: limited training data availability and sparse feature representations from CSI data. ProFi-Net combines prototype-based metric learning with a feature-level attention mechanism that dynamically refines distance calculations by emphasizing discriminative feature dimensions. The framework also introduces a curriculum-inspired data augmentation strategy applied exclusively to the query set, progressively incorporating Gaussian noise to improve generalization.

The experimental results demonstrate significant performance improvements over conventional prototype networks and other state-of-the-art few-shot learning methods. ProFi-Net achieves 66.5% accuracy in 5-way 1-shot and 84.1% in 5-way 5-shot settings, representing improvements of up to 7.1% over baseline methods. These results were validated across diverse real-world environments, showcasing the framework's effectiveness in practical scenarios.

## Method Summary
ProFi-Net employs a prototype-based metric learning architecture enhanced with feature-level attention and curriculum augmentation. The model extracts features from Channel State Information (CSI) data using convolutional layers, then generates class prototypes through a weighted averaging mechanism. The feature attention module dynamically refines Euclidean distances by emphasizing the most discriminative feature dimensions, addressing the sparsity issue in CSI representations. A novel curriculum augmentation strategy is applied exclusively to the query set, progressively introducing Gaussian noise of increasing magnitude during training. This approach improves generalization and robustness to overfitting while maintaining computational efficiency. The framework is designed to work with limited labeled data, making it suitable for real-world deployment where collecting extensive gesture samples is impractical.

## Key Results

## Why This Works (Mechanism)
The effectiveness of ProFi-Net stems from its multi-faceted approach to addressing WiFi-based gesture recognition challenges. The prototype-based metric learning architecture enables the model to learn meaningful representations from limited data by comparing query samples to class prototypes. The feature-level attention mechanism solves the sparsity problem inherent in CSI data by dynamically adjusting which feature dimensions contribute most to distance calculations. This selective emphasis on discriminative features improves the model's ability to distinguish between similar gestures. The curriculum augmentation strategy applied to the query set forces the model to generalize better by exposing it to increasingly noisy samples during training. This progressive noise injection mimics real-world variability and prevents overfitting to clean training data. The combination of these techniques creates a robust framework that can learn effectively from minimal examples while maintaining high accuracy in practical deployment scenarios.

## Architecture Onboarding

**Component Map**: CSI Input -> Feature Extraction -> Prototype Generation -> Feature Attention -> Distance Calculation -> Classification

**Critical Path**: The most critical sequence is CSI Input → Feature Extraction → Feature Attention → Distance Calculation → Classification. This path determines the final classification accuracy and computational efficiency.

**Design Tradeoffs**: The framework prioritizes generalization over raw accuracy by applying augmentation only to the query set, which reduces training complexity but may limit the model's exposure to augmented support samples. The feature attention mechanism adds computational overhead but significantly improves performance on sparse CSI data. The curriculum augmentation strategy balances between preventing overfitting and maintaining training stability.

**Failure Signatures**: Potential failures include prototype collapse when classes are too similar, attention mechanism getting stuck on noisy features, and curriculum augmentation causing training instability if noise levels increase too rapidly. The model may also struggle with cross-domain generalization if the support and query distributions differ significantly.

- The curriculum augmentation strategy's exclusive application to the query set lacks theoretical justification for its superiority over support set augmentation
