# Estimating Heterogeneous Causal Effect on Networks via Orthogonal Learning

## Quick Facts
- arXiv ID: 2509.18484
- Source URL: https://arxiv.org/abs/2509.18484
- Reference count: 40
- Primary result: Proposed orthogonal learning framework achieves state-of-the-art PEHE scores (0.03-0.05) on benchmark network interference datasets

## Executive Summary
This paper tackles the challenge of estimating heterogeneous causal effects in networked systems where treatments on one unit can spill over to affect connected units. The key innovation is a two-stage orthogonal learning framework that first estimates complex network dependencies using graph neural networks, then uses these estimates to robustly identify direct and spillover causal effects. The method achieves state-of-the-art performance on benchmark datasets while maintaining theoretical guarantees against bias from nuisance estimation errors.

## Method Summary
The approach employs a two-stage orthogonal learning framework to address network interference in causal inference. First, graph neural networks estimate nuisance components including conditional outcome means and propensity scores, capturing complex network dependencies. Second, an attention-based interference model estimates direct and spillover effects using these nuisance estimates. The orthogonal design ensures that errors in nuisance estimation contribute only at second order, making the causal effect estimates robust to bias. The framework handles observed confounders while addressing network homophily that creates spurious correlations between treatment exposure and outcomes.

## Key Results
- Achieves PEHE scores for individual direct effects (IDE) and individual spillover effects (ISE) around 0.03-0.05 on Flickr dataset, closely matching oracle performance
- Successfully recovers heterogeneous pairwise influences and identifies influential neighbors with over 80% recall
- Accurately recovers spillover sign directions in synthetic experiments
- Outperforms existing methods on benchmark datasets for network interference causal effect estimation

## Why This Works (Mechanism)
The orthogonal learning framework works by decoupling nuisance estimation from causal effect estimation through a two-stage process. By first using graph neural networks to estimate conditional outcome means and propensity scores, the method captures complex network dependencies that would otherwise confound causal effect estimation. The attention-based interference model then uses these estimates to identify direct and spillover effects while being robust to errors in the nuisance components. This orthogonal design ensures that any bias from imperfect nuisance estimation enters only at second order, preserving the accuracy of the causal effect estimates.

## Architecture Onboarding

Component Map: GNN -> Nuisance Estimators -> Attention Model -> Causal Effects

Critical Path: The causal effect estimates depend critically on accurate nuisance component estimation followed by attention-based interference modeling. The orthogonal design ensures that errors propagate at second order, making the path robust to first-stage estimation errors.

Design Tradeoffs: The two-stage approach trades computational efficiency for robustness - while more computationally intensive than single-stage methods, it provides stronger theoretical guarantees. The attention mechanism adds interpretability but increases model complexity.

Failure Signatures: Poor performance may indicate either inadequate GNN architecture for capturing network dependencies, or insufficient attention capacity to model heterogeneous spillover effects. Network structure errors (missing edges, noisy connections) can also degrade performance.

## Limitations
- Assumes accurate knowledge of network structure, may struggle with missing or noisy edge information common in real-world applications
- Computational complexity of two-stage approach may limit scalability to very large networks with millions of nodes
- Does not explicitly address unobserved confounding, which could bias estimates if network homophily creates unmeasured confounding pathways

## Confidence
- High: Methodological framework and theoretical guarantees are sound
- Medium: Empirical performance on benchmark datasets (potential overfitting concerns)
- Low: Robustness claims in real-world deployment scenarios with noisy networks and unobserved confounders
