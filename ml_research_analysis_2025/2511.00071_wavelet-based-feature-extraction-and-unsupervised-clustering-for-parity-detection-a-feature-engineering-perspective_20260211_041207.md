# Wavelet-Based Feature Extraction and Unsupervised Clustering for Parity Detection: A Feature Engineering Perspective

## Executive Summary
This study investigates whether wavelet-based feature extraction and unsupervised clustering can detect number parity (odd/even) without using modular arithmetic. Integers are converted to binary signals, decomposed using the Haar wavelet transform, and three statistical features (energy, L2 norm, mean absolute value) are extracted at multiple decomposition levels. Each feature is clustered using k-means (k=2), and an oddness score is derived from cluster assignments to classify numbers. Experiments on integers 0â€“10,000 achieve a maximum accuracy of 69.67%, significantly above the 50% random baseline. Results indicate that structural differences in binary representations are partially captured by wavelet features, revealing latent patterns correlated with parity, though the approach cannot fully replace symbolic arithmetic reasoning.

## Method Summary
The methodology converts integers to binary signals, applies Haar wavelet transform, extracts statistical features (energy, L2 norm, mean absolute value) at multiple decomposition levels, and uses k-means clustering (k=2) to derive oddness scores for parity classification. The approach processes integers 0-10,000 through this pipeline and evaluates classification accuracy.

## Key Results
- Achieved maximum accuracy of 69.67% on parity detection task
- Performance significantly exceeds 50% random baseline
- Wavelet features partially capture structural differences in binary representations correlated with parity

## Why This Works (Mechanism)
Wavelet decomposition reveals structural patterns in binary representations that correlate with parity. The Haar wavelet transform captures both frequency and location information in the binary signal, while statistical features extract relevant patterns at different scales. These patterns are then clustered to identify latent structures that distinguish odd from even numbers.
