# Enhanced Sentiment Analysis of Iranian Restaurant Reviews Utilizing Sentiment Intensity Analyzer & Fuzzy Logic

## Quick Facts
- arXiv ID: 2503.12141
- Source URL: https://arxiv.org/abs/2503.12141
- Reference count: 14
- Integrates fuzzy logic with VADER sentiment scores to reduce neutrality bias in restaurant review analysis

## Executive Summary
This study addresses the challenge of sentiment bias toward neutrality in restaurant review analysis by developing a hybrid approach that combines VADER sentiment scores with fuzzy logic transformations. The researchers applied mathematical transformations (square-root and fourth-root) to refine sentiment scores before processing them through a Fuzzy Inference System (FIS), producing continuous sentiment values rather than discrete classifications. The method significantly improves sentiment classification accuracy, better aligning with actual customer star ratings while reducing the tendency of traditional sentiment analyzers to default to neutral classifications.

## Method Summary
The approach integrates VADER sentiment intensity analyzer scores with fuzzy logic transformations to address neutrality bias in sentiment classification. The method applies square-root and fourth-root transformations to VADER sentiment scores, converting them into more nuanced continuous values. These refined scores are then processed through a Fuzzy Inference System (FIS) that uses triangular membership functions and predefined linguistic rules to produce final sentiment scores. The fuzzy system processes the transformed values through fuzzy operations and defuzzification to generate continuous sentiment outputs that better capture the full spectrum of customer opinions rather than forcing binary or neutral classifications.

## Key Results
- Square-root transformation method showed slight performance advantage over fourth-root approach
- Refined fuzzy logic approach significantly reduced neutrality bias in sentiment classification
- Sentiment classifications aligned more closely with actual customer star ratings
- Minimal misclassifications observed in refined sentiment categorization

## Why This Works (Mechanism)
The integration of fuzzy logic with sentiment scores works by transforming discrete sentiment values into continuous representations that better capture the nuanced nature of customer opinions. The mathematical transformations (square-root and fourth-root) adjust the distribution of sentiment scores, preventing the clustering around neutral values that typically occurs with standard sentiment analyzers. The Fuzzy Inference System then processes these refined scores through membership functions and linguistic rules, allowing for gradual transitions between sentiment categories rather than abrupt binary classifications.

## Architecture Onboarding
- Component Map: VADER Analyzer -> Mathematical Transformation -> Fuzzy Inference System -> Defuzzification -> Final Sentiment Score
- Critical Path: Raw Review Text -> VADER Sentiment Scores -> Square-Root/Fourth-Root Transformation -> FIS Processing -> Continuous Sentiment Output
- Design Tradeoffs: The approach sacrifices computational simplicity for improved sentiment differentiation. While VADER alone provides fast binary/neutral classifications, the fuzzy refinement adds processing overhead but captures more nuanced sentiment gradients.
- Failure Signatures: If transformations are too aggressive, sentiment scores may become overly compressed, reducing differentiation between positive and negative reviews. Conversely, insufficient transformation leaves neutrality bias intact.
- First 3 Experiments: 1) Apply both transformation methods to a small review sample and compare score distributions. 2) Test FIS with different membership function shapes on transformed scores. 3) Validate refined scores against known star ratings from the same reviews.
