# Sparse and nonparametric estimation of equations governing dynamical systems with applications to biology

## Quick Facts
- arXiv ID: 2511.00579
- Source URL: https://arxiv.org/abs/2511.00579
- Reference count: 40
- Primary result: Introduces KB-Sindy, a hybrid sparse parametric/nonparametric framework for discovering governing equations from data

## Executive Summary
This paper presents Kernel-based Sindy (KB-Sindy), a novel framework that combines sparse parametric estimation with nonparametric kernel-based techniques to discover governing equations from data. The method addresses a fundamental limitation of the original Sindy algorithm by integrating a sparse parametric component with a nonparametric kernel component, allowing it to capture complex nonlinearities without requiring a priori knowledge of their functional form. KB-Sindy is demonstrated on biological examples including gene regulation, calcium signaling, and stochastic dynamical systems, showing improved accuracy and interpretability compared to traditional approaches.

## Method Summary
KB-Sindy extends the original Sindy algorithm by modeling system dynamics as the sum of a sparse parametric part (described by a limited number of basis functions) and a nonparametric part (estimated using kernel methods). This hybrid approach allows the method to capture nonlinearities that cannot be parsimoniously described by parametric models alone. The sparse component identifies the most relevant basis functions from a predefined library, while the kernel component captures residual dynamics that are difficult to model parametrically. The method employs optimization techniques to jointly estimate both components, balancing model complexity with prediction accuracy.

## Key Results
- KB-Sindy accurately estimates system parameters and captures complex nonlinearities in biological systems
- The method outperforms the original Sindy algorithm in prediction accuracy and model interpretability
- Demonstrated effectiveness on gene regulation, spatio-temporal calcium signaling, and logistic map with stochastic forcing
- Successfully identifies governing equations without requiring a priori information about nonlinear functional forms

## Why This Works (Mechanism)
KB-Sindy works by leveraging the complementary strengths of sparse parametric and nonparametric modeling approaches. The sparse parametric component provides interpretability and parsimony by identifying the most relevant physical relationships, while the nonparametric kernel component captures complex residual dynamics that are difficult to express with simple basis functions. This hybrid structure allows the method to maintain the interpretability advantages of parametric models while avoiding their limitations in capturing arbitrary nonlinearities. The kernel component acts as a flexible residual model that can adapt to unknown functional forms, while the sparse component ensures that the discovered equations remain interpretable and physically meaningful.

## Architecture Onboarding

Component map: Data -> Kernel estimation -> Sparse regression -> Model selection -> Governing equations

Critical path: Data collection and preprocessing -> Kernel component estimation -> Sparse parametric component selection -> Joint optimization -> Model validation

Design tradeoffs: The method balances interpretability (favoring sparse parametric models) against flexibility (requiring nonparametric components), with the tradeoff controlled by regularization parameters

Failure signatures: Poor performance when system dimensionality is high, when kernel choice is inappropriate for the underlying dynamics, or when the sparse component fails to identify relevant physical relationships

First experiments: 1) Test on simple nonlinear systems with known analytical solutions; 2) Evaluate sensitivity to kernel parameter choices; 3) Compare performance across different biological system complexities

## Limitations
- Performance on high-dimensional systems with many interacting components is not fully explored
- Computational complexity scaling with system size is unclear
- Interpretability challenges may persist when the nonparametric kernel component is large
- Systematic sensitivity analysis for kernel function and parameter choices is lacking
