# Deep Sequence Models for Predicting Average Shear Wave Velocity from Strong Motion Records

## Quick Facts
- arXiv ID: 2503.05224
- Source URL: https://arxiv.org/abs/2503.05224
- Reference count: 12
- Primary result: Hybrid CNN-LSTM architecture achieves 34.63% average absolute error in Vs30 prediction

## Executive Summary
This study explores deep learning approaches for predicting Vs30 (average shear wave velocity in the top 30 meters) from strong motion records. The researchers develop a hybrid CNN-LSTM architecture that processes seismic signals to extract both spatial and temporal features. By incorporating auxiliary information such as P/S wave arrival times and employing transfer learning strategies, the model demonstrates improved prediction accuracy compared to baseline approaches. The work addresses a critical need in seismic site characterization, where accurate Vs30 values are essential for seismic hazard assessment and engineering applications.

## Method Summary
The study employs a hybrid CNN-LSTM architecture to predict Vs30 values from strong motion records. The model architecture combines convolutional neural networks for spatial feature extraction with long short-term memory networks for temporal pattern recognition. Experiments are conducted using the KiK-net database, with systematic exploration of different input representations including signal segmentation around P-wave and PGA regions. The research investigates the impact of incorporating P/S wave arrival time information and applies transfer learning techniques to enhance model performance. Performance is evaluated across different site classes and regional clusters to assess consistency and generalizability.

## Key Results
- Hybrid CNN-LSTM architecture achieves average absolute error of 34.63% across site classes
- Transfer learning with P/S wave information provides optimal performance improvement
- Regional analysis shows Clusters 2 and 4 demonstrate more consistent predictions
- CNN-LSTM outperforms single architecture approaches in capturing seismic signal characteristics

## Why This Works (Mechanism)
The CNN-LSTM hybrid architecture effectively captures both spatial patterns in the frequency domain (through convolutional layers) and temporal dependencies in the seismic signal sequence (through LSTM layers). This dual capability is essential because Vs30 prediction requires understanding both the spectral content of ground motion and its temporal evolution. The convolutional layers identify local patterns and features within the seismic waveforms, while the LSTM layers model the sequential dependencies that are crucial for characterizing site response. The incorporation of P/S wave arrival information provides critical temporal anchors that help the model distinguish between different seismic phases and better understand the site's response characteristics.

## Architecture Onboarding

**Component Map:**
Raw seismic signal -> CNN feature extractor -> LSTM temporal encoder -> Dense layers -> Vs30 prediction

**Critical Path:**
The critical path flows from raw seismic input through CNN layers that extract spatial features, followed by LSTM layers that model temporal dependencies, and finally through dense layers that map the learned representations to Vs30 values. The P/S wave arrival information is incorporated as auxiliary input to provide temporal context.

**Design Tradeoffs:**
- CNN-LSTM vs. pure CNN: LSTM adds temporal modeling capability but increases complexity
- Signal segmentation strategies: P-wave centered vs. PGA centered approaches capture different aspects of site response
- Auxiliary information inclusion: P/S arrival times improve accuracy but require additional processing steps
- Transfer learning approach: Pretraining on related tasks improves performance but requires careful architecture design

**Failure Signatures:**
- Overfitting to regional characteristics when training data is geographically concentrated
- Poor performance on sites with Vs30 values outside the training distribution
- Sensitivity to noise in P/S wave arrival time estimation
- Reduced accuracy for extreme site classes (very soft or very hard soil conditions)

## Limitations
- Regional performance varies significantly, suggesting potential overfitting to training data characteristics
- 34.63% average absolute error represents substantial uncertainty for critical engineering applications
- Reliance on KiK-net database may limit generalizability to global site conditions
- Limited discussion of individual prediction errors and outlier cases
