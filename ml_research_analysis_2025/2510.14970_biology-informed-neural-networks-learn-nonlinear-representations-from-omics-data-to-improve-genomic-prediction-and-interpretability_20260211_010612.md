# Biology-informed neural networks learn nonlinear representations from omics data to improve genomic prediction and interpretability

## Quick Facts
- **arXiv ID**: 2510.14970
- **Source URL**: https://arxiv.org/abs/2510.14970
- **Reference count**: 40
- **Primary result**: BINNs achieve up to 56% improvement in rank-correlation accuracy compared to conventional GBLUP models by embedding pathway-level inductive biases derived from multi-omics data.

## Executive Summary
Biology-informed neural networks (BINNs) integrate omics data and biological knowledge into neural network architectures to improve genomic prediction accuracy and interpretability. The framework embeds pathway-level inductive biases derived from multi-omics data to shape network sparsity and connectivity, enabling capture of nonlinear genotype-to-phenotype relationships without requiring omics measurements at inference time. BINNs use prior biological knowledge (e.g., gene expression and pathway information) to construct sparse neural network architectures that constrain connections between SNPs and latent biological entities, improving prediction accuracy while reducing overfitting.

## Method Summary
BINNs construct sparse neural network architectures by embedding biological pathway information as architectural constraints. The method first uses ElasticNet regression on gene expression data to select relevant genes, then applies eQTL mapping to link SNPs to these genes. This creates a "mask" that initializes the neural network with zeros for irrelevant connections and allows training only on biologically informed pathways. The network consists of pathway-specific subnetworks (small fully connected networks) that process relevant SNPs, whose outputs concatenate into an integrator network that predicts the phenotype. The framework supports both supervised training (predicting phenotype directly) and semi-supervised approaches using soft constraints to align latent variables with biological entities.

## Key Results
- On synthetic metabolomics benchmark with known pathways, BINNs reduced prediction error by 75% and correctly identified the most important nonlinear pathway (sucrose)
- Applied to maize gene-expression data, BINNs achieved up to 56% improvement in rank-correlation accuracy compared to conventional GBLUP models
- Sensitivity analyses revealed that BINN latent variables correlated with the biological entities they represent, even when not explicitly trained on them

## Why This Works (Mechanism)
BINNs work by encoding biological knowledge as architectural constraints that shape the neural network's inductive bias. By using pathway information to define which SNPs connect to which latent biological entities, the network learns biologically meaningful representations while maintaining sparsity. This prevents overfitting on high-dimensional genotype data and enables the model to capture nonlinear relationships between genetic variants and phenotypes through intermediate biological layers. The biological priors act as regularizers that guide the learning process toward physiologically plausible solutions.

## Architecture Onboarding

**Component Map**: SNPs -> Pathway Subnetworks -> Integrator FCN -> Phenotype

**Critical Path**: The pathway-specific subnetworks are the critical component where biological knowledge is embedded. These subnetworks receive only the SNPs mapped to their corresponding genes/metabolites, process them through biologically informed architectures, and their outputs feed into the integrator network. The quality of the eQTL mapping and gene selection directly determines the effectiveness of these subnetworks.

**Design Tradeoffs**: The framework trades model flexibility for biological interpretability. While standard neural networks can learn any function given enough parameters, BINNs sacrifice some representational capacity to enforce biologically plausible structures. This improves generalization when biological priors are accurate but may underperform when priors are noisy or incomplete. The choice between supervised (BINN-MSE) and semi-supervised (BINN-soft) training represents another tradeoff between prediction accuracy and latent variable interpretability.

**Failure Signatures**: 
- If BINN performs worse than Ridge Regression, the biological mask is likely poor quality (genes selected don't correlate with phenotype)
- If latent variables don't correlate with ground truth metabolites, noise levels may be too high or the soft-constraint loss λ is mis-tuned
- If training is unstable, the sparse initialization may create dead neurons that never activate

## Limitations
- BINN performance depends heavily on the quality of biological priors, degrading when pathway information is noisy or incomplete
- The framework requires multi-omics data for training but cannot use omics measurements at inference time, limiting its applicability in some breeding scenarios
- Specific hyperparameters like the soft-constraint loss weight λ are not fully specified, introducing implementation variability
