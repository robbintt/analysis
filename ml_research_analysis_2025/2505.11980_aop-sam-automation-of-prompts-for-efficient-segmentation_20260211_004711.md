# AoP-SAM: Automation of Prompts for Efficient Segmentation
## Quick Facts
- arXiv ID: 2505.11980
- Source URL: https://arxiv.org/abs/2505.11980
- Reference count: 8
- The paper proposes AoP-SAM, a method for automating prompt generation for the Segment Anything Model (SAM) to improve efficiency and accuracy.

## Executive Summary
The paper addresses the challenge of automating prompt generation for the Segment Anything Model (SAM), which is typically resource-intensive and impractical for real-world applications due to the need for manual input. The proposed method, AoP-SAM, introduces a lightweight Prompt Predictor that integrates with SAM to automatically generate essential prompts by leveraging image embeddings, preserving SAM's zero-shot generalization. Additionally, an Adaptive Sampling and Filtering (ASF) mechanism is employed to refine prompt selection, reducing redundant mask generations and improving efficiency. Evaluations on three datasets demonstrate that AoP-SAM significantly enhances both prompt generation efficiency and mask generation accuracy, achieving higher mIoU scores while reducing inference latency and memory usage compared to existing methods.

## Method Summary
AoP-SAM automates prompt generation for SAM by integrating a lightweight Prompt Predictor that leverages image embeddings to generate essential prompts, preserving SAM's zero-shot generalization. The method employs an Adaptive Sampling and Filtering (ASF) mechanism to refine prompt selection, reducing redundant mask generations and improving efficiency. Evaluations on three datasets demonstrate that AoP-SAM significantly enhances both prompt generation efficiency and mask generation accuracy, achieving higher mIoU scores while reducing inference latency and memory usage compared to existing methods.
