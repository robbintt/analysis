# Residual-SwinCA-Net: A Channel-Aware Integrated Residual CNN-Swin Transformer for Malignant Lesion Segmentation in BUSI

## Quick Facts
- arXiv ID: 2512.08243
- Source URL: https://arxiv.org/abs/2512.08243
- Reference count: 0
- Dice score of 0.9041 achieved on BUSI dataset

## Executive Summary
This study presents Residual-SwinCA-Net, a hybrid deep learning framework designed for accurate segmentation of malignant lesions in breast ultrasound images. The model combines residual CNN blocks for robust local feature extraction with customized Swin Transformer blocks to capture global contextual dependencies. The framework incorporates Laplacian-of-Gaussian and boundary-aware operators for noise suppression and edge preservation, along with a Multi-Scale Channel Attention and Squeezing module for adaptive feature refinement. A Pixel-Attention mechanism enhances lesion-specific spatial cues while suppressing background interference. Evaluated on the BUSI dataset, the approach achieved state-of-the-art performance with a Dice score of 0.9041, IoU of 98.74%, and mean accuracy of 99.29%.

## Method Summary
The proposed Residual-SwinCA-Net integrates residual convolutional neural network blocks with Swin Transformer architectures to leverage both local and global feature extraction capabilities. The model employs Laplacian-of-Gaussian and boundary-aware operators for preprocessing to enhance edge preservation and reduce noise. A Multi-Scale Channel Attention and Squeezing (MSCAS) module dynamically refines features by emphasizing lesion-specific channels. Pixel-level attention mechanisms further enhance spatial discrimination of malignant regions. The hybrid architecture aims to address the challenges of accurate lesion boundary delineation and contextual understanding in breast ultrasound imaging.

## Key Results

## Why This Works (Mechanism)
The integration of residual CNN blocks with Swin Transformer modules enables the model to capture both fine-grained local features and broader contextual relationships essential for accurate lesion segmentation. The MSCAS module dynamically adjusts channel-wise feature importance, improving the model's ability to distinguish between malignant and benign tissue. The Pixel-Attention mechanism further refines spatial localization by emphasizing lesion-specific regions while suppressing irrelevant background information. The combination of LoG and boundary-aware preprocessing enhances edge preservation, which is critical for accurate boundary delineation in medical imaging.

- **Pixel-Attention Mechanism**: Enhances spatial localization by weighting individual pixel contributions. Quick check: Ensure attention maps highlight lesion boundaries accurately.

## Architecture Onboarding

Component Map: Input Images -> LoG/Boundary-Aware Preprocessing -> Residual CNN Blocks -> MSCAS Module -> Swin Transformer Blocks -> Pixel-Attention -> Output Segmentation

Critical Path: The critical path involves feature extraction through residual CNN blocks, channel-wise refinement via MSCAS, global context capture through Swin Transformers, and final spatial refinement through Pixel-Attention.

Design Tradeoffs: The hybrid architecture balances computational complexity with segmentation accuracy. While the combination of CNN and Transformer modules provides superior performance, it increases model complexity and inference time compared to single-architecture approaches.

Failure Signatures: Potential failure modes include overfitting to the BUSI dataset due to its limited size, failure to generalize across different ultrasound imaging protocols, and computational inefficiency that may prevent real-time clinical deployment.
