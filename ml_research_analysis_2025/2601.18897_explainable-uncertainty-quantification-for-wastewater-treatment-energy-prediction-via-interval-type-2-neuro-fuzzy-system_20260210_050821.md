# Explainable Uncertainty Quantification for Wastewater Treatment Energy Prediction via Interval Type-2 Neuro-Fuzzy System

## Quick Facts
- arXiv ID: 2601.18897
- Source URL: https://arxiv.org/abs/2601.18897
- Authors: Qusai Khaled; Bahjat Mallak; Uzay Kaymak; Laura Genga
- Reference count: 36
- Key outcome: IT2-ANFIS provides both accurate predictions and explainable uncertainty quantification for wastewater treatment plant energy consumption

## Executive Summary
This study addresses the challenge of energy consumption prediction in wastewater treatment plants, which account for 1-3% of global electricity use. The research develops an Interval Type-2 Adaptive Neuro-Fuzzy Inference System (IT2-ANFIS) that provides both accurate predictions and explainable uncertainty quantification. Unlike black-box models, IT2-ANFIS uses fuzzy rule structures to decompose uncertainty across three levels: feature-level analysis identifying ambiguous variables, rule-level confidence assessment, and instance-level prediction intervals. Validated on Melbourne Water's Eastern Treatment Plant dataset, IT2-ANFIS achieves comparable predictive performance to first-order ANFIS (test MSE ≈1600 MWh) with substantially reduced variance across training runs. The framework enables operators to identify uncertain operating conditions, prioritize data collection in problematic input regions, and make risk-aware decisions about maintenance and alarm handling.

## Method Summary
The proposed approach combines interval type-2 fuzzy sets with ANFIS architecture to create a neuro-fuzzy system capable of uncertainty quantification. The model uses fuzzy membership functions with uncertainty intervals at the input layer, allowing for representation of ambiguous or imprecise data. Through hybrid learning (backpropagation for premise parameters, least squares for consequent parameters), the system optimizes both prediction accuracy and uncertainty representation. The key innovation lies in the interval type-2 fuzzy inference mechanism, which propagates uncertainty through the rule base and generates prediction intervals rather than point estimates. This architecture maintains interpretability through explicit fuzzy rules while providing quantifiable uncertainty measures at multiple levels of the prediction process.

## Key Results
- IT2-ANFIS achieves comparable predictive performance to first-order ANFIS with test MSE ≈1600 MWh
- The model demonstrates substantially reduced variance across training runs compared to traditional ANFIS
- Three-level uncertainty decomposition enables operators to identify uncertain operating conditions and prioritize data collection in problematic input regions

## Why This Works (Mechanism)
The mechanism works by leveraging interval type-2 fuzzy sets, which extend type-1 fuzzy sets by allowing uncertainty in the membership function itself. This creates a three-dimensional membership structure (x, μ, α) where x is the input, μ is the primary membership, and α represents the secondary membership function. During inference, the footprint of uncertainty (FOU) captures the range of possible memberships for each input, and type-reduction methods like Karnik-Mendel algorithm collapse this to a representative interval. The neuro-fuzzy learning process simultaneously optimizes the rule base for accuracy while maintaining uncertainty representation. The decomposition into feature, rule, and instance levels allows operators to trace uncertainty sources back to specific variables or operating conditions, making the model's confidence estimates actionable for decision-making.

## Architecture Onboarding

Component map: Input variables → Fuzzification (Type-2) -> Rule Evaluation -> Type Reduction -> Defuzzification -> Prediction Intervals

Critical path: Data preprocessing → Type-2 fuzzification → Rule firing strength computation → Type reduction (Karnik-Mendel) → Output interval calculation

Design tradeoffs: The interval type-2 approach trades computational complexity for uncertainty quantification and interpretability. While traditional ANFIS offers faster inference, IT2-ANFIS provides prediction intervals and uncertainty decomposition at the cost of increased computation during both training and inference.

Failure signatures: Poor uncertainty calibration indicates inadequate coverage of the FOU or insufficient rule granularity. High variance in prediction intervals suggests the model is either overly conservative or unable to adequately characterize uncertainty in certain input regions.

## Limitations
- Evaluation on a single wastewater treatment plant dataset restricts generalizability to other facilities with different operational characteristics
- Variance reduction claim lacks quantitative comparison - exact numerical differences in variance across training runs are not reported
- Explainable uncertainty quantification relies on subjective interpretation of fuzzy rule structures that may vary between domain experts
