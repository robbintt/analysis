# On the Complexity of the Grounded Semantics for Infinite Argumentation Frameworks

## Quick Facts
- arXiv ID: 2511.22376
- Source URL: https://arxiv.org/abs/2511.22376
- Reference count: 39
- Primary result: Grounded extension membership is Π₁¹-complete for computable infinite argumentation frameworks

## Executive Summary
This paper investigates the grounded semantics for infinite argumentation frameworks using tools from computability theory and set theory. The authors show that while grounded semantics is polynomial-time computable for finite frameworks, it becomes maximally complex (Π₁¹-complete) for computable infinite frameworks. They establish that the grounding ordinal is bounded by ω₁^CK for computable frameworks and characterize which Turing degrees can arise as grounded extensions. The work provides the first comprehensive analysis of grounded semantics complexity in the infinite setting.

## Method Summary
The authors employ tree-rank analysis to connect groundedness to ranks of computable trees, enabling complexity classification. They construct a computable argumentation framework F_T from any computably enumerable tree T, then reduce the Π₁¹-complete problem of path existence in T to grounded membership in F_T. The method uses transfinite iteration of the defense operator, with tree-rank analysis providing bounds on the required ordinal stages. For set-theoretic results, they use forcing arguments to characterize which Turing degrees can arise as grounded extensions.

## Key Results
- Grounded membership for computable AFs is Π₁¹-complete
- Grounding ordinal for any computable AF is at most ω₁^CK
- Not every Π₁¹ Turing degree can arise as a grounded extension degree
- KP set theory suffices to define grounded extension without full Axiom of Choice

## Why This Works (Mechanism)

### Mechanism 1
- Claim: Tree-rank analysis bounds the grounding ordinal of computable argumentation frameworks at ω₁^CK (Church-Kleene ordinal).
- Mechanism: For each argument a, construct a computable tree T_a where paths encode self-defending extensions containing attackers of a. The rank of T_a directly corresponds to the least ordinal α where a ∈ G_{α+1}. By Theorem 12, any pathless computable tree has rank < ω₁^CK, giving the upper bound.
- Core assumption: The attack relation is computable (Definition 8); trees are computably enumerable.
- Evidence anchors:
  - [abstract] "We identify the exact ordinal number corresponding to the length of this iterative process"
  - [Section 4.1] "For any ρ ∈ T_S, if the rank of ρ is ≤ α, then there is a member of G_{α+1} which attacks some element of mran_S(ρ)"
  - [corpus] Related work on finitary argumentation (arxiv:2508.16986) explores computational tractability but for restricted subclasses.
- Break condition: If AF is not computable, the bound relativizes to ω₁^CK(d) for Turing degree d (Theorem 36).

### Mechanism 2
- Claim: The grounded extension equals G_{ω₁^CK} for any computable AF via transfinite fixed-point iteration.
- Mechanism: Define G_0 = ∅, G_{β+1} = f_F(G_β) (defense function), and G_λ = ⋃_{β<λ} G_β for limit ordinals. The rank analysis (Mechanism 1) guarantees every grounded argument appears by stage < ω₁^CK; conversely, Theorems 20-22 show this bound is sharp.
- Core assumption: Countable argument set; well-foundedness of ordinals (Theorem 4).
- Evidence anchors:
  - [abstract] "the grounded extension is defined as the least fixed-point of a defense operator, which for infinite frameworks requires transfinite iterations"
  - [Section 2.1] Definition 5 gives the inductive construction; Corollary 38 characterizes grounding ordinals for countable AFs.
  - [corpus] Corpus focuses on finite/tractable settings; no direct contradiction.
- Break condition: For uncountable AFs, grounding ordinal can exceed ω₁^CK (Observation 7 notes countability constraint).

### Mechanism 3
- Claim: Grounded membership for computable AFs is Π₁¹-complete, achieved by reducing tree-path existence to non-groundedness.
- Mechanism: Construct a single computable AF F as a disjoint union of frameworks F_T for all c.e. trees T (Lemma 23). Argument a_{λ_T} is grounded iff T has no path. Since pathless-tree indices form a Σ₁¹-complete set, grounded membership is Π₁¹-complete.
- Core assumption: Kleene's theorem (Theorem 16) that Σ₁¹ sets exactly correspond to path-existence in computable trees.
- Evidence anchors:
  - [Section 3] "Lemma 23: There is a computable AF F isomorphic to the disjoint union of {F_T : T a computably enumerable tree}"
  - [Section 2.2] Definition 17 formalizes Π₁¹-completeness via many-one reducibility.
  - [corpus] No corpus papers address Π₁¹ complexity directly.
- Break condition: If AF is not computable, complexity relativizes to Π₁¹(d).

## Architecture Onboarding

- Component map:
  AF representation -> Defense operator -> Transfinite iteration -> Tree construction -> Rank analysis

- Critical path:
  1. Verify AF is computable (has computable index e).
  2. For argument a, construct T_a via T_{S} subtrees for attackers.
  3. Determine if T_a has a path (Σ₁¹ query); if not, compute rank to find grounding stage.
  4. Accumulate G_{ω₁^CK} via transfinite iteration if full extension needed.

- Design tradeoffs:
  - Computable vs. general AFs: Computable AFs have bounded grounding ordinal; general countable AFs can have arbitrary countable ordinals.
  - Direct iteration vs. tree-rank: Iteration is constructive but may not halt; tree-rank gives stage bounds but requires higher-order reasoning.
  - Assumption: KP set theory suffices (Theorem 39), avoiding full Axiom of Choice.

- Failure signatures:
  - Non-computable attack relations: Grounding ordinal may exceed ω₁^CK; complexity bounds fail.
  - Uncountable argument sets: Observation 7's cardinality argument fails; grounding ordinal may be uncountable.
  - Missing Π₁¹ uniformly self-witnessing property: Not every Π₁¹ set is realizable as a grounded extension degree (Theorem 43).

- First 3 experiments:
  1. Implement the F_T construction for small computable trees T; verify Theorem 19's rank-to-stage correspondence empirically.
  2. Build a finitary AF (each argument finitely attacked) and confirm grounding completes in ω steps with G = G_ω.
  3. Attempt to construct a computable AF with grounding ordinal ω·2 using the Baumann-Spanring pattern (Figure 1); validate against Theorem 21.

## Open Questions the Paper Calls Out

- Question: Is the precise characterization of the collection of Turing degrees of grounded extensions in computable AFs exactly the degrees of sets which are Π₁¹ uniformly self-witnessing by paths?
- Basis in paper: [explicit] "A natural direction for future research is to characterize exactly which Turing degrees contain grounded extensions of computable AFs. Is it the degrees of sets which are Π₁¹ uniformly self-witnessing by paths?"
- Why unresolved: The paper proves that not all Π₁¹ Turing degrees can arise as grounded extensions (via the uniform self-witnessing property), but leaves the full characterization open.
- What evidence would resolve it: A proof that every Π₁¹ uniformly self-witnessing-by-paths set is the grounded extension of some computable AF, or a counterexample showing additional restrictions apply.

- Question: What is the complexity of other standard argumentation semantics (preferred, stable, complete) for infinite computable AFs?
- Basis in paper: [inferred] The paper focuses exclusively on grounded semantics, noting that preferred and naive semantics require the Axiom of Choice for existence (citing Spanring), but does not analyze their computational complexity.
- Why unresolved: The technical approach using tree-rank analysis is specifically tailored to the least fixed-point structure of the grounded extension.
- What evidence would resolve it: Complexity classifications (Π₁¹-completeness or otherwise) for skeptical/credulous acceptance under other semantics in the infinite computable setting.

- Question: Can the Π₁¹-completeness result be strengthened by constructing a single computable AF whose grounded extension has exactly a given Π₁¹ degree (rather than just showing some degrees are impossible)?
- Basis in paper: [inferred] Theorem 2 shows there exists a computable AF with Π₁¹-complete grounded extension, but the characterization of achievable degrees remains partial.
- Why unresolved: The paper shows restrictions via the self-witnessing property but does not fully characterize which degrees within that class are realizable.
- What evidence would resolve it: A precise algebraic or descriptive-set-theoretic characterization of the realizable degrees.

## Limitations
- The analysis is restricted to countable argumentation frameworks; uncountable cases are not fully characterized
- Tree-rank analysis depends critically on the computability of the attack relation
- The forcing argument for degree characterization relies on advanced set-theoretic machinery
- Practical implementation implications of KP definability without Axiom of Choice remain unclear

## Confidence
- Π₁¹-completeness result: High confidence - well-supported by tree-rank reduction argument
- Turing degree characterization: Medium-High confidence - established via forcing argument but harder to verify independently
- Grounding ordinal bounds: High confidence - follows directly from tree-rank analysis
- KP definability result: Medium confidence - set-theoretic result with limited independent verification
