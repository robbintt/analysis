# Surfel-based 3D Registration with Equivariant SE(3) Features

## Quick Facts
- arXiv ID: 2508.20789
- Source URL: https://arxiv.org/abs/2508.20789
- Reference count: 34
- Primary result: Rotation error of 0.25 degrees, translation error of 7.03 cm, and registration recall of 99.85% on KITTI dataset

## Executive Summary
This paper introduces a novel surfel-based approach for 3D point cloud registration that incorporates point orientations and uncertainties. The method initializes surfels from LiDAR point clouds using virtual perspective camera parameters and employs an SE(3) equivariant convolutional network to learn both position and rotation features. The approach demonstrates superior performance compared to state-of-the-art methods while maintaining low model complexity, achieving high accuracy in both rotation and translation metrics on the KITTI dataset.

## Method Summary
The proposed method transforms LiDAR point clouds into surfel representations by leveraging virtual perspective camera parameters, capturing both geometric information and uncertainty estimates. An SE(3) equivariant convolutional network is then used to process these surfels, consisting of an equivariant convolutional encoder, cross-attention mechanism, and fully-connected decoder. The network learns rotation-equivariant features that enable accurate 6-DoF registration. A non-linear Huber loss function is employed for training, which provides robustness to outliers while maintaining differentiability.

## Key Results
- Rotation error of 0.25 degrees on KITTI dataset
- Translation error of 7.03 cm on KITTI dataset
- Registration recall of 99.85% on KITTI dataset
- Outperforms state-of-the-art methods while maintaining low model complexity

## Why This Works (Mechanism)
The method's effectiveness stems from its ability to capture rich geometric information through surfels, which include orientation and uncertainty data beyond simple point coordinates. The SE(3) equivariant network architecture ensures that learned features transform consistently under rigid body motions, directly addressing the 6-DoF registration problem. The cross-attention mechanism allows the network to focus on relevant correspondences while the equivariant convolutional layers maintain geometric consistency throughout the feature extraction process.

- Tradeoff between equivariance enforcement and network expressivity
- Choice of non-linear loss function for robustness vs. optimization stability

- Poor surfel initialization leading to geometric artifacts
- Loss of equivariance in deeper network layers
- Cross-attention failing to identify correct correspondences
