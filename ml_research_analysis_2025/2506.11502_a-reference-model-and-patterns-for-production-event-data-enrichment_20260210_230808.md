# A Reference Model and Patterns for Production Event Data Enrichment

## Quick Facts
- arXiv ID: 2506.11502
- Source URL: https://arxiv.org/abs/2506.11502
- Authors: Mark van der Pas; Remco Dijkman; Alp Akçay; Ivo Adan; John Walker
- Reference count: 40
- Primary result: A reference model and generic patterns for enriching heterogeneous production event data, implemented as SPARQL templates and validated across four industrial datasets

## Executive Summary
This paper addresses the challenge of extracting hidden information from heterogeneous production event data in manufacturing environments. The authors propose a reference model based on ISA-95 and Event Knowledge Graph formalism, along with a collection of generic patterns for enriching production event data. The patterns, derived from empirical observations across four industrial data sets, include operations like counting events over intervals, calculating elapsed time, and deriving relationships between entities and events. The work is implemented as SPARQL query templates, with a dummy data set and evaluation showing broad applicability across different manufacturing settings. Most patterns apply to at least three data sets, demonstrating their relevance and practical utility. The approach offers a systematic, standardized method to uncover valuable insights from event data, improving efficiency and data quality in production monitoring and analysis.

## Method Summary
The authors developed a reference model for production event data enrichment by combining ISA-95 standards with Event Knowledge Graph formalism. They identified generic enrichment patterns through empirical analysis of four industrial datasets, covering operations such as counting events over time intervals, calculating elapsed time between events, and deriving relationships between entities and events. These patterns were implemented as SPARQL query templates, allowing systematic application across different manufacturing environments. The approach uses a dummy dataset for initial validation and demonstrates broad applicability through pattern coverage across multiple datasets.

## Key Results
- Reference model based on ISA-95 and Event Knowledge Graph formalism successfully captures production event data semantics
- 12 generic enrichment patterns derived from empirical observations across four industrial datasets
- SPARQL query templates provide systematic implementation approach for pattern application
- Most patterns apply to at least three of four datasets, demonstrating broad practical utility
- Approach enables uncovering valuable insights from event data to improve production monitoring and analysis

## Why This Works (Mechanism)
The approach works by providing a standardized semantic framework (ISA-95 + Event Knowledge Graph) that can capture the heterogeneous nature of production event data across different manufacturing environments. The generic patterns serve as reusable building blocks that can be applied to extract meaningful information from raw event streams. By implementing these patterns as SPARQL query templates, the authors create a systematic method that can be consistently applied across different datasets while maintaining flexibility for domain-specific adaptations. The empirical derivation of patterns from multiple industrial datasets ensures that the identified enrichment operations represent common needs across manufacturing contexts.
