# Landslide Detection and Mapping Using Deep Learning Across Multi-Source Satellite Data and Geographic Regions

## Quick Facts
- arXiv ID: 2507.01123
- Source URL: https://arxiv.org/abs/2507.01123
- Reference count: 19
- Primary result: ResNet34-based U-Net achieves F1 score of 0.7470 for landslide detection across four geographic regions

## Executive Summary
This study presents a comprehensive framework for landslide detection using deep learning with multi-source satellite imagery. The authors evaluate various segmentation architectures on the Landslide4Sense dataset, which contains labeled patches from four distinct geographic regions. The framework integrates Sentinel-2 multispectral data with ALOS PALSAR-derived elevation and slope layers. The ResNet34-based U-Net model emerges as the top performer, achieving strong balance between precision (0.7737) and recall (0.7267). A Streamlit-based frontend enables interactive landslide prediction using pre-trained models.

## Method Summary
The framework uses multi-source satellite data (14 channels total: Sentinel-2 bands 1-12 plus ALOS PALSAR DEM and slope layers) to train deep learning models for pixel-wise landslide classification. The study evaluates multiple segmentation architectures including U-Net, DeepLabV3+, and various encoder backbones (ResNet34, VGG16, EfficientNet-B0). Models are trained on 128×128 pixel patches from the Landslide4Sense dataset using Adam optimizer with combined BCE+Dice loss, early stopping, and data augmentation. The ResNet34-based U-Net achieves the highest performance with F1 score of 0.7470.

## Key Results
- ResNet34-based U-Net achieves highest performance with F1 score of 0.7470
- Model balances precision (0.7737) and recall (0.7267) effectively
- Multi-source data integration (Sentinel-2 + ALOS PALSAR) enhances detection accuracy
- Streamlit-based frontend enables interactive landslide prediction

## Why This Works (Mechanism)
The framework works by combining high-resolution optical imagery with elevation and slope data to capture both spectral and topographic landslide signatures. The U-Net architecture with residual connections enables effective feature extraction and precise boundary delineation. The combined BCE+Dice loss addresses class imbalance while maintaining segmentation accuracy. Multi-source data integration provides complementary information that improves detection across diverse geographic conditions and landslide types.
