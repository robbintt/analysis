# Gateways to Tractability for Satisfiability in Pearl's Causal Hierarchy

## Quick Facts
- arXiv ID: 2511.08091
- Source URL: https://arxiv.org/abs/2511.08091
- Reference count: 40
- Key result: SATlinprob is in FPT with respect to combined parameter d + tw(ϕ), and SATcausal is NP-complete even with d = 2 and pairwise vertex-disjoint edges

## Executive Summary
This paper investigates the computational complexity of satisfiability problems within Pearl's Causal Hierarchy (PCH), a framework for reasoning about probabilistic, interventional, and counterfactual statements. The authors explore whether treewidth-based algorithms, effective for Boolean satisfiability and constraint satisfaction problems, can be applied to PCH satisfiability problems. They establish the first parameterized algorithms and matching lower bounds for these problems, mapping the boundaries of tractability within the expressivity matrix of PCH.

## Method Summary
The authors employ a novel algorithmic approach that avoids the typical dynamic programming methods used for treewidth-based algorithms. Instead, they leverage structural characterizations of well-formed causal models and utilize linear programming to verify the existence of structural causal models (SCMs) while avoiding exponential dependencies on input size. This innovative technique enables the development of fixed-parameter tractable (FPT) and XP algorithms for various PCH satisfiability problems, providing both theoretical advances and practical insights into causal reasoning complexity.

## Key Results
- SATlinprob is in FPT with respect to the combined parameter d + tw(ϕ), and in XP with respect to tw(ϕ) alone
- SATlinprob is W[1]-hard with respect to the number of variables n
- SATcausal is NP-complete even if d = 2 and all edges in Gϕ are pairwise vertex-disjoint
- SATlincounterfact is in FPT with respect to the combined parameter d + n

## Why This Works (Mechanism)
The paper's approach works by exploiting structural properties of causal models and using linear programming to verify the existence of SCMs. This avoids the exponential blow-up typically associated with dynamic programming methods for treewidth-based algorithms. By focusing on the interplay between causal hierarchy levels and structural graph parameters, the authors identify specific parameter combinations that enable tractable algorithms while establishing matching lower bounds that define the boundaries of parameterized tractability.

## Foundational Learning
- Treewidth-based algorithms: Fundamental technique for designing efficient algorithms on graphs with bounded treewidth; needed for extending SAT tractability results to causal hierarchies; quick check: verify graph decomposition into tree structure
- Structural causal models (SCMs): Formal framework for representing causal relationships; essential for defining PCH satisfiability problems; quick check: validate model consistency with intervention and counterfactual constraints
- Parameterized complexity theory: Framework for analyzing computational problems with respect to multiple parameters; crucial for establishing FPT and XP results; quick check: identify appropriate parameter combinations for tractability
