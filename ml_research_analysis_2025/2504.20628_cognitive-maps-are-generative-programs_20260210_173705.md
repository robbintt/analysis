# Cognitive maps are generative programs
## Quick Facts
- arXiv ID: 2504.20628
- Source URL: https://arxiv.org/abs/2504.20628
- Reference count: 11
- Primary result: Human cognitive maps are represented as generative programs that exploit structure, enabling more efficient navigation through structured environments

## Executive Summary
This study proposes that human cognitive maps are not direct spatial representations but rather generative programs that exploit structural regularities in environments. The researchers developed a computational model using LLMs to identify repeated fragments in structured mazes and plan piecewise policies within each fragment. Through behavioral experiments, they demonstrated that people navigate structured environments using modular strategies that align with their model's predictions but differ from both optimal planning and heuristic approaches with limited planning horizons. The model achieved superior computational efficiency and memory usage while better predicting human behavior than traditional unstructured planning algorithms.

## Method Summary
The researchers developed a computational model that synthesizes programs to identify repeated structural fragments in mazes using an LLM. The model then plans piecewise policies within each identified fragment. They conducted behavioral experiments where participants navigated both structured and unstructured virtual environments. The study compared human performance and strategies against the generative model, optimal planning algorithms, and heuristic models with limited planning horizons. The experiments tested navigation efficiency, strategy patterns, and computational requirements across different maze structures.

## Key Results
- Participants used highly modular navigation strategies in structured environments that matched the generative model's predictions
- The generative model required drastically less memory than traditional planning approaches while maintaining accuracy
- Human behavior in structured environments diverged from both optimal planning solutions and heuristic models with limited horizons

## Why This Works (Mechanism)
The mechanism works because human cognitive systems naturally exploit environmental structure through pattern recognition and abstraction. By identifying repeated fragments and treating them as reusable components, the brain can compress spatial information into generative programs rather than storing exhaustive maps. This programmatic representation allows for efficient generalization across similar situations and reduces cognitive load during navigation tasks.

## Architecture Onboarding
Component map: LLM Fragment Identification -> Program Synthesis -> Piecewise Policy Planning -> Navigation Execution

Critical path: The system identifies structural fragments using the LLM, synthesizes a program that encodes these patterns, then executes piecewise policies for each fragment during navigation.

Design tradeoffs: The model trades computational overhead for pattern recognition against memory efficiency gains. Using an LLM for fragment identification provides flexibility but introduces latency and dependency on external models.

Failure signatures: The approach may fail in environments lacking clear structural patterns, when fragments are too complex to generalize, or when environmental changes invalidate learned patterns.

## Limitations
- Experiments conducted in simplified grid-based virtual environments that may not reflect real-world navigation complexity
- Limited exploration of whether participants used memorization versus true generative representations
- Focus on performance differences between structured and unstructured environments without broader ecological validation

## Confidence
- Human cognitive maps use generative programs: Medium
- Model outperforms traditional planning in predicting human behavior: Medium
- Computational efficiency and memory claims: Medium

## Next Checks
1. Replication of behavioral findings in more ecologically valid virtual environments with richer visual and spatial complexity
2. Neuroimaging studies to identify neural signatures of programmatic cognitive map representations during navigation tasks
3. Cross-cultural validation to determine whether structured planning advantages generalize across different spatial reasoning backgrounds and experiences
