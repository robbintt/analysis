# Efficient License Plate Recognition via Pseudo-Labeled Supervision with Grounding DINO and YOLOv8

## Quick Facts
- **arXiv ID**: 2510.25032
- **Source URL**: https://arxiv.org/abs/2510.25032
- **Authors**: Zahra Ebrahimi Vargoorani; Amir Mohammad Ghoreyshi; Ching Yee Suen
- **Reference count**: 0
- **Primary result**: ALPR system achieves 94% recall and 3.5% CER on CENPARMI, outperforming OpenALPR and CNN-RNN + CTC models

## Executive Summary
This paper addresses challenges in automatic license plate recognition (ALPR) systems caused by environmental factors like lighting, rain, and dust, as well as varying camera angles and image quality. The proposed deep learning framework uses YOLOv8 for license plate detection and character recognition, enhanced by a semi-supervised learning approach that leverages Grounding DINO to generate pseudo-labels for unlabeled data. The system was evaluated on CENPARMI and UFPR-ALPR datasets, demonstrating superior performance compared to established ALPR models.

The research presents a practical solution for real-world ALPR applications by combining efficient object detection with semi-supervised learning techniques. The approach significantly improves recognition accuracy while reducing the need for extensive labeled training data, making it particularly valuable for scenarios where labeled datasets are limited or expensive to obtain.

## Method Summary
The framework employs YOLOv8 as the core detection and recognition model, utilizing its capabilities for both license plate localization and character segmentation. Grounding DINO serves as the pseudo-label generator, creating synthetic annotations for unlabeled images through visual grounding techniques. This semi-supervised approach allows the model to leverage large amounts of unlabeled data while maintaining high accuracy through the quality assurance provided by Grounding DINO's visual grounding capabilities. The training process involves an iterative cycle where pseudo-labels are generated, filtered, and used to augment the training dataset, gradually improving the model's performance without requiring additional manual annotation effort.

## Key Results
- Achieved 94% recall and 3.5% character error rate (CER) on CENPARMI dataset
- Obtained 91% recall and 7.5% CER on UFPR-ALPR dataset
- Outperformed established models like OpenALPR and CNN-RNN + CTC in both accuracy metrics

## Why This Works (Mechanism)
The system's effectiveness stems from combining YOLOv8's strong detection capabilities with semi-supervised learning through Grounding DINO. YOLOv8 provides accurate license plate localization and character recognition through its unified detection architecture, while Grounding DINO generates high-quality pseudo-labels that expand the training dataset without manual annotation. The iterative refinement process allows the model to learn from both labeled and pseudo-labeled data, improving generalization across diverse conditions.

## Architecture Onboarding

**Component Map**: Image Input -> Grounding DINO -> Pseudo-Label Generator -> YOLOv8 Training -> Detection/Recognition Output

**Critical Path**: The pseudo-label generation and filtering pipeline represents the critical path, as label quality directly impacts YOLOv8's training effectiveness and final recognition accuracy.

**Design Tradeoffs**: The system balances computational efficiency with accuracy by using YOLOv8's unified architecture instead of separate detection and recognition models, while the semi-supervised approach reduces labeling costs at the expense of potential pseudo-label noise.

**Failure Signatures**: Performance degradation typically manifests as increased CER in challenging lighting conditions or when pseudo-labels contain systematic errors from Grounding DINO's grounding inaccuracies.
