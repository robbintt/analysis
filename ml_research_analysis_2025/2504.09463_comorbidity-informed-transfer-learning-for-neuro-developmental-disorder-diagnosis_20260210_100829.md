# Comorbidity-Informed Transfer Learning for Neuro-developmental Disorder Diagnosis

## Quick Facts
- **arXiv ID**: 2504.09463
- **Source URL**: https://arxiv.org/abs/2504.09463
- **Reference count**: 40
- **Primary result**: CITL achieves 76.32% accuracy for ASD and 73.15% for ADHD diagnosis, outperforming existing transfer learning methods

## Executive Summary
This paper introduces a Comorbidity-Informed Transfer Learning (CITL) framework to improve diagnosis of neurodevelopmental disorders (ASD and ADHD) using fMRI data. The method addresses the challenge of complex spatio-temporal fMRI features and limited labeled data by integrating comorbidity mechanisms between disorders with transfer learning and semi-supervised learning. CITL employs a pseudo-labeling approach combined with an autoencoder to filter low-correlation dynamic functional connectivity data and generate enhanced representations. The core idea is to leverage shared comorbid patterns between ASD and ADHD to improve feature learning.

## Method Summary
The CITL framework integrates comorbidity mechanisms with transfer learning and semi-supervised learning for neurodevelopmental disorder diagnosis. It uses pseudo-labeling combined with an autoencoder to filter low-correlation dynamic functional connectivity data from fMRI scans. The framework leverages shared comorbid patterns between ASD and ADHD to improve feature learning, addressing the challenges of complex spatio-temporal features and limited labeled data in neuroimaging diagnosis.

## Key Results
- CITL achieves 76.32% accuracy for ASD detection
- CITL achieves 73.15% accuracy for ADHD detection
- Outperforms existing transfer learning methods by 7.2% for ASD and 0.5% for ADHD

## Why This Works (Mechanism)
The framework works by leveraging the inherent comorbidity relationships between neurodevelopmental disorders to improve feature learning from fMRI data. By using pseudo-labeling with an autoencoder, CITL can extract meaningful patterns from limited labeled data while filtering noise from dynamic functional connectivity measurements. The comorbidity-informed approach allows the model to learn shared pathological mechanisms between disorders, enhancing its diagnostic capabilities.

## Foundational Learning
- **Dynamic Functional Connectivity (DFC)**: Captures temporal variations in brain connectivity patterns
  - *Why needed*: Neurodevelopmental disorders show time-varying connectivity changes
  - *Quick check*: Verify DFC captures relevant temporal patterns in disorder-specific ways

- **Semi-supervised Learning**: Utilizes both labeled and unlabeled data for training
  - *Why needed*: Limited availability of labeled fMRI data for rare disorders
  - *Quick check*: Confirm performance gain from unlabeled data integration

- **Autoencoder Architecture**: Learns compressed representations of input data
  - *Why needed*: Reduces dimensionality while preserving diagnostic information
  - *Quick check*: Validate reconstruction quality and feature preservation

## Architecture Onboarding

**Component Map**: fMRI Data -> Preprocessing -> DFC Extraction -> Autoencoder -> Pseudo-labeling -> Comorbidity Integration -> Classification

**Critical Path**: The most critical components are the autoencoder for representation learning and the comorbidity integration mechanism, as they directly impact diagnostic accuracy.

**Design Tradeoffs**: The framework balances between computational complexity (autoencoder training) and diagnostic accuracy (comorbidity-informed features). The pseudo-labeling approach trades potential label noise for expanded training data.

**Failure Signatures**: Poor performance may indicate inadequate comorbidity pattern capture, insufficient quality in pseudo-labels, or suboptimal autoencoder training leading to loss of diagnostic features.
