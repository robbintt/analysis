## Executive Summary

Computational materials science relies heavily on first-principles calculations, particularly using packages like the Vienna Ab initio Simulation Package (VASP), to predict material properties. However, these computational workflows are inherently complex, requiring significant domain expertise to correctly configure input files, manage parameter dependencies, and execute sequences of operations. This manual overhead creates a bottleneck in high-throughput materials discovery and increases the risk of human error. Addressing this challenge requires a system capable of understanding high-level scientific goals and autonomously executing the intricate, low-level technical procedures required to achieve them.

The paper introduces a modular, Large Language Model (LLM)-driven agentic framework designed to automate these complex computational workflows. The core technical innovation lies in the separation of concerns through two key abstractions: **"Workflows,"** which define high-level strategies for scientific procedures, and **"Modular Components,"** which encapsulate reusable operations such as File I/O, system commands, data processing, and the LLM interface. The framework utilizes hierarchical prompt templates to map user requests and input files to the appropriate workflows. It then executes these by chaining together specific modular components, effectively bridging the gap between natural language instructions and the rigorous syntax required by scientific software.

The framework was evaluated using a benchmark dataset comprising 80 practical scenarios across four distinct tasks: Structural Relaxation (SR), Band Structure (BS), Adsorption Energy (AE), and Transition State (TS). When tested with models like GPT-4o and DeepSeek-V3, the agentic approach significantly outperformed baseline methods. For GPT-4o, the framework improved the Completion Rate from 66.46% to 97.92% and increased Accuracy from 45.74% to 73.07%. Task-specific analysis revealed high robustness in Structural Relaxation (accuracy >95%) and strong performance in Band Structure and Adsorption Energy tasks (>80%), while the Transition State task proved more challenging due to complexities in tag initialization and context management.

This research represents a significant advancement toward the "autonomous laboratory" by demonstrating that LLM-based agents can reliably manage sophisticated scientific computing tasks with minimal human intervention. By drastically improving the success rate and accuracy of complex simulations, the framework has the potential to accelerate materials discovery and democratize access to computational tools for researchers who may lack deep programming expertise. Furthermore, the authors' analysis of failure modes‚Äîsuch as tag interdependence and workflow context issues‚Äîprovides critical insights for future iterations, establishing a foundation for more robust, intelligent agents in computational science and engineering.

## Technical Details

The proposed approach is a modular, LLM-driven agentic framework specifically targeting first-principles calculations using VASP.

### Core Abstractions
*   **Workflows:** High-level strategies that define scientific procedures and logic.
*   **Modular Components:** Reusable building blocks that perform specific operations, including:
    *   File I/O
    *   System Operations
    *   Data Processing
    *   LLM Interface

### Execution Logic
1.  **Mapping:** User requests and input files are mapped to appropriate workflows.
2.  **Implementation:** Workflows are implemented via sequences of operations.
3.  **Prompting:** The system utilizes hierarchical prompt templates to guide the LLM through the execution process.

## Results

The framework was evaluated against a benchmark of **80 practical scenarios** covering four distinct tasks.

### Performance Metrics (GPT-4o)
The agentic framework yielded significant improvements over baseline methods:

*   **Completion Rate:** Increased from **66.46%** to **97.92%**
*   **Accuracy:** Improved from **45.74%** to **73.07%**

### Task-Specific Accuracy
*   **Structural Relaxation (SR):** > 95% (High Robustness)
*   **Band Structure (BS):** > 80%
*   **Adsorption Energy (AE):** > 80%
*   **Transition State (TS):** Remained challenging; identified as the most difficult task.

### Identified Failure Modes
*   Incorrect tag initialization.
*   Poor handling of tag interdependence.
*   Issues with workflow context management.
