## üìù Executive Summary

This research addresses the challenge of optimizing communication for assistive agents operating in dynamic environments, where human cognitive capacity is a finite, zero-sum resource. The problem is significant because existing assistive systems often treat information delivery as isolated events or fail to account for how specific messages shift a user's attention over time. Consequently, agents may fail to align the user's mental model with the actual state of the environment, particularly when high-stakes or critical information is involved.

The authors argue that effective assistance requires moving beyond simple priority-based signaling to a holistic approach that combines **multi-step planning** with a **realistic model of user awareness**. The key innovation is the development of a theoretical framework that applies the **Rational Speech Act (RSA)** framework to Human-AI Interaction (HAI) within dynamic settings‚Äîa first in the field.

Technically, the model utilizes a Literal Speaker ($S_0$) and a Pragmatic Listener ($L_1$) that performs Bayesian inference, supported by a Dynamic Belief Update Mechanism. This mechanism employs an Attention Function ($A_T$) and a Belief Update Function ($B$) to simulate how prior messages shape user attention and interpretation.

As the source text focuses on the theoretical formulation, the study does not currently provide numerical performance data. Instead, results are presented as defined evaluation metrics designed to validate the framework's efficacy, specifically a Reward Function ($R_t$) and Belief Alignment ($b_t \approx s_t$). This work makes a substantial contribution by bridging the gap between rational communication principles in cognitive science and the practical design of assistive agents.

## ‚öô Technical Details

The paper proposes an RSA-based framework for dynamic, multi-step assistive signaling. The system is formalized using the following tuple structure:

$$ (\mathcal{S}, \mathcal{P}, \mathcal{U}, H, \mathcal{B}, R) $$

*   $\mathcal{S}$: State Space
*   $\mathcal{P}$: Properties
*   $\mathcal{U}$: Utterances
*   $H$: Planning Horizon
*   $\mathcal{B}$: Beliefs
*   $R$: Reward

### Core Components

1.  **Literal Speaker ($S_0$):** Selects utterances stochastically based on a lexicon matrix.
2.  **Pragmatic Listener ($L_1$):** Performs Bayesian inference to interpret messages.
3.  **Dynamic Belief Update Mechanism:**
    *   **Attention Function ($A_T$):** Models user focus.
    *   **Belief Update Function ($B$):** Updates user beliefs based on attention.
4.  **Baseline Pragmatic Speaker ($S_{baseline}^2$):** Greedily selects utterances to maximize a reward function based on belief alignment and criticality.
