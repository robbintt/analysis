## Executive Summary

This paper addresses the critical challenge of effective task planning within Large Language Model (LLM)-based Multi-Agent Systems (MAS), particularly when operating in complex environments characterized by uncertainty. Current MAS architectures frequently struggle with the dynamic decomposition of tasks and are prone to generating misleading or hallucinated outputs due to insufficient constraints on agent behaviors.

The key innovation of the paper is the introduction of **XAgents**, a unified framework that synthesizes a **Multipolar Task Processing Graph (MTPG)** with an **IF-THEN Rule-based Decision Mechanism (ITRDM)**. Technically, the MTPG utilizes a directed acyclic graph (DAG) structure employing SIMO (Single Input Multiple Output) for task decomposition and MISO (Multiple Input Single Output) for result fusion to effectively manage uncertainty. The ITRDM enforces a dual-layer control mechanism consisting of domain-specific rules, which constrain individual agent behaviors at the micro level, and global rules, which foster inter-agent collaboration at the macro level.

XAgents was evaluated using GPT-4 on three distinct datasets covering both knowledge-typed and logic-typed question-answering tasks: Trivia Creative Writing (TCW), Codenames Collaborative (CC), and Logic Grid Puzzle (LGP). The evaluation demonstrated substantial quantitative improvements, most notably achieving **92.59% accuracy** on the Logic Grid Puzzle dataset compared to 87.04% for the best performing multi-agent baseline. These results confirmed that the MTPG effectively enables dynamic task planning while the IF-THEN rules successfully reduce misleading outputs across all evaluated scenarios.

## Technical Details

The XAgents framework is inspired by multipolar neurons and consists of two core modules: the **Multipolar Task Processing Graph (MTPG)** and the **IF-THEN Rule-based Decision Mechanism (ITRDM)**.

### Core Architecture

| Component | Structure | Function |
| :--- | :--- | :--- |
| **MTPG** | Directed Acyclic Graph (DAG) | Uses **SIMO** (Single Input Multiple Output) for task decomposition and **MISO** (Multiple Input Single Output) for result fusion to mitigate uncertainty. |
| **ITRDM** | Logic-based Constraint System | Constrains agent behavior and reduces hallucinations using domain-specific and global rules. |

### Agent Roles & Workflow

The framework coordinates specialized agents through a structured flow of planning, rule-based execution, fusion, and validation:

*   **Planner Agent (PA):** Initiates the flow.
*   **Domain Analyst Agent (DAA):** Analyzes specific domains.
*   **Domain Expert Agents (DEA):** Execute specific subtasks.
*   **Fusion Expert Agent (FEA):** Handles result fusion.
*   **Global Expert Agent (GEA):** Performs global constraint checking.
