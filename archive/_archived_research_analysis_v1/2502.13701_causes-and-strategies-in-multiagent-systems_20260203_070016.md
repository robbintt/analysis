# Causes and Strategies in Multiagent Systems

*Sylvia S. Kerkhove; Natasha Alechina; Mehdi Dastani*

## Executive Summary

This research addresses the critical disconnect between causality and strategic reasoning in multi-agent systems. While Structural Causal Models (SCMs) excel at determining why outcomes occur, and Concurrent Game Structures (CGS) excel at modeling strategic interactions, there has previously been no unified framework to analyze actual causality within strategic settings. This gap matters because as autonomous agents become more prevalent in complex environments, the ability to rigorously attribute causality to specific strategic decisionsâ€”rather than just observing state changesâ€”is essential for accountability, verification, and understanding system behavior.

The key innovation is the introduction of the **"Causal Concurrent Game Structure,"** a novel model that formally integrates Pearlâ€™s Structural Causal Models with Concurrent Game Structures. The authors developed a systematic translation method where state transitions are formally defined as interventions on agent variables, thereby bridging causal actions with game-theoretic moves. Technically, the model employs an Agent Ranking Function ($\rho$) to map variables to ranks based on dependency hierarchy and utilizes a "Causal Strategy Profile" ($\sigma_M$) to define baseline behavior derived from structural equations. This allows the framework to apply the modified Halpern-Pearl definition of actual causality directly to the decisions made by interacting agents.

The study provides rigorous theoretical results regarding the structural properties and complexity of the proposed model. The authors proved that the size of the generated causal CGS is linear in the size of the extension of the structural equations $F$, ensuring computational feasibility. Specific metrics derived include a Model Extension Size of $|V| \times \prod_{v \in V} |R(v)|$ and a defined number of Leaf States equal to $\prod_{v \in V_{agt}} |R(v)|$. Furthermore, the Total States within the system are strictly bounded by $2 \times \prod_{v \in V_{agt}} |R(v)|$, demonstrating the model's ability to maintain structural compactness while representing complex causal interactions.

This work significantly advances the field by providing the first formal mechanism to reason about how strategic interventions causally impact other variables in a multi-agent environment. By bridging the research gap between causal inference and game theory, it enhances the capability to analyze complex, interactive systems where understanding the "why" behind an outcome is as vital as predicting the outcome itself. This framework establishes a foundation for future research in formal verification and explainable AI, offering a robust tool for designers to verify agent strategies and attribute causal responsibility in automated systems.

## Technical Details

The authors propose a **'Causal Concurrent Game Structure' (CGS)** that unifies Pearl's Structural Causal Models and Concurrent Game Structures to analyze actual causality in multi-agent interactions.

**Core Components:**

*   **Variable Classification:**
    *   **Exogenous ($U$):** External variables not determined by the model.
    *   **Endogenous ($V$):** Variables determined by the model.
    *   **Agent Variables ($V_{agt}$):** Variables controlled by agents.
    *   **Environment Variables ($V_{env}$):** Variables representing the environment state.

*   **Agent Ranking Function ($\rho$):** Determines turn order by mapping variables to ranks ($0$ to $k$) based on dependency hierarchy.

*   **State Representation:** The state is represented as a tuple $(t, \vec{v})$, where $t$ is the rank/time step and $\vec{v}$ is the value assignment.

*   **Dynamics:**
    *   **Transitions:** Modeled as interventions on agent variables.
    *   **Causal Strategy Profile ($\sigma_M$):** Defines the baseline behavior derived from the original SCM, where agents choose actions matching the structural equations.
