## ðŸ§  Technical Details

The analyzed papers utilize various advanced architectures for trajectory prediction in autonomous driving:

*   **M2I:** Proposes a goal-driven, two-stage framework with a marginal predictor and an interactive predictor using transformers.
*   **HiVT:** Employs a hierarchical Vector Transformer with local interaction and global context encoders using vector representations.
*   **Implicit Latent Variable Model:** Uses Neural Implicit Representation to ensure scene consistency.
*   **LookOut:** Integrates prediction and planning using diversity loss functions to generate diverse futures.
*   **JFP:** Uses a unified model with Graph Neural Networks or Transformers for joint prediction to preserve correlations between agents.
